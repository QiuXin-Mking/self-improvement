# q
路由是什么？它在网络层中的作用是什么？
# a
路由是网络层的核心功能，指路由器根据目标设备的逻辑地址（通常是IP地址）确定数据包从源到目标的最佳路径，并将数据包从一个接口转发到另一个接口的过程。它实现了不同子网和网络之间的通信，确保数据有效、可靠地传输。

# q
路由表包含哪些关键信息？
# a
路由表包含：1. 目标网络地址（IP地址范围和子网掩码）；2. 下一跳（下一跳路由器的IP地址）；3. 出接口（数据包离开的物理或逻辑接口）；4. 距离/成本（度量，如跳数、带宽、延迟）。

# q
动态路由和静态路由有何区别？
# a
静态路由由网络管理员手动配置，路由表不会自动适应网络变化；动态路由使用路由协议（如RIP、OSPF、BGP）自动学习网络拓扑，并根据变化自动更新路由表，确保适应动态环境。

# q
路由器选择最佳路径时通常基于哪些度量？
# a
最佳路径选择基于度量标准，如距离（跳数）、成本、带宽、延迟等。路由器会选择最低成本的路径转发数据包。
