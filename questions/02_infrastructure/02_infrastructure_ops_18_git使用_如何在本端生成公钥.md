# q
如何使用 `ssh-keygen` 生成一个 RSA 4096 位的密钥对并添加邮箱注释？
# a
使用以下命令：
```bash
ssh-keygen -t rsa -b 4096 -C "1322053516@qq.com"
```
该命令会生成 RSA 算法的公钥和私钥，密钥长度为 4096 位，注释为邮箱地址 `1322053516@qq.com`。

# q
`ssh-keygen` 命令中的 `-t rsa` 参数的作用是什么？
# a
`-t rsa` 指定生成使用 RSA 算法的密钥对。

# q
`ssh-keygen` 中的 `-b` 参数有什么作用？示例中 4096 代表什么？
# a
`-b` 用于指定密钥的长度（位数）。示例中的 `4096` 表示生成 4096 位的密钥，提供更高的安全性。

# q
`ssh-keygen` 命令里的 `-C` 参数是做什么的？
# a
`-C` 参数用于给生成的密钥对添加一个注释，通常用邮箱或用户名来标识和备注该密钥的归属。
