# q
路由是什么？
# a
路由是指路由器从一个接口上收到数据包，根据数据包的目的地址进行定向并转发到另一个接口的过程。它工作在OSI参考模型的第三层（网络层），主要包含两个基本动作：确定最佳路径和通过网络传输信息（即数据交换）。

# q
路由与桥接的核心区别是什么？
# a
主要区别在于工作层次：桥接发生在OSI第二层（数据链路层），而路由发生在第三层（网络层）。因此二者在传递信息时使用不同的信息，完成任务的方式也不同。

# q
路由工作包含哪两个基本动作？
# a
1. 确定最佳路径
2. 通过网络传输信息（也称为数据交换）
其中选择路径相对复杂，交换相对简单。

# q
静态路由和动态路由分别指什么？动态路由有哪些常见协议？
# a
- 静态路由：路由转发规则事先确定好，例如固定路由法（用于稳定的网络）和扩散法（如军事网络）。
- 动态路由：路由规则根据网络状态动态调整，常见协议有RIP和OSPF。
