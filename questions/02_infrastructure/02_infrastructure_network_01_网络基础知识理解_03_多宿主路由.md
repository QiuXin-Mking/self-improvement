# q
什么是多宿主路由？
# a
多宿主路由是一种网络设计和配置方式，允许一个网络设备（通常是服务器）同时连接到多个路由器，通过分散流量、提供备用路径等手段实现负载均衡、冗余和容错性，从而提高网络可用性和性能。

# q
多宿主路由通过哪三个方面提升网络可靠性？
# a
- **负载均衡**：将流量分散到多个路由器，避免单一路径过载，提升整体性能。
- **冗余**：当某一路径或路由器故障时，自动切换到其他可用路径，保持连通性。
- **容错性**：即使个别路由器不可用，网络仍然能继续运行，降低单点故障影响。

# q
实现多宿主路由的常见协议有哪些？它们分别如何工作？
# a
- **ECMP (Equal-Cost Multipath)**：当到达目标网络存在多条开销相等的路径时，将流量平均分布到这些路径上。
- **VRRP (Virtual Router Redundancy Protocol)**：多个路由器共享一个虚拟IP地址，选举主路由器处理流量；主路由器故障时，备份路由器接管该虚拟IP。
- **HSRP (Hot Standby Router Protocol)**：与VRRP类似，也是通过虚拟IP提供默认网关冗余，主路由器失效后由备份路由器接替。
