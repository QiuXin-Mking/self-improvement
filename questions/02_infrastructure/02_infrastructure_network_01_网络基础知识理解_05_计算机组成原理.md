# q
网络按作用范围如何分类，分别是什么？
# a
按照网络的作用范围可分为三类：
- 广域网（WAN）
- 城域网（MAN）
- 局域网（LAN）

# q
如何理解往返时间RTT？
# a
RTT（Round-Trip Time）是数据报文在端到端通信中从发送端到接收端再返回发送端一次所经历的总时间，是评估网络时延的重要指标。

# q
单工、半双工和全双工通信信道有何区别？
# a
- **单工通信信道**：只能单向通信，没有反方向反馈能力。
- **半双工通信信道**：双方都能发送和接收，但不能同时进行，必须交替通信。
- **全双工通信信道**：双方可以同时发送和接收，实现双向同时通信。

# q
数据链路层的主要作用和关键组件有哪些？
# a
数据链路层在物理层之上为网络层提供可靠的数据传输，基本数据单位为帧。
主要作用包括：物理地址寻址、数据成帧、流量控制、差错检测与重发。
关键协议：以太网协议。
重要设备：网桥和交换机。
