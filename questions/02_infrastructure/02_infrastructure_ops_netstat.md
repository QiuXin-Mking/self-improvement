# q
netstat 命令的核心功能是什么？
# a
`netstat` 是一个用于显示系统网络状态的命令。

# q
netstat -ntlp 命令的作用是什么？
# a
`netstat -ntlp` 用于显示所有正在监听（LISTEN）的 TCP 端口，并列出使用这些端口的进程。其中 `-n` 以数字形式显示地址和端口号，`-t` 仅显示 TCP 连接，`-l` 仅显示监听套接字，`-p` 显示关联进程。

# q
netstat -ntp 命令的作用是什么？
# a
`netstat -ntp` 用于显示所有正在进行中的 TCP 连接，并列出对应的进程。选项 `-n`、`-t`、`-p` 分别表示数字格式、TCP 协议和显示关联进程。

# q
netstat 命令中 -n、-t、-l、-p 选项分别代表什么含义？
# a
- `-n`：以数字形式显示地址和端口号
- `-t`：仅显示 TCP 连接
- `-l`：仅显示处于监听（LISTEN）状态的套接字
- `-p`：显示与套接字关联的进程信息

# q
netstat -ntlp 与 -ntp 的主要区别是什么？
# a
`-ntlp` 专注于显示正在监听（LISTEN）的 TCP 端口及进程，用于查看哪些服务在等待连接；`-ntp` 则显示所有已建立的 TCP 连接及进程，用于查看当前活跃的网络连接情况。两者核心差异在于是否包含 `-l`（监听过滤）。
