# q
这个脚本的核心功能是什么？
# a
实现在多台服务器之间批量配置SSH免密登录，使任意两台服务器都能通过SSH互访而无需输入密码。

# q
脚本中如何使用 `mapfile` 命令从文件读取服务器IP列表？
# a
通过 `mapfile -t SERVERS < /root/ip` 将文件 `/root/ip` 的每一行读入数组 `SERVERS`，`-t` 选项会去除行尾的换行符，替换了注释掉的硬编码IP数组。

# q
脚本通过哪三个主要步骤完成服务器间的SSH公钥交换？
# a
1. 从控制节点生成公钥并分发给所有服务器（`ssh-copy-id`）；
2. 从每台服务器拉取其已有的公钥，合并到控制节点的 `authorized_keys`；
3. 将汇总的公钥文件推送到所有服务器，实现任意两台服务器间的互信。

# q
脚本最后如何验证免密登录配置是否成功？
# a
通过循环遍历 `SERVERS` 数组，使用 `ssh "$ser" date` 命令，如果能在每台服务器上远程执行 `date` 命令并正常返回，说明免密登录已生效。
