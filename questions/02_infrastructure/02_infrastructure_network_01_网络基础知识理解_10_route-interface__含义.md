# q
在基于Red Hat/CentOS的Linux系统中，/etc/sysconfig/network-scripts/route-<interface>文件的作用是什么？
# a
该文件用于存储特定网络接口（如 eno1）的静态路由配置信息，系统启动时会应用这些路由规则，确保流向指定网络地址段的流量通过定义的网关发送。

# q
route-<interface>文件的标准条目格式是什么？
# a
格式为每行一条路由，使用 `ADDRESS/MASK via GATEWAY` 的写法，例如：
```
192.168.1.0/24 via 192.168.1.1
10.0.0.0/8 via 10.0.0.1
```
表示到达 `192.168.1.0/24` 的流量使用网关 `192.168.1.1`，到达 `10.0.0.0/8` 的流量使用网关 `10.0.0.1`。

# q
route-<interface>文件中定义的路由规则在什么时候被应用？
# a
这些路由规则在系统启动时被网络脚本加载并应用，从而持久化特定的网络路由。
