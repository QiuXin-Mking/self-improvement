# q
二层网络（数据链路层）的核心功能是什么？主要使用哪些设备和协议？
# a
核心功能是将原始的、不可靠的物理链路转化为逻辑上可靠的通信链路。主要设备包括交换机（Switch）和网卡（NIC），常见协议有以太网（Ethernet）和无线局域网（Wi-Fi）。

# q
三层网络（网络层）的核心功能是什么？主要使用哪些设备和协议？
# a
核心功能是在不同网络之间传递数据包，选择最佳路径，提供路由和转发功能。主要设备是路由器（Router），主要协议是互联网协议（IP）。

# q
二层网络和三层网络在地址标识上的主要区别是什么？
# a
二层网络使用 MAC 地址（物理地址）唯一标识设备，三层网络使用 IP 地址（逻辑地址）标识设备。

# q
二层网络和三层网络在转发设备及作用范围上的主要区别是什么？
# a
二层网络通过交换机在本地网络内部进行数据帧转发，作用范围限于同一局域网；三层网络通过路由器在不同网络之间进行数据包转发，实现跨网络的端到端通信。
