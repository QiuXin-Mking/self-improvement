# q
HAProxy是什么？
# a
HAProxy是一个使用C语言编写的自由及开放源代码软件，提供高可用性、负载均衡，以及基于TCP和HTTP的应用程序代理。

# q
HAProxy特别适用于哪些场景？其性能特点是什么？
# a
特别适用于负载特大的web站点，这些站点通常需要会话保持或七层处理。HAProxy可以支持数以万计的并发连接，并且可以简单安全地整合进现有架构，同时保护后端web服务器不被暴露到网络上。

# q
负载均衡集群的基本工作原理是什么？
# a
通过一个或多个前端负载均衡器将客户访问请求分发到后端的一组服务器上，从而实现整个系统的高性能和高可用性。高可用性集群和负载均衡集群常会使用类似技术，或同时兼具两种特性。

# q
HAProxy工作在网络的哪些层级？支持哪些应用协议？
# a
HAProxy基于四层和七层技术，可提供TCP和HTTP应用的负载均衡综合解决方案。
