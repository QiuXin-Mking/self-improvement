# q
广播流量是什么，它使用什么地址进行通信？
# a
广播流量是一种网络通信方式，一台设备将数据发送到网络中的所有设备（而非特定目标）。在IPv4中，广播流量通常使用网络的广播地址，例如`192.168.1.255`，路由器通常不转发广播流量，以限制其在本地网络内传播。

# q
本地流量是什么，回环地址通常是多少？
# a
本地流量是指同一台主机内部进程之间通信的流量，不经过物理网络。它通常使用回环地址`127.0.0.1`，并在主机内部直接处理，不经过网络接口。

# q
广播流量与本地流量的核心区别有哪些？
# a
- **范围**：广播流量涉及整个本地网络，本地流量限于同一台主机内部。
- **地址**：广播流量使用网络广播地址（如`192.168.1.255`），本地流量使用回环地址`127.0.0.1`。
- **处理**：广播流量可能需要通过网络设备传播（但路由器不转发），本地流量直接在主机内部处理，不经过网络接口。
