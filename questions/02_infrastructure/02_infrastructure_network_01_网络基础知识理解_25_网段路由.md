# q
如何通过静态路由实现两个不同网段（192.168.251.0/24 和 192.169.251.0/24）之间的互通？
# a
需要在每台服务器上添加指向对方网段的静态路由，指定各自的网关。

示例命令：
```bash
# 在 192.168.251.0/24 网段的服务器上执行
sudo ip route add 192.169.251.0/24 via 192.168.251.1

# 在 192.169.251.0/24 网段的服务器上执行
sudo ip route add 192.168.251.0/24 via 192.169.251.1
```
其中 `192.168.251.1` 和 `192.169.251.1` 是各网段的网关 IP。配置完成后使用 ping 验证互通性。
