# q
Unix 文件类型包含哪些？
# a
- 普通文件（regular file）
- 目录
- 符号链接
- 面向块的设备文件（block-oriented device file）
- 面向字符的设备文件（character-oriented device file）
- 管道（pipe）和命名管道（named pipe）
- 套接字（socket）

# q
Unix 中用于进程间通信的文件类型有哪些？
# a
管道（pipe）、命名管道（named pipe）和套接字（socket）

# q
Unix 的设备文件分为哪两种？
# a
面向块的设备文件（block-oriented device file）和面向字符的设备文件（character-oriented device file）
