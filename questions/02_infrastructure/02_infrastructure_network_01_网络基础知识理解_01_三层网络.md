# q
三层网络架构指的是哪三个层次？
# a
三层网络架构指的是计算机网络中的三层分层模型，通常参考 OSI 模型的底层三层，分别是：  
1. 物理层（Physical Layer）  
2. 数据链路层（Data Link Layer）  
3. 网络层（Network Layer）

# q
物理层的核心任务是什么？
# a
物理层关注传输介质和物理连接，确保比特流能够在网络中传输，处理硬件设备之间的直接通信，包括电缆、光纤、无线信号等物理媒介。

# q
数据链路层的作用及其两个子层是什么？
# a
数据链路层提供可靠的点对点通信，处理物理层传输的数据帧（Frame），负责错误检测和纠正，确保相邻节点间的可靠传输。  
它的两个子层是：MAC（Media Access Control，媒体访问控制）子层和 LLC（Logical Link Control，逻辑链路控制）子层。

# q
网络层的主要功能是什么？常用的网络层协议是什么？
# a
网络层负责整个网络中的寻址和路由，使用逻辑地址（如 IP 地址）标识不同主机间的通信，并选择合适的路径将数据包从源主机传输到目标主机。常见的网络层协议是 IP（Internet Protocol）。
