# q
网关是什么？
# a
网关是连接不同子网的设备，负责在不同子网之间进行路由，将数据包从一个子网传递到另一个子网。它实质上是在子网或网络之间进行数据传输的设备或计算机，起到桥梁和转发器的作用。

# q
网关在计算机网络中常见的角色有哪些？
# a
1. **数据包转发**：将计算机尝试与其他网络通信的数据包通过网关进行转发，确保正确到达目标。
2. **连接不同网络**：连接不同类型或协议的网络，实现不同子网之间的通信。
3. **网络地址转换（NAT）**：将内部私有IP地址映射到外部公共IP地址，实现对外通信。
4. **安全性控制**：实施防火墙、过滤流量或执行访问控制列表（ACL）保护网络。
5. **路由选择**：选择最佳路径传递数据包，确保高效到达目的地。

# q
为什么不同子网之间通信需要网关？
# a
如果两个服务器分别位于不同的子网（例如 192.168.0.0/16 和 192.169.0.0/16），需要确保每个子网都有一个网关（通常是路由器），才能在不同子网之间正常转发流量，实现通信。
