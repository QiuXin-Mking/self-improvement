# q
在 `find` 命令中，`-type f` 选项的作用是什么？
# a
`-type f` 用于指定只查找普通文件，排除目录、链接等其他类型的对象。

# q
使用 `find -name` 进行文件模式匹配时，为什么建议用双引号包裹模式字符串？
# a
因为 Shell 可能会对通配符（如 `*`）进行自动展开，使用双引号可以防止这种展开，确保模式原样传递给 `find` 命令。

# q
`find -exec` 中 `{}` 和 `\;` 分别代表什么？
# a
`{}` 是占位符，会被每一个查找到的文件路径替换。`\;` 是 `-exec` 命令的结束标志，由于分号在 Shell 中有特殊含义，需要用反斜线转义。

# q
如何用一条 `find` 命令，从当前目录递归找出所有 `.sh` 文件并赋予执行权限？
# a
```bash
find . -type f -name "*.sh" -exec chmod +x {} \;
```
