# q
如何在 Linux 中使用 `ip route` 命令添加一条指定网关的静态路由？
# a
使用 `sudo ip route add <目标网络/IP> via <网关IP>`。例如：
```bash
sudo ip route add 192.168.234.105 via 172.22.251.105
```
这条命令会添加一条到达 192.168.234.105 的路由，网关为 172.22.251.105，并自动选择出口网卡（如 `dev eno1`）。

# q
`ip route add` 命令中 `via` 后面的参数应该填写什么类型的地址？
# a
`via` 后面必须填写**网关的具体IP地址**，不能是网络地址。例如正确的写法是 `via 172.22.251.105`，而不是一个网络号。网关地址必须是一个可到达的接口地址。

# q
如何删除一条已添加的静态路由？
# a
使用与添加时相同的路由描述，但将 `add` 换成 `del`。例如：
```bash
sudo ip route del 192.168.234.105 via 172.22.251.105
```
删除后该路由条目会从路由表中消失。

# q
如何查看当前系统的路由表，并识别其中新增的静态路由？
# a
使用 `ip route` 命令。输出中每行代表一条路由条目，包括目标网络、网关或出接口、作用域（scope）、来源（proto）及 metric 等信息。例如添加静态路由后会出现类似：
```
192.168.234.105 via 172.22.251.105 dev eno1
```
对比删除后的输出，该条目不再出现。
