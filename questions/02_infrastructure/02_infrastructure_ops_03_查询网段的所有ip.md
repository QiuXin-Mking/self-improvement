# q
如何使用 nmap 扫描一个网段中的所有存活主机 IP？
# a
使用 `nmap -sn` 命令，例如扫描 172.23.10.0/24 网段：
```bash
nmap -sn 172.23.10.0/24
```
该命令会进行 ping 扫描，输出存活的 IP 地址到终端。

# q
如何将 nmap 扫描的结果同时显示在命令行并保存到文件？
# a
使用 `tee` 命令将输出重定向，例如：
```bash
nmap -sn 172.23.10.0/24 | tee output.txt
```
这样扫描结果既会实时显示在终端，也会写入到 `output.txt` 文件中。
