# q
网段路由是什么？
# a
网段路由是指将网络流量从一个特定的IP地址范围（网段）导向特定的网络路径的配置。在路由表中，通过设置路由规则，可以决定如何处理来自不同目标网段的数据包。

# q
一条路由规则通常包含哪些关键信息？
# a
路由规则主要包含：
- **目标网段（Destination Network）**：数据包要送达的目标IP地址范围，如 `172.34.0.0/16`。
- **下一跳（Next Hop）**：数据包离开当前网络后进入目标网段的下一跳路由器的IP地址，如 `172.22.0.1`。

# q
如何在Linux中使用`ip route`命令添加一条到达网段`172.34.0.0/16`，下一跳为`172.22.0.1`的路由？
# a
```bash
sudo ip route add 172.34.0.0/16 via 172.22.0.1
```

# q
如何在Windows中使用`route add`命令添加一条到达网段`172.34.0.0`，子网掩码`255.255.0.0`，下一跳为`172.17.0.1`的持久路由？
# a
```cmd
route add 172.34.0.0 mask 255.255.0.0 172.17.0.1 -p
```
其中 `-p` 参数表示添加为持久路由（重启后保留）。
