# q
Keystone 是什么？
# a
Keystone 是 OpenStack 项目中的开源身份认证服务，负责提供用户认证、角色权限控制、多租户隔离、服务目录管理和 Token 发放等核心能力，是 OpenStack 安全架构的基础组件。

# q
Keystone 包含哪些关键功能？
# a
Keystone 的核心功能包括：
- **身份认证**：支持用户名/密码、API 密钥等多种认证方式。
- **权限管理**：基于角色的访问控制（RBAC），可定义用户、角色和权限关系。
- **多租户支持**：允许多个项目或组织共享同一 OpenStack 部署，且彼此隔离。
- **服务目录**：维护所有 OpenStack 服务的端点列表，供服务发现与调用。
- **Token 服务**：认证成功后生成与管理 token，用于后续服务请求的凭证传递。
