# q
CIDR表示法中的 `192.168.0.0/16` 表示什么？
# a
表示一个网络，其网络前缀长度为16位，即网络部分为 `192.168`，主机部分为16位。该网络包含从 `192.168.0.0` 到 `192.168.255.255` 共 2^16 (65536) 个IP地址。

# q
`192.168.0.0` 和 `192.168.0.0/16` 有什么区别？
# a
- `192.168.0.0` 是无子网掩码的网络地址，默认按类别掩码解释（对192.168.0.0即为/16），含义不够明确。
- `192.168.0.0/16` 是明确的CIDR表示法，直接指定了网络前缀长度和涵盖的子网范围，常用于路由表和子网划分配置中。
