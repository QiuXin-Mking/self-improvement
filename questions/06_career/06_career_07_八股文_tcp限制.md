# q
TCP连接的理论最大数量由什么决定？
# a
由TCP四元组（源IP、源端口、目标IP、目标端口）的组合数量决定。对于IPv4，单端口的理论最大连接数为2^48（源IP 2^32 × 源端口 2^16），所有端口总和理论最大为2^64。

# q
实际中，单台服务器能够支持的TCP并发连接数主要受哪些因素限制？
# a
主要受文件描述符限制（如Linux的`ulimit`和`sysctl`参数）、内存分配（每条连接约消耗3-4 KB内存）以及端口复用技术（如`SO_REUSEADDR`）的影响，实际可优化至百万级并发。

# q
服务器TCP连接数是否存在固定上限？
# a
无固定上限，需要根据系统资源与优化策略动态平衡。
