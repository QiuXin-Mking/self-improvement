# q
三数之和问题中，如何避免选择重复的第一个元素？
# a
在对 `nums` 排序后，遍历第一个元素时，若当前元素与前一个元素相同则跳过。实现方式：在 for 循环内先跳过相同元素，如 `while (fir > 0 && nums[fir] == nums[fir-1]) fir++;`，可配合 for 的递增避免重复三元组。

# q
该代码片段中 `ans.append(nums)` 和 `res.append(...)` 犯了什么错误？
# a
标准库 `std::vector` 的添加元素方法是 `push_back`，而不是 `append`。应使用 `ans.push_back(res)` 和 `res.push_back(nums[fir])` 等。另外，逻辑上添加的应是 `res` 向量而非 `nums`，且代码仅处理了找到第一个合理组合就 break，可能遗漏其他组合。

# q
在内部双指针移动中，何时移动 `thr` 指针？
# a
当 `nums[sec] + nums[thr] > target` 时，说明两数之和过大，需要减小，因此将 `thr` 指针左移（`thr--`），直到和小于等于目标值。

# q
这段代码的输出类型是什么？
# a
返回类型为 `vector<vector<int>>`，即一个二维整型向量，包含所有和为 0 且不重复的三元组。
