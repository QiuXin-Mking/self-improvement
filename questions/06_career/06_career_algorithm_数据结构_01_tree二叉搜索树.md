# q
二叉搜索树对于任意节点，其左子树和右子树中的节点值需要满足什么条件？
# a
对于任意节点 r：
- 左子树中的所有节点的值都不大于 r 的值（都 ≤ r）。
- 右子树中的所有节点的值都不小于 r 的值（都 ≥ r）。

# q
如何通过中序遍历判断一棵二叉树是否为二叉搜索树？
# a
当且仅当该二叉树的中序遍历序列单调非降（单调不减）时，它才是二叉搜索树。

# q
二叉搜索树的查找（search）操作在遇到当前节点为空时如何处理？
# a
如果当前节点为 NULL，则说明查找失败。伪代码中通常返回一个极小值（如 `-max`）表示未找到。

# q
二叉搜索树查找的伪代码中，当查找值大于当前节点值时，下一步如何操作？
# a
递归查找当前节点的右子节点：`search(a.rc, v)`。
