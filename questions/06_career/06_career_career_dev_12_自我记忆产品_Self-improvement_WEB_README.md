# q
如何启动 Web 版本的本地服务器，默认端口是什么？
# a
在安装依赖后，执行命令:
```bash
python web_app.py
```
服务器默认监听 `http://0.0.0.0:5000`，可在同一局域网下通过电脑 IP 地址和端口访问（如 `http://192.168.1.100:5000`）。

# q
学习流程中有哪些记忆反馈选项，分别代表什么含义？
# a
反馈选项包括：
- ✅ **熟练**：记得很清楚
- 👍 **一般**：记得但不熟练
- 😐 **忘记**：忘记了部分内容
- ❌ **完全忘记**：完全不记得

# q
Web 版本的智能复习机制基于什么原理？
# a
基于艾宾浩斯遗忘曲线，自动安排复习间隔，并根据用户反馈的记忆程度动态调整下次复习时间。

# q
Web 版本使用什么技术栈，数据如何与命令行版本互通？
# a
后端使用 Flask (Python)，前端为原生 HTML + CSS + JavaScript。Web 版本与命令行版本共享同一份 `data/learning_data.json` 数据文件，数据存储格式一致，可以混用。

# q
若要将 Web 应用部署到生产环境，需要注意哪些关键配置？
# a
- 设置 `host='0.0.0.0'` 以允许外部访问
- 生产环境应使用 Gunicorn 等 WSGI 服务器，并关闭 Flask 调试模式 (`debug=False`)
- 确保防火墙开放对应端口（默认 5000）
- 可选配置 HTTPS 增强安全性
- 可使用 Docker 或内网穿透工具（如 ngrok、frp）进行公网部署
