# q
现代存储引擎（如 LevelDB、RocksDB、Cassandra、HBase）共同的基石性数据结构是什么？它如何匹配 SSD 特性？
# a
LSM-Tree（Log-Structured Merge-Tree）。它通过将随机写转换为顺序写，完美匹配 SSD 的特性，因此成为大量 NoSQL/NewSQL 数据库存储层的核心设计。原始论文发表于 VLDB '99。

# q
在分布式存储系统中，FaRM 如何利用 RDMA 实现极低延迟？
# a
FaRM 利用 RDMA 网络绕过内核和 CPU，直接在远程机器的内存中进行操作，从而将分布式事务和一致性协议的开销降到极低。该工作发表在 NSDI '14，其思想对后续低延迟分布式文件系统（如 Crail、HERD）产生了深远影响。

# q
Facebook 的 f4 温数据存储系统如何实现低成本高可靠？它在生产中大规模实践了哪种关键技术？
# a
f4（发表于 OSDI '14）为海量“温数据”（如长期保存的图片、视频）设计了低成本高可靠的对象存储系统，其核心是在大规模生产环境中实践 Erasure Coding（纠删码），并详细讨论了其中的工程挑战与解决方案。

# q
当持久内存（如 Intel Optane DC PMM）与传统 NAND SSD 组成异构存储时，论文《Strata》提出了哪一新颖的文件系统设计方法？
# a
Strata（SOSP '19）没有简单将持久内存当作更快的块设备，而是提出了一种“日志分层”方法，在 PM 和 NAND SSD 之间构建跨介质的文件系统，实现面向异构存储层次的可扩展设计。

# q
LegoOS（OSDI '18）体现了哪种数据中心系统架构理念？它将传统 OS 分解为哪些独立服务？
# a
LegoOS 是硬件资源分解（Resource Disaggregation）理念的集大成者。它将传统单机操作系统分解为处理器、内存、存储等独立的网络化服务，通过高速网络组合在一起，为云原生、超大规模存储与计算基础设施提供了全新设计视角。
