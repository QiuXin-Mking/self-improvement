# q
ByteNas 的设计目标主要针对哪些场景和需求？
# a
ByteNas 的设计目标聚焦于 NAS 下的可扩展性、低时延和高可用性，主要服务于大规模机器学习与优化场景。

# q
该岗位要求熟悉哪些分布式系统一致性理论？
# a
要求熟悉 Paxos 和 Raft 等分布式系统相关理论。

# q
该岗位要求的编程与系统调优能力包括哪些？
# a
需要熟悉 C/C++，精通网络编程和多线程编程，并掌握 Linux 环境下的性能调优方案。

# q
需要了解的业内分布式文件系统有哪些？
# a
需要熟悉 HDFS、CephFS、GlusterFS 等主流分布式文件系统。

# q
岗位要求掌握的网络文件系统或协议有哪些？
# a
需要熟悉 NFS、FUSE、SMB 等网络文件系统。
