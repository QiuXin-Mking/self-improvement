# q
RAII的基本原理是什么？
# a
RAII（资源获取即初始化）在对象的构造函数中获取资源（如内存、文件句柄、锁），在析构函数中释放资源；利用 C++ 对象生命周期自动管理资源，避免泄漏和异常安全问题。

# q
RAII如何通过作用域实现资源自动释放？
# a
将资源封装为局部对象，该对象的生命周期由作用域决定；当执行流离开作用域时，局部对象自动销毁，析构函数被调用，从而释放资源。

# q
RAII在C++中有哪些典型应用场景？
# a
- 智能指针：`std::unique_ptr`、`std::shared_ptr` 管理动态内存。
- 锁管理：`std::lock_guard`、`std::unique_lock` 自动加锁/解锁。
- 文件流：`std::ifstream`、`std::ofstream` 自动关闭文件。
