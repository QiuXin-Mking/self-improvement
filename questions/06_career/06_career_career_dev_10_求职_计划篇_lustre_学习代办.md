# q
Lustre学习代办中列出的需要掌握的文件系统核心概念有哪些？
# a
FID、layout、xattr、ACL、quota、OSS affinity、对象预取与回写

# q
Lustre学习代办中关于网络通信的部分需要学习哪些内容？
# a
LNET：跨节点通信与路由；熟悉 LNet NID、路由策略、网络容错与多网卡绑定

# q
Lustre客户端架构需要了解哪三个层次？
# a
Client：VFS/LOV/OSC 三层

# q
Lustre学习代办中与I/O路径相关的关键点有哪些？
# a
DIO、aio、mmap、read-ahead/write-back、lockless I/O 路径、理解 page cache

# q
Lustre分布式锁管理器(LDLM)的学习要点是什么？
# a
LDLM的锁模式、范围锁、glimpse 的语义
