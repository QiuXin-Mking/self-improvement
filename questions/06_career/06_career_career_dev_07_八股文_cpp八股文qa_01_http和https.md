# q
https的“s”代表什么含义？
# a
代表安全（Secure），表示HTTP协议增加了SSL/TLS加密层。

# q
https和http在数据传输安全性上的核心区别是什么？
# a
http传输不加密，数据明文传输；https通过SSL/TLS加密传输，更安全，常用于登录、支付等敏感场景。

# q
http和https分别默认使用哪个端口？
# a
http默认使用80端口，https默认使用443端口。

# q
https是否需要SSL证书？
# a
需要，https必须使用SSL证书来验证服务器身份并建立加密连接；http不需要。

# q
https与http在性能上有何差异？
# a
https性能比http差，因为增加了SSL/TLS握手和加解密过程，多了一层处理开销。
