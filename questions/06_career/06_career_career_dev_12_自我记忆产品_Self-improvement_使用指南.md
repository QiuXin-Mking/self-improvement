# q
问题文件需要按照什么格式编写？
# a
在 `questions/` 目录下创建 `.md` 文件，每个问答对以 `# q` 独占一行，下一行写问题内容；接着 `# a` 独占一行，下一行写答案内容。可以配置多个目录，编辑 `config.json` 中的 `questions_dirs` 数组。

# q
如何初始化知识库？
# a
运行命令：
```bash
python train.py --init
```
该命令会扫描所有配置目录下的 `.md` 文件，提取问题和答案。

# q
开始每日训练以及反馈记忆程度的命令和规则是什么？
# a
每天运行：
```bash
python train.py
```
训练时程序会显示问题，你输入 `a` 或 `answer` 查看答案，然后根据记忆程度输入反馈：
- `1` 或 `熟练` - 记得很清楚
- `2` 或 `一般` - 记得但不熟练
- `3` 或 `忘记` - 忘记了部分内容
- `4` 或 `完全忘记` - 完全不记得
系统会根据反馈自动安排下次复习时间。

# q
系统如何根据艾宾浩斯遗忘曲线安排复习间隔？
# a
根据反馈调整复习间隔：
- **熟练** → 7天后复习
- **一般** → 3天后复习
- **忘记** → 1天后复习
- **完全忘记** → 2小时后复习
连续正确回答后，复习间隔会自动延长。
