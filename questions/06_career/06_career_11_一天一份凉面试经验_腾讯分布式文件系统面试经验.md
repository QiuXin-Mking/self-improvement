# q
腾讯分布式文件系统面试主要考察哪些 Lustre 相关的核心知识点？
# a
面试完全围绕 Lustre 文件系统展开，重点考察以下方面：  
- Lustre 代码熟悉程度  
- 大目录 list 时遇到的问题  
- PFL（Progressive File Layout）特性  
- DNE（Distributed Namespace Environment）特性  
- 源码级别的故障分析与调试能力  
- 产品架构是否触及 Lustre 的性能瓶颈  
- C++ 相关的基础知识（八股文）

# q
面试者在腾讯分布式文件系统面试中因哪些 Lustre 能力欠缺而失败？
# a
失败原因包括：  
1. 不熟悉 Lustre 源代码  
2. 不清楚大目录 list 时的典型问题  
3. 不清楚 PFL 和 DNE 特性的细节  
4. 没有深入源码分析过问题和故障  
5. 当前所做的产品架构未达到 Lustre 瓶颈，缺乏高性能调优经验  
6. 未对 Lustre 进行过源码级别的分析  
7. 没有背诵 C++ 八股文
