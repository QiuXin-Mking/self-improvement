# q
通过 PCC 在 Lustre 客户端上使用此 SSD 缓存的主要优势是什么？
# a
- 对于缓存数据，I/O 栈变得更简单，因为没有来自其他客户端的 I/O 干扰，这使得性能优化更容易；
- 客户端节点内的硬件要求很低，任何类型的 SSD 甚至 HDD 都可以用作缓存设备；
- 缓存减轻了对象存储目标（OST）的压力，因为小的或随机的 I/O 可以规范化为大的顺序 I/O，临时文件甚至不需要刷新到 OST。
相比于条带化来说，IO 时延低。

# q
PCC 可以加速哪些具有特定 I/O 模式的应用程序？
# a
- 来自单个客户端的小尺寸随机写入（< 1MB）
- 重复读取大于 RAM 的数据（用便宜的本地 SSD 作为“超大页缓存”，突破内存容量限制，避免重复的网络传输，特别适合数据分析、机器学习等需要反复访问大数据集的工作负载！）
- 具有高网络延迟的客户端
