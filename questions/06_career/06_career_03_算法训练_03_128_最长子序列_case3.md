# q
最长子序列算法在处理长度为1的数组时返回错误结果，典型根因是什么？
# a
算法未正确处理 `len == 1` 的边界条件，导致直接返回 0。例如输入 `[0]`，期望输出 `1`，实际输出 `0`。

# q
如何通过测试用例快速定位最长子序列算法对单元素数组的兼容性问题？
# a
使用数组 `nums = [0]` 作为输入，观察输出。若输出为 `0` 而非期望的 `1`，即可定位到算法缺少对 `len == 1` 场景的处理逻辑。

# q
解决最长子序列算法在单元素数组上错误的修复流程是什么？
# a
1. 在算法入口处增加 `if len == 1: return 1` 的判断。
2. 覆盖单元素数组的单元测试，确保输入 `[0]` 返回 `1`。
3. 验证其他边界情况（如空数组仍返回 `0`）不受影响。
