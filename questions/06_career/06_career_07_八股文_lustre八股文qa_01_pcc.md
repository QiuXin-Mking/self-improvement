# q
通过 PCC 在 Lustre 客户端上使用 SSD 缓存的主要优势是什么？
# a
- 缓存数据的 I/O 栈更简单，无其他客户端干扰，性能优化更容易
- 硬件要求低，任何 SSD 甚至 HDD 均可作为缓存设备
- 减轻 OST 压力：小/随机 I/O 可规范化为大顺序 I/O，临时文件无需刷回 OST
- 相比条带化，I/O 时延更低

# q
PCC 适合加速哪些特定 I/O 模式的应用程序？
# a
- 单客户端的小尺寸随机写入（< 1MB）
- 重复读取大于 RAM 的数据（利用本地 SSD 作为“超大页缓存”突破内存容量限制，避免重复网络传输，适合数据分析、机器学习等反复访问大数据集的场景）
- 具有高网络延迟的客户端
