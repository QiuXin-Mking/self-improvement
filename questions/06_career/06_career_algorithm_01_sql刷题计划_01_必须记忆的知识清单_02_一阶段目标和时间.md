# q
这份SQL刷题一阶段（10月到11月）需要完成哪些数据结构和算法的简单题？
# a
需要完成的简单题覆盖的数据结构：数组、图、hash、字符串、堆栈、树。算法包括：排序、动态规划、贪心、递归、二分、双指针、深度优先、广度优先、sliding window。动态规划是重点。

一阶段学习中推荐了哪些算法学习资料和教程？

- CS-Notes（https://cyc2018.github.io/CS-Notes/#/README）
