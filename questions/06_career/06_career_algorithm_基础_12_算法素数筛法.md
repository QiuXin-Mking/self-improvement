# q
在线性筛（极致线性筛）中，`if ( i%pri[j] == 0 ) break;` 这一行的作用是什么？
# a
确保每个合数只被其最小质因子筛掉，从而避免重复标记，使整个筛法的时间复杂度达到线性 O(n)。如果 `i` 能被 `pri[j]` 整除，则 `pri[j]` 是 `i` 的最小质因子，继续用更大的质数去筛 `i * pri[k]` 时，这些合数的最小质因子就不是当前使用的质数，会被后续的循环再次标记，所以要 `break`。

# q
暴力筛（埃拉托斯特尼筛法）与线性筛在实现和复杂度上的主要区别是什么？
# a
暴力筛对每个质数 `i` 枚举其所有倍数 `j = i*2, i*3, ...` 进行标记，一个合数会被多个质因子重复标记，时间复杂度为 O(n log log n)。线性筛只在 `i` 乘以已收集的质数 `pri[j]` 时标记合数，并通过 `i % pri[j] == 0` 时 `break` 来保证每个合数只被其最小质因子标记一次，时间复杂度为 O(n)。

# q
在提供的欧拉函数例子中，如何利用欧拉函数判断一个数是否为质数？
# a
对于大于 1 的整数 n，质数的欧拉函数值满足 φ(n) = n-1。因此如果 `euler(n) == n-1` 成立，就可以判定 n 是质数。代码中遍历 1 到 1000，利用该性质打印出了所有满足条件的数（即质数），但这种方式效率低于专门的筛法，源文件标注为“没必要”。
