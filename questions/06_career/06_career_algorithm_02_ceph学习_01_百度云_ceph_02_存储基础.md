# q
计算机的五大部件分别是什么？
# a
1. 控制器  
2. 运算器  
3. 数据存储  
4. 输入设备  
5. 输出设备  
控制器和运算器合称CPU，数据存储包括内存和硬盘。

# q
DAS（直连存储）的特点是什么？
# a
DAS (Direct Attached Storage) 直接连接物理磁盘到主机，主机接口如SATA需要存储驱动。

# q
NAS（网络附加存储）的特点是什么？
# a
NAS (Network Attached Storage) 通过网络（LAN/WAN）提供文件挂载，支持多种协议如NFS，限制和接口较复杂，每个对象以URL形式展示。

# q
SAN（存储区域网络）的特点是什么？
# a
SAN (Storage Area Network) 将大量磁盘组成存储池并通过网络提供块设备挂载。
