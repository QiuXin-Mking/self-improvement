# q
这个知识库学习系统基于什么理论设计复习调度？
# a
基于艾宾浩斯遗忘曲线，根据反馈记忆程度自动调整复习间隔，并在连续正确回答后延长间隔。

# q
如何初始化知识库，将配置目录下的 Markdown 问题文件导入系统？
# a
首次使用时运行命令：
```bash
python train.py --init
```
这将扫描所有配置的目录（如默认的 `questions/` 及自定义目录）下的 `.md` 文件，提取 `# q` 和 `# a` 格式的问答对。

# q
训练过程中，用户可以通过哪四种反馈来标记记忆程度？各自的初始复习间隔是多少？
# a
用户可输入以下四种反馈及其初始复习间隔：
- `1` 或 `熟练`：记得很清楚，间隔 7 天
- `2` 或 `一般`：记得但不熟练，间隔 3 天
- `3` 或 `忘记`：忘记了部分内容，间隔 1 天
- `4` 或 `完全忘记`：完全不记得，间隔 2 小时

# q
如果想按主题组织问题（如基础问题、工作知识、面试题），如何配置多个问题目录？
# a
编辑 `question_input` 文件（JSON 格式），设置 `"questions_dirs"` 列表，例如：
```json
{
  "questions_dirs": [
    "questions",
    "work_knowledge",
    "interviews"
  ]
}
```
每个目录下的 `.md` 文件都会被解析并导入知识库。
