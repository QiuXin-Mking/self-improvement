# q
快速幂算法的核心思想是什么？
# a
通过将指数不断二分，把幂运算的时间复杂度从 O(n) 降到 O(log n)。每次循环中将底数平方，指数减半，当当前指数为奇数时，将当前底数乘入结果。

# q
上述代码中 `mod_pow` 函数计算的是什么，为什么需要取模？
# a
计算 `(base^exp) % MOD`，其中 `MOD = 1e9 + 7`。取模为了防止中间结果溢出，并满足算法题中要求返回大数取模后的结果，保证结果在整型范围内。

# q
快速幂算法中，为什么在 `exp % 2 == 1` 时将 `result` 乘以当前 `base`？
# a
当指数为奇数时，将当前的底数乘入结果，相当于把该次幂拆分：`base^exp = base * base^(exp-1)`。随后指数减半（整除2）时，这个单独乘入的底数不会丢失。
