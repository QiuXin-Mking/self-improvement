# q
在反转链表 II（92题）中，为什么通常需要添加一个虚拟头节点（dummy node）？
# a
添加虚拟头节点可以统一处理头节点可能被反转的情况，避免单独判断头节点是否发生变化，简化边界处理逻辑。

# q
在寻找链表中间节点（876题）时，两种快慢指针循环条件的差异是什么？
# a
- 条件 `while (fast->next != nullptr && fast->next->next != nullptr)`：快指针确保能走两步时才移动，停止时快指针在最后一个节点或倒数第二个节点。  
  链表节点数为奇数时，慢指针指向正中间；链表节点数为偶数时，慢指针指向第一个中间节点（靠左）。
- 条件 `while (fast != nullptr && fast->next != nullptr)`：快指针能走一步就尝试走，停止时快指针可能为 `nullptr`。  
  链表节点数为奇数时，慢指针指向正中间；链表节点数为偶数时，慢指针指向第二个中间节点（靠右，LeetCode 876 的默认要求）。
