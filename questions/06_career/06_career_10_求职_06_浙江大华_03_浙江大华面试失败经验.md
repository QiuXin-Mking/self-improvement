# q
线程和进程在资源共享方面有什么区别？
# a
同一个进程内的线程之间资源共享；不同进程之间资源不共享。

# q
在 Linux 中如何查看当前 shell 的栈空间大小限制？
# a
使用命令 `ulimit -s`，输出单位为 KB。

# q
如何临时修改当前 shell 的栈空间大小限制？
# a
使用命令 `ulimit -s <stack_size_limit>`，其中 `<stack_size_limit>` 为想要设置的栈大小，单位通常为 KB。

# q
如何在 Linux 系统中永久修改所有用户的栈空间大小限制？
# a
编辑 `/etc/security/limits.conf` 文件，添加类似以下的行：
```
hard stack <stack_size_limit>
soft stack <stack_size_limit>
```
其中 `<stack_size_limit>` 是以 KB 为单位的栈大小限制值。保存后重启系统生效。
