# q
C++ STL中string相比char数组的主要优势是什么？
# a
string 封装了常用方法，支持动态变化的空间大小，减少了不必要的内存开销，并提供了方便的字符串操作（如插入、删除、截取、比较等）。

# q
使用 `cin` 和 `getline` 读取字符串有什么区别？
# a
- `cin >> s` 读取到空格或换行符前结束，且会自动跳过前导的空白字符，因此不会得到空串。
- `getline(cin, s)` 读取整行直到遇到换行符，会保留空格，但不会自动跳过前导换行符，可能因缓冲区残留的换行导致读取到空串。
- 混用时，若 `cin` 后使用 `getline`，需先用 `getline` 或 `cin.ignore()` 吃掉残留的换行符。

# q
string 常用的增删查改成员函数有哪些？并说明其时间复杂度。
# a
- `size()`：获取字符串长度。
- `insert(pos, str)`：在指定位置插入字符串。
- `append(str)` / `+=`：在末尾追加字符串。
- `erase(pos, n)`：删除从 pos 开始的 n 个字符，时间复杂度 O(n)。
- `substr(pos, len)`：截取子串。
- `back()`：获取最后一个字符。
- `pop_back()`：删除最后一个字符，时间复杂度 O(1)。
- `empty()`：判断是否为空。
- `clear()` 或赋空串 `s=""` 清空字符串。

# q
C++ 中如何比较两个 string 对象的大小？举例说明“bbc”与“bbbc”的比较结果。
# a
string 的比较按字典序逐个字符比较 ASCII 码。
- 先比较第一个字符，若不等则直接决定大小；若相等则继续比较下一个字符。
- 若短字符串所有字符与长字符串前缀相同，则短字符串较小。
例如 `"bbc"` 与 `"bbbc"`：前两个字符相同，第三个字符 `'c'` vs `'b'`，`'c' > 'b'`，因此 `"bbc" > "bbbc"`。

# q
string 有哪两种基本的遍历方式？请给出示例。
# a
- **下标遍历**：通过 `s[i]` 随机访问，循环 `for(int i=0; i<s.size(); ++i) cout << s[i];`
- **范围 for 遍历**：`for(char c : s) cout << c;`
