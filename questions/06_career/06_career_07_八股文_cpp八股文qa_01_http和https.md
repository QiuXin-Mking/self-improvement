# q
HTTPS中的“S”代表什么含义？
# a
表示安全（Secure），即通过SSL/TLS协议对传输数据进行加密。

# q
HTTP和HTTPS在数据传输上的核心区别是什么？
# a
HTTP传输不加密（明文），而HTTPS使用SSL/TLS加密，保护数据安全。

# q
HTTP和HTTPS分别使用哪些默认端口？
# a
HTTP默认使用80端口，HTTPS默认使用443端口。

# q
HTTPS是否需要SSL证书？
# a
需要，HTTPS必须配置SSL证书来实现加密和身份验证；HTTP不需要。

# q
HTTPS的性能与HTTP相比如何？为什么？
# a
HTTPS性能略低于HTTP，因为增加了SSL/TLS握手和加密解密的处理开销。
