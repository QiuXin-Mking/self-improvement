# q
下列哪个IP地址可以分配给一台计算机()

A. 256.1.3.4

B. 197.3.11.0

C. 199.5.89

D. 11.15.33.235
# a
正确答案：D

解析：

选项A错误：256.1.3.4
- 第一个字节256超过255，IPv4地址每个字节范围是0-255
- 无效IP地址

选项B错误：197.3.11.0
- 格式有效，但.0通常是网络地址（在/24子网中）
- 不能分配给主机使用

选项C错误：199.5.89
- 只有3个字节，IPv4地址必须包含4个字节
- 格式不完整

选项D正确：11.15.33.235
- 格式正确：4个字节，每个字节在0-255范围内
- 可以分配给主机使用

关键知识点：
- IPv4地址由4个字节组成，每个字节范围0-255
- 网络地址（如.0）和广播地址不能分配给主机
- 只有有效格式且非保留地址才能分配给计算机

# q
IPv4地址的格式要求是什么？
# a
IPv4地址格式要求：

1. 字节数量
- 必须包含4个字节（4个八位组）
- 用点号（.）分隔

2. 字节范围
- 每个字节范围：0-255
- 不能超过255

3. 格式示例
```
正确：192.168.1.1
错误：192.168.1    （只有3个字节）
错误：256.1.1.1    （第一个字节超过255）
```

关键点：
- 4个字节，用点分隔
- 每个字节0-255
- 格式：X.X.X.X（X为0-255）

# q
哪些IP地址不能分配给主机？
# a
不能分配给主机的IP地址：

1. 网络地址
- 主机位全为0的地址
- 例如：192.168.1.0（在/24子网中）

2. 广播地址
- 主机位全为1的地址
- 例如：192.168.1.255（在/24子网中）

3. 无效格式
- 字节数不是4个
- 字节值超过255
- 格式不完整

4. 特殊保留地址
- 127.0.0.1（本地回环）
- 0.0.0.0（未指定地址）
- 255.255.255.255（广播地址）

关键点：
- 网络地址和广播地址不能分配给主机
- 只有主机地址可以分配给计算机
- 需要根据子网掩码判断网络地址和广播地址

# q
如何判断IP地址是否有效？
# a
判断IP地址有效性的步骤：

1. 格式检查
- 是否包含4个字节
- 是否用点号分隔
- 每个字节是否为数字

2. 范围检查
- 每个字节是否在0-255范围内
- 不能有负数或超过255的值

3. 用途检查
- 是否为网络地址（主机位全0）
- 是否为广播地址（主机位全1）
- 是否为保留地址

示例：
```c
// 有效：11.15.33.235 ✅
// 无效：256.1.3.4    ❌（超过255）
// 无效：199.5.89     ❌（只有3个字节）
// 无效：197.3.11.0   ❌（网络地址）
```

关键点：
- 格式正确（4个字节，0-255）
- 非网络地址和广播地址
- 非保留地址
