# q
什么是Lustre的渐进式文件布局（PFL）？
# a
渐进式文件布局（Progressive File Layout, PFL）是一种根据文件大小动态调整条带参数的机制。文件较小时采用保守的条带配置，文件增长后自动切换到高性能的宽条带配置，以兼顾小文件效率和大文件高带宽。例如布局可设置为：0-128MB区间条带宽度1、条带大小1MB；128MB-1GB区间条带宽度4、条带大小4MB；1GB-8GB区间条带宽度16、条带大小8MB；8GB以上区间条带宽度32、条带大小8MB。

# q
与普通条带化相比，PFL的核心优势是什么？
# a
普通条带化对小文件会造成资源浪费和碎片多的问题，对大文件优化较好。PFL解决了小文件条带分散、资源浪费的问题，同时在文件增大时自动切换到宽条带，利用集群高带宽，实现小文件和大文件的最佳性能。

# q
在示例PFL布局中，文件大小达到128MB时条带参数如何变化？
# a
文件大小达到128M后，条带宽度从1变为4，条带大小从1MB变为4MB，从而增加并发度，提升写入性能。
