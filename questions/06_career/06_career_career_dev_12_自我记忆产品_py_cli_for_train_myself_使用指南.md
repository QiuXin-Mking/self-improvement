# q
问题文件的格式要求是什么？
# a
在 `questions/` 目录下创建 `.md` 文件，每个问答对按以下格式：
```

# q
问题内容
# a
答案内容
```
每个 `# q` 和 `# a` 独占一行，后跟内容。

# q
如何初始化知识库？
# a
运行命令 `python train.py --init`，它会扫描配置中所有目录下的 `.md` 文件，提取问题和答案。

# q
训练过程中如何查看答案以及反馈记忆程度的选项有哪些？
# a
输入 `a` 或 `answer` 查看答案；反馈选项：
- `1` 或 `熟练` → 记得很清楚
- `2` 或 `一般` → 记得但不熟练
- `3` 或 `忘记` → 忘记了部分内容
- `4` 或 `完全忘记` → 完全不记得

# q
不同反馈对应的复习间隔是如何安排的？
# a
- 熟练 → 7天后复习
- 一般 → 3天后复习
- 忘记 → 1天后复习
- 完全忘记 → 2小时后复习
连续正确回答后复习间隔会自动延长。

# q
如何查看训练统计数据？
# a
运行 `python train.py --stats`。
