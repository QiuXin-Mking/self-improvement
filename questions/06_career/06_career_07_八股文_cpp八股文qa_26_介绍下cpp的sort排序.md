# q
C++ 的 std::sort 内部主要采用哪三种排序算法？
# a
一种混合排序算法：
- 快速排序（Quick Sort）为主
- 小区间用插入排序（Insertion Sort）优化
- 递归深度过深时用堆排序（Heap Sort）

# q
在 std::sort 中，元素数量达到什么条件时会转而使用插入排序？
# a
当元素数量少于等于一定数量（典型值为 16 个）时，使用插入排序。

# q
在 std::sort 中，递归深度达到什么阈值会切换为堆排序？
# a
当递归深度大约超过 log₂N 的 2 倍时，改用堆排序以避免快排的最坏情况。

# q
如何用一句话记忆 std::sort 的混合排序策略？
# a
“小用插入，多用快排，太深用堆排。”
