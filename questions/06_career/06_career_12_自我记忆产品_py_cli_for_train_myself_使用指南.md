# q
如何准备训练用的问答文件？
# a
在配置的问题目录（默认为 `questions/`）下创建 `.md` 文件，每个文件内使用 `# q` 和 `# a` 标记问题和答案，格式如下：
```markdown

# q
你的问题内容
# a
你的答案内容
```

# q
如何初始化知识库？
# a
运行 `python train.py --init`，程序会扫描所有配置目录下的 `.md` 文件并提取问答对。

# q
每日训练的基本流程和反馈选项是什么？
# a
运行 `python train.py`，程序显示问题，你在纸上默写答案；输入 `a` 或 `answer` 查看答案；然后根据记忆程度输入反馈：
- `1` 或 `熟练` —— 记得很清楚
- `2` 或 `一般` —— 记得但不熟练
- `3` 或 `忘记` —— 忘记了部分内容
- `4` 或 `完全忘记` —— 完全不记得
系统会根据反馈自动安排下次复习时间。

# q
系统依据艾宾浩斯遗忘曲线对不同记忆程度安排的复习间隔是什么？
# a
复习间隔如下：
- 熟练 → 7天后复习
- 一般 → 3天后复习
- 忘记 → 1天后复习
- 完全忘记 → 2小时后复习
连续正确回答后复习间隔会自动延长。
