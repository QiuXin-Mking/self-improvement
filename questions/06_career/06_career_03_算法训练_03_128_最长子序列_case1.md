# q
这段代码中“unodered_set”错误的典型根因是什么？
# a
拼写错误，正确的容器名称为 `unordered_set`（注意中间是“ordered”，不是“odered”）。

# q
如何修正使用 `append` 向 `unordered_set` 添加元素的错误？
# a
`unordered_set` 没有 `append` 成员函数，应使用 `insert` 替代。`append` 通常属于字符串或顺序容器的操作。  
正确写法：`set_a.insert(num);`

# q
解决这段代码中“else { continue; }”错误的标准流程是什么？
# a
直接移除多余的 `else { continue; }` 分支。该分支会打断正常循环逻辑，去掉后即可让代码按预期继续执行。
