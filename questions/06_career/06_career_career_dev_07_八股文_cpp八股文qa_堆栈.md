# q
栈和堆在数据结构上有何区别？
# a
栈是一种线性数据结构，遵循后进先出（LIFO）原则；堆是一种二叉树结构，通常用于实现动态内存分配，按优先级管理数据项。

# q
栈的内存分配方式有什么特点？
# a
栈上的内存分配是静态的，由编译器自动管理。函数调用时局部变量和参数被分配到栈上，函数返回后自动释放。

# q
堆的内存分配方式有什么特点？
# a
堆上的内存分配是动态的，需要程序员手动申请和释放，通常使用 `malloc`/`free` 或 `new`/`delete` 等函数或语言特性进行管理。

# q
栈和堆上数据的生存周期有何不同？
# a
栈数据的生命周期与其所在作用域相同，退出作用域后自动销毁；堆数据可存在于整个程序生命周期，直到显式释放该内存。

# q
栈和堆在分配效率和大小限制上有何区别？
# a
栈分配和释放极快，由编译器自动完成，但大小通常固定，可能引发栈溢出；堆分配较慢，涉及手动管理及可能的内存碎片，其大小可根据需要动态增长，但受系统内存限制。
