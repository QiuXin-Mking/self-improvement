# q
std::vector 底层采用的是什么数据结构？
# a
std::vector 底层采用的是**动态数组（dynamic array）**，内部维护一块连续内存，支持动态增长。

# q
std::vector 的容量扩容机制是怎样的？
# a
当插入元素后元素数量等于或大于当前容量时，会重新分配一块更大的连续内存（通常扩容至当前容量的 2 倍或 1.5 倍，具体依赖实现），将原有元素移动到新内存，然后释放旧内存。扩容会导致迭代器失效，因为底层内存地址可能改变。

# q
std::vector 随机访问和尾部插入的时间复杂度分别是多少？
# a
随机访问（`operator[]`、`at()`）时间复杂度为 O(1)。尾部插入（`push_back`）在不触发扩容时为 O(1)，触发扩容时整体摊销时间复杂度为 O(1)。

# q
在 GCC 的 libstdc++ 实现中，std::vector 内部通常使用哪三个核心指针？
# a
使用 `T* _start`（起始指针）、`T* _finish`（尾后指针，指向最后一个元素之后的位置）和 `T* _end_of_storage`（容量尾后指针，指向分配内存的末尾）。`size()` 返回 `_finish - _start`，`capacity()` 返回 `_end_of_storage - _start`。
