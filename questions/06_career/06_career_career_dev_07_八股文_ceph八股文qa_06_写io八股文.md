# q
核桃书中，写io有哪四种？
# a
覆盖写、对齐写、非对齐写、追加写

# q
Linux文件系统架构分为哪几层？各层主要包含什么？
# a
用户态：应用程序；内核态：VFS、页面缓存、文件系统、IO驱动；硬件层：机械盘、SSD

# q
POSIX对于删除操作的语义是什么？以覆盖写为例说明。
# a
truncate；需要先把文件truncate为0，表示删除文件原来所有的数据，再写入新数据

# q
VFS的4个构成元素是什么？
# a
inode、dentry、superblock、file

# q
inode的作用是什么？
# a
inode负责记录每个文件的元数据信息（在Linux系统中一切皆文件，目录和设备都会封装为文件一样调用）
