# q
如何启动知识库学习系统的Web版本？
# a
在安装依赖后执行 `python web_app.py`，默认在 `http://0.0.0.0:5000` 启动服务器，即可通过同一局域网内的设备访问。

# q
在手机上使用Web版本时，如何找到并输入正确的访问地址？
# a
手机与电脑连接同一WiFi，在手机浏览器中输入 `http://你的电脑IP地址:5000`（例如 `http://192.168.1.100:5000`）；电脑IP地址可通过 `ipconfig`（Windows）或 `ifconfig`/`ip addr`（Mac/Linux）查看。

# q
学习过程中可以给出哪些记忆程度反馈？系统会如何处理？
# a
可选的反馈包括✅熟练、👍一般、😐忘记、❌完全忘记；系统会根据反馈自动调整下次复习时间，并立即进入下一题。

# q
Web版本的复习安排基于什么原理？数据存储与命令行版本的关系是什么？
# a
复习安排基于艾宾浩斯遗忘曲线，根据记忆反馈自动调整间隔；数据存储在与命令行版本共享的 `data/learning_data.json` 文件中，可混用。
