# q
如何停止并删除一个 Docker 容器？
# a
使用 `docker stop <容器名>` 停止容器，再使用 `docker rm <容器名>` 删除容器。例如：
```bash
docker stop vue3-dev
docker rm vue3-dev
```

# q
在 `docker run` 命令中如何挂载多个宿主机目录到容器内？
# a
通过多次使用 `-v` 参数指定挂载，格式为 `-v "宿主机路径:容器内路径"`。例如：
```bash
docker run -itd \
  --name vue3-dev \
  -p 5173:5173 \
  -v "C:\Users\Administrator\Desktop\vue3-demo:/app" \
  -v "D:\qiuxin_aliyun_back\04_职场修炼\12_自我记忆产品\Self-improvement:/Self-improvement" \
  -w /app \
  node:20 \
  bash
```

# q
如何给已有容器添加额外的端口映射？
# a
先使用 `docker commit` 将当前容器保存为新镜像，然后基于该镜像重新运行容器时添加新端口。例如：
```bash
docker commit vue3-dev vue3-dev-with-deps
docker stop vue3-dev
docker run -d \
  --name vue3-dev-new \
  -p 5173:5173 \
  -p 5000:5000 \
  -p 3000:3000 \
  -v "C:\Users\Administrator\Documents\vue3-demo:/app" \
  -v "D:\qiuxin_aliyun_back\04_职场修炼\12_自我记忆产品\py_cli_for_train_myself:/app/Self-improvement" \
  vue3-dev-with-deps \
  tail -f /dev/null
```

# q
如何让 Docker 容器启动后保持运行而不退出？
# a
将容器的主进程设置为 `tail -f /dev/null`，该命令会持续运行并阻塞，从而保持容器存活。
