# q
什么是分类？它与聚类有什么区别？
# a
**分类（Classification）**：是监督学习的一种，目标是找出一个模型或函数来描述和区分不同数据类，并用该模型预测未知对象所属的类别。

**与聚类的区别：**
- **分类**：有监督学习，训练数据带有类别标签，目标是预测未知数据的类别。
- **聚类**：无监督学习，训练数据没有类别标签，目标是发现数据的内在分组结构。

# q
什么是孤立点？孤立点为什么不是无用的？
# a
**孤立点（Outlier）**：指与数据的一般行为或模式不一致的数据对象，也称为异常值或离群点。

**孤立点不是无用的原因：**
1. **异常检测**：发现设备故障、系统入侵等异常情况。
2. **欺诈检测**：识别信用卡欺诈、保险欺诈等异常交易。
3. **数据质量**：发现数据录入错误、传感器故障等问题。
4. **业务洞察**：发现特殊客户行为、市场异常、罕见事件等，孤立点往往包含比正常数据更有价值的信息。

# q
数据挖掘中的分类、聚类和孤立点检测分别属于什么类型的任务？
# a
- **分类（Classification）**：属于**监督学习**任务，需要带标签的训练数据。
- **聚类（Clustering）**：属于**无监督学习**任务，不需要带标签的训练数据。
- **孤立点检测（Outlier Detection）**：可以是监督学习或非监督学习，但通常是**无监督学习**任务。

| 任务类型 | 学习类型 | 训练数据要求 | 主要目标 |
|---------|---------|-------------|---------|
| 分类 | 监督学习 | 需要标签 | 预测未知对象类别 |
| 聚类 | 无监督学习 | 不需要标签 | 发现数据分组结构 |
| 孤立点检测 | 通常为无监督学习 | 通常不需要标签 | 识别异常数据 |
