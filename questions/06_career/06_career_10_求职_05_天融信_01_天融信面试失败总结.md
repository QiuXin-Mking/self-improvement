# q
40亿个不重复的unsigned int整数，如何用少量内存快速判断一个给定数是否存在？
# a
使用位图（bitmap），每个数字用一个bit表示是否存在，约需40亿bit（约476.83 MB内存）。构建位图时间复杂度O(n)，查找时间复杂度O(1)。具体计算：40*100000000 / 1024 / 1024 / 8 ≈ 476.83 MB。

# q
对于100GB的IP日志文件，在只有8GB内存的服务器上，如何找出重复出现次数最多的前10个IP？
# a
采用分治思想。设计一个哈希函数，将文件分成约1000份，每份约100MB可加载入内存。分别统计每份中出现次数最多的前10个IP。最后从所有分片的top10（共约1000*10个IP）中再次统计，找出全局出现次数最多的前10个IP。
