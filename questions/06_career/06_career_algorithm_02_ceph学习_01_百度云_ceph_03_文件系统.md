# q
常见的Linux文件系统有哪些？
# a
ext2、ext3、ext4、xfs。

# q
索引节点（inode）中主要存储哪些文件信息？
# a
innode编号、文件大小、访问权限、创建时间等。

# q
磁盘分区通常包含哪几个区域？
# a
超级块（供操作系统使用）、索引节点区（存放索引节点及文件数据信息）、数据块区（存放文件和目录数据）。

# q
文件存储的两种方式是什么？各有什么优缺点？
# a
连续空间存储和非连续空间存储。连续存储读写效率高但删除文件后容易产生磁盘碎片；非连续存储（类似链表）可消除碎片，但查找效率低，需要额外资源消耗。

# q
DAS、NAS、SAN三种存储架构中，哪些自带文件系统？
# a
NAS（网络附加存储）自有文件系统，DAS（直连式存储）和SAN（存储区域网络）没有文件系统。
