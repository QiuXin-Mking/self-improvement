# q
如何初始化基于艾宾浩斯遗忘曲线的知识库学习系统，扫描所有配置目录下的Markdown文件？
# a
运行 `python train.py --init`，程序会扫描 `question_input` 中配置的所有问题目录（`questions_dirs`）下的 `.md` 文件，提取 `# q` 和 `# a` 格式的问题与答案，存入知识库。

# q
训练过程中，用户如何查看当前问题的标准答案，并用哪种方式反馈自己的记忆程度？
# a
输入 `a` 或 `answer` 查看标准答案；反馈记忆程度时输入：
- `1` 或 `熟练` 表示记得很清楚
- `2` 或 `一般` 表示记得但不熟练
- `3` 或 `忘记` 表示忘记了部分内容
- `4` 或 `完全忘记` 表示完全不记得

# q
根据艾宾浩斯遗忘曲线算法，当用户反馈为“熟练”或“一般”时，初始复习间隔和后续递增系数分别是多少？
# a
- **熟练**：7天后复习，正确后间隔递增 2.5 倍
- **一般**：3天后复习，正确后间隔递增 1.8 倍

# q
如何为系统配置多个不同的Markdown问题目录，以实现按主题组织知识？
# a
编辑 `question_input` 文件（如 `question_input` 文件），使用 JSON 格式指定 `questions_dirs` 数组，例如：
```json
{
  "questions_dirs": [
    "questions",
    "my_questions",
    "work_knowledge"
  ]
}
```
每个目录下放置符合 `# q` / `# a` 格式的 `.md` 文件即可。

# q
学习数据和复习历史自动保存到哪个文件，包含了哪些关键信息？
# a
学习数据保存在 `data/learning_data.json`，包含所有问题和答案、复习历史、记忆等级、下次复习时间以及学习统计。
