# q
Python 列表的 `remove` 方法有什么作用？执行 `lists = [1, 1, 1, 3, 5]; lists.remove(1)` 后，列表内容是什么？
# a
`remove` 方法会移除列表中指定值的**第一个匹配项**。执行后列表变为 `[1, 1, 3, 5]`。

# q
Python 列表的 `index` 方法的功能是什么？接上例，`lists.index(1)` 的返回值是什么？
# a
`index()` 方法返回列表中指定值**第一个匹配项的索引位置**。接上例，列表变为 `[1, 1, 3, 5]` 后，第一个 `1` 的索引是 `0`，因此返回 `0`。
