# q
如何初始化间隔重复学习系统的知识库？
# a
运行 `python train.py --init`，系统会递归扫描配置目录中的所有 `.md` 文件，解析问答对并去重后存入 `data/learning_data.json`。

# q
在艾宾浩斯遗忘曲线算法中，四个记忆级别（熟练/一般/忘记/完全忘记）的复习间隔和正确回答后的间隔乘数分别是多少？正确率超过80%时有何额外加成？
# a
- Level 1（熟练）：间隔 168 小时（7天），乘数 2.5  
- Level 2（一般）：间隔 72 小时（3天），乘数 1.8  
- Level 3（忘记）：间隔 24 小时（1天），乘数 1.3  
- Level 4（完全忘记）：间隔 2 小时，乘数 1.0  
当准确率 > 80% 时，额外乘以 1.2。

# q
系统如何通过问题 ID 实现去重？
# a
使用 `q_{hash(question.strip())}` 生成唯一 ID，相同问题文本（忽略首尾空白）始终生成相同 ID，初始化添加问题时自动去重。

# q
Markdown 问题文件需要遵循什么格式？
# a
使用 `# q` 独占一行后跟问题，`# a` 独占一行后跟答案，一个文件可包含多对，例如：
```markdown

# q
什么是...？
# a
答案是...
```

# q
如何将项目构建为独立的可执行文件或使用 Docker 部署？
# a
- 独立可执行文件：运行 `python build.py`，通过 PyInstaller 生成到 `dist/`，部署包到 `deploy/`
- Docker 部署：`docker-compose up -d`，使用 Python 3.11-slim 镜像，默认暴露 5000 端口
