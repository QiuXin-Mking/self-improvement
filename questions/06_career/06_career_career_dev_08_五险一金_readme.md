# q
五险一金具体包括哪些保险和公积金？
# a
五险一金包括：
- 医疗保险：报销比例约 50%，一般要求缴纳 20-25 年可享受终身医保，断缴小于 3 个月次月恢复
- 养老保险：需累计缴纳 15 年，断缴可累计计算
- 失业保险：缴满一年，被裁员可领取失业金（如每月约两千元）
- 工伤保险：覆盖上下班途中的伤害
- 生育保险：用于产检和生产费用
- 住房公积金：由公司和个人共同缴存，可用于购房贷款，利息较低

# q
职工医保转为灵活就业医保需要注意什么？
# a
职工医保可转为灵活就业医保，需先办理停保，并满足户籍或居住条件。缴费压力可能降低，但要注意待遇衔接，避免断保导致保障中断。建议根据个人经济状况和医疗需求选择，并优先咨询当地医保部门获取精准政策指导。

# q
养老保险如果断缴是否影响累计年限？
# a
养老保险断缴不会清零，已缴年限可以累计计算，只要累计缴满 15 年，达到退休年龄后即可领取养老金。
