# q
Google的三驾马车是指哪三项核心技术？
# a
Google的三驾马车指：Google File System (GFS)、MapReduce 和 Bigtable。它们是支撑Google大规模数据处理的基础设施。

# q
Google File System（GFS）论文解决的问题是什么？
# a
GFS介绍了一种可扩展的分布式文件系统，专门为大规模数据密集型应用设计。

# q
MapReduce论文的核心贡献是什么？
# a
MapReduce描述了一种编程模型及实现，用于在大型集群上并行处理大规模数据集，简化了分布式计算。

# q
Bigtable是什么类型的存储系统？
# a
Bigtable是一个面向列的分布式存储系统，能够处理非常大的结构化数据集。
