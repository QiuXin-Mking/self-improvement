# q
该学习系统根据反馈设置了哪四种记忆程度？分别对应的初始复习间隔是多少？
# a
- **熟练**：7天后复习
- **一般**：3天后复习
- **忘记**：1天后复习
- **完全忘记**：2小时后复习

# q
如何在首次使用时初始化该系统的知识库？
# a
运行命令 `python train.py --init`，将扫描配置的所有目录下的 `.md` 文件，提取问题和答案到知识库。

# q
当熟练程度反馈为“熟练”且连续正确回答后，系统的间隔调整策略是什么？
# a
正确后间隔递增 2.5 倍，系统会自动提高记忆等级，延长复习间隔。

# q
如何配置多个不同主题的问题目录？
# a
编辑 `question_input` 文件，在 `"questions_dirs"` 数组中添加目录路径，例如：
```json
{
  "questions_dirs": [
    "questions",
    "work_knowledge"
  ]
}
```
