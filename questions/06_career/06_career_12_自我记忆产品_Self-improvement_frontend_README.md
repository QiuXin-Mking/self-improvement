# q
这个前端模板的技术栈是什么？
# a
Vue 3 + TypeScript + Vite，使用 `<script setup>` 单文件组件（SFC）语法。

# q
该模板使用的 Vue 3 单文件组件特性是什么？
# a
使用 `<script setup>` 语法糖，简化组件编写。详细信息可查阅 [script setup 文档](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup)。

# q
如果想了解推荐的项目设置和 IDE 支持，应该查看哪个指南？
# a
Vue 官方 TypeScript 指南中的 Project Setup 部分：[Vue Docs TypeScript Guide](https://vuejs.org/guide/typescript/overview.html#project-setup)。
