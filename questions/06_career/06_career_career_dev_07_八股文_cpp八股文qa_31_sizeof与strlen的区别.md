# q
sizeof 是什么，有什么特点？
# a
sizeof 是编译时运算符（操作符），用于返回类型或变量所占的内存字节数。它在编译期就能确定结果，与运行时内容无关，适用于内置类型、结构体、数组等。

# q
strlen 是什么，有什么特点？
# a
strlen 是 C/C++ 库函数（头文件 `<cstring>` 或 `<string.h>`），用于计算以 `\0` 结尾的字符串的实际字符个数（不包含末尾 `\0`）。它在程序运行时执行，仅适用于以 `\0` 结尾的字符串。

# q
sizeof 和 strlen 的核心区别是什么？
# a
sizeof 是编译时运算符，返回类型或变量占用的字节数，不依赖内容；strlen 是运行时库函数，计算字符串中字符数量（不含结束符 `\0`），仅用于 C 风格字符串。
