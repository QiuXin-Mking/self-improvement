# q
vector 在什么条件下会触发扩容？
# a
当插入元素时，如果当前容量（`capacity()`）不足以容纳新元素，就会触发扩容。例如调用 `push_back()` 且在 `size() == capacity()` 时插入，就会触发扩容。

# q
`reserve(n)` 和 `resize(n)` 在影响容量方面有何区别？
# a
- `reserve(n)`：仅预留至少能容纳 n 个元素的容量，不改变元素数量（`size()` 不变）。如果 n > 当前容量，则会重新分配内存并扩容。
- `resize(n)`：调整容器中的元素数量到 n。如果 n 大于当前 `capacity()`，则会先扩容，随后插入/删除元素以匹配新的大小。
