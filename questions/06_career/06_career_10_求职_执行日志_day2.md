# q
分布式系统中“可扩展性（Scalability）”的定义是什么？
# a
可扩展性是分布式系统应对需求增长（如用户量、数据量、业务负载增加）的核心能力，指系统通过高效、低成本的方式（如添加资源、调整架构）提升性能或容量，且不显著影响现有服务质量（如响应时间、吞吐量）的能力。

# q
垂直扩展（Vertical Scalability）与水平扩展（Horizontal Scalability）的主要区别是什么？
# a
垂直扩展是通过增强单个节点的硬件性能来提升系统容量；水平扩展是通过在分布式集群中新增服务器节点来提升系统容量。

# q
实现可扩展性需要满足哪些关键原则？
# a
成本可控，运维复杂度低，可用性不中断。
