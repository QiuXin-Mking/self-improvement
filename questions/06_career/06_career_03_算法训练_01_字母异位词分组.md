# q
在LeetCode字母异位词分组问题中，如何将互为异位词的字符串归一化为哈希表的键？
# a
对每个字符串的字符排序后拼接，如 `"eat"` 排序后得到 `['a','e','t']`，通过 `"".join` 生成键 `"aet"`，所有排序后相同的字符串即为同一组异位词。

# q
C++中 `std::unordered_map` 如何插入键值对？如果键已存在，`operator[]` 和 `insert` 有何区别？
# a
- `mp[key] = value;`：若键不存在则插入，若存在则覆盖值。
- `mp.insert({key, value});`：若键已存在则不插入，保持原值不变。
- `mp.emplace(key, value);`：就地构造元素，避免拷贝，键存在时不覆盖。

# q
Python中 `collections.defaultdict(list)` 在字母异位词分组中起什么作用？
# a
`defaultdict(list)` 会为不存在的键自动创建一个空列表作为值。例如 `mp["aet"]` 不存在时直接返回 `[]`，无需手动判断 `if key not in mp`，然后直接 `mp[key].append(st)` 将原字符串加入对应分组。最终 `list(mp.values())` 返回所有分组列表。
