# q
LSM-Tree 论文的核心思想是什么？为什么它对 SSD/闪存友好？
# a
LSM-Tree 将通过将随机写转换为顺序写来优化写入性能，这正好匹配了 SSD 的特性。它是 LevelDB、RocksDB、Cassandra、HBase 等现代存储引擎的基石。

# q
《Strata》论文提出了什么新颖的存储架构？
# a
Strata 是为持久内存（如 Intel Optane DC PMM）和传统 NAND SSD 构成的异构存储层次设计的跨介质文件系统。它不把 PM 当作更快的块设备，而是采用一种“日志分层”方法，构建跨 PM 和 SSD 的文件系统。

# q
《FaRM》论文的主要贡献是什么？
# a
FaRM 展示了如何利用 RDMA 网络绕过内核和 CPU，直接在远程机器的内存中操作，将分布式事务和一致性协议的开销降到极低。它是构建超低延迟分布式存储系统的里程碑，其思想被广泛应用于 Crail、HERD 等后续系统。

# q
《LegoOS》论文提出的分解式操作系统架构是怎样的？
# a
LegoOS 把传统的单机操作系统分解为处理器、内存、存储等独立的服务组件，并通过网络组合在一起，实现硬件资源的分解与聚合，为云原生、超大规模的计算和存储基础设施提供了全新设计视角。

# q
Facebook 的《f4》论文主要讨论了什么内容？
# a
f4 是 Facebook 为其海量“温数据”（不常访问但需长期保存的图片、视频等）设计的低成本、高可靠对象存储系统。论文详细讨论了 Erasure Coding（纠删码）在大规模生产环境中的实践和挑战，是大规模存储系统中理论与工程结合的范例。
