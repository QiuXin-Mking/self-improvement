# q
Linux 一个进程的固定线性内存空间由哪些主要段构成？
# a
大致分为：
- `.text` 段：程序的机器指令及编译时静态链接的库代码
- `.rodata` 段：常量字符串等只读内容
- `.data` 段：已初始化的全局变量
- `.bss` 段：未初始化的全局变量
- 堆和栈区域

# q
BSS 段存储什么内容？有什么主要特点？
# a
存储未初始化的全局变量。在程序执行之前，BSS 段内容会自动清零。这样做的目的是为了节省可执行文件所占的磁盘空间（因为在目标文件或可执行文件中不需要真正为这些变量分配空间）。

# q
已初始化的全局变量存放在哪个段？
# a
存放在 `.data` 段。
