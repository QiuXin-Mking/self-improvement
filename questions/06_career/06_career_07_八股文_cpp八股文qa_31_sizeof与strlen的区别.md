# q
sizeof 是什么？它的执行时机和返回值的含义是什么？
# a
sizeof 是 C/C++ 中的一个编译时运算符（操作符）。它在编译期即可确定结果，返回**类型或变量所占的字节数**，与运行时的内容无关。适用于内置类型、结构体、数组等。

# q
strlen 是什么？它的执行时机和计算规则是什么？
# a
strlen 是 C/C++ 标准库函数（头文件为 `<cstring>` 或 `<string.h>`）。它需要**运行时执行**才能得出结果，计算的是以 `\0` 结尾的字符串中**实际字符的个数**，**不包括末尾的 `\0`**。只适用于以 `\0` 结尾的字符串。

# q
sizeof 和 strlen 的核心区别有哪些？
# a
- **性质**：sizeof 是运算符，strlen 是库函数。
- **执行阶段**：sizeof 在**编译期**求值，strlen 在**运行时**求值。
- **用途**：sizeof 测量**类型或变量所占的内存空间大小**（字节数），strlen 计算**字符串的字符数**（不含结尾 `\0`）。
- **适用范围**：sizeof 适用于各种类型（内置类型、结构体、数组等）；strlen **仅适用于以 `\0` 结尾的字符串**。
