# q
多线程中实现线程间交互的主要方式有哪些？
# a
- 共享变量：多个线程通过读写同一变量进行通信，需配合锁保证原子性。
- 条件变量：当线程需要等待特定条件时，主动释放锁并进入阻塞状态；其他线程修改条件后通过通知机制唤醒等待线程。

# q
条件变量在生产者-消费者模型中的作用是什么？
# a
生产者生产数据后通过条件变量通知消费者；消费者等待数据就绪，被通知后消费数据。其目的在于避免其他线程因锁被占用而无法修改共享条件。

# q
线程调用条件变量的 `wait(lock)` 时内部发生了什么？
# a
自动释放互斥锁（mutex）→ 线程进入阻塞状态，等待其他线程的 `notify_one` 或 `notify_all` → 被唤醒后重新竞争锁 → 成功获取锁后继续执行。即“释放锁 → 阻塞 → 唤醒 → 重获锁”的过程。
