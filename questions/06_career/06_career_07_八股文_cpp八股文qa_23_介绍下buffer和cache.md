# q
buffer 和 cache 在概念上的核心区别是什么？
# a
buffer（缓冲）是内存中的临时区域，用于在两个存在速度/处理节奏差异的系统之间暂存数据，典型“先攒后写”；cache（缓存）是内存中保存最近或高频访问数据副本的区域，用于加速后续读取，典型“读多写少、命中优先”。

# q
buffer 和 cache 分别侧重优化哪些 I/O 开销？
# a
buffer 侧重合并/整批 I/O、降低频繁小 I/O 引发的系统调用与寻道成本；cache 侧重加速读取、减少磁盘访问、提升命中率。

# q
在 Linux 的 `free` 输出中，`buffers` 和 `cached` 分别对应什么内容？
# a
`buffers` 偏向块设备缓冲，`cached` 偏向文件内容缓存，两者都属于“占用内存但可回收”的部分。
