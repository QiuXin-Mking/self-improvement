# q
Lambda表达式的基本语法是什么？
# a
Lambda表达式语法为：
```cpp
[捕获列表](参数列表) -> 返回类型 {
    // 函数体
}
```
例如：`auto add = [](int a, int b) -> int { return a + b; };`

# q
Lambda的捕获列表有哪些常见形式？各代表什么含义？
# a
- `[x]`：值捕获x，创建x的副本
- `[&x]`：引用捕获x，内部修改会影响外部
- `[=]`：隐式值捕获所有外部变量
- `[&]`：隐式引用捕获所有外部变量
- `[=, &x]`：默认值捕获，但x是引用捕获
- `[&, x]`：默认引用捕获，但x是值捕获
- `[]`：不捕获任何外部变量

# q
用代码举例说明Lambda值捕获和引用捕获的区别。
# a
```cpp
int x = 10;
auto by_val = [x]() { std::cout << x; }; // 内部是x的副本
auto by_ref = [&x]() { x += 100; };      // 内部是x的引用

by_ref();           // 外部x被修改为110
by_val();           // 仍输出10（原始副本）
```
