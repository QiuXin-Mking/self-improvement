# q
分类和聚类的主要区别是什么？
# a
**分类**（Classification）是监督学习，训练数据带有类别标签，目的是找出一个模型或函数来描述和区分不同数据类，并用该模型预测未知对象所属的类别。  
**聚类**（Clustering）是无监督学习，不需要预先知道类别标签，目的是将数据对象分组为多个类或簇，使同一个簇中的对象具有较高的相似度，不同簇之间相似度较低。  
两者核心区别在于：分类需要标签进行预测，聚类不需要标签，自动发现数据的内在分组结构。

# q
什么是孤立点？为什么孤立点反而是有价值的数据？
# a
孤立点（Outlier/Anomaly）是与数据的一般行为或模式不一致的数据对象，也称为异常值或离群点。  
孤立点并不是无用的数据，因为孤立点检测本身就是数据挖掘的重要任务，孤立点可能包含关键信息：
- **异常检测**：发现设备故障、系统入侵
- **欺诈检测**：识别信用卡欺诈、保险欺诈
- **数据质量**：发现数据录入错误
- **业务洞察**：发现特殊客户行为、市场异常  
因此，孤立点往往比正常数据更能揭示隐藏的问题或机会。

# q
分类、聚类和孤立点检测分别属于哪种学习类型？
# a
- **分类**（Classification）：属于监督学习，需要带标签的训练数据。
- **聚类**（Clustering）：属于无监督学习，不需要带标签的训练数据。
- **孤立点检测**（Outlier Detection）：通常是无监督学习（多数情况下没有标签），也可以是监督学习（当已知哪些是孤立点时）。
