# q
在实现“最长连续序列”时，使用 `set(num)` 初始化集合会导致什么错误？
# a
`set(num)` 中的变量 `num` 在此时尚未定义（或引用的是外层循环变量），导致集合只包含单个元素，而非输入列表 `nums` 的全部数字。正确写法应为 `set(nums)`。

# q
上述代码中 `while (num+1) in set_a` 的典型根因是什么？
# a
循环条件中的 `num` 在循环体内始终未被更新，导致死循环。应当使用临时变量（如 `cur`）追踪当前连续数，并检查 `cur + 1 in set_a`，每次循环将 `cur` 递增 1。示例修正片段：
```python
cur = num
while cur + 1 in set_a:
    cur = cur + 1
    lev = lev + 1
```
