# q
io_uring 是什么？它在哪个内核版本中引入？
# a
io_uring 是 Linux 内核自 5.1 版本引入的高性能异步 I/O 框架，旨在提高传统系统调用（如 epoll、aio）的并发性能并降低调用开销。

# q
io_uring 的轮询模式（polling mode）是如何工作的？
# a
轮询模式由内核分配一个线程持续主动扫描用户空间的 I/O 提交队列，使应用提交 I/O 时无需进行系统调用，从而避免频繁陷入内核的开销。

# q
io_uring 轮询模式适用于哪些场景？它带来了什么主要好处？
# a
适合高性能、高并发场景，可以换取极致的并发能力和低延迟。
