# q
公交车辆GPS定位的精度要求和位置信息更新频率分别是多少？
# a
定位精度需达到10米以内，系统每隔30秒更新一次公交车辆的位置信息。

# q
系统对数据采集到信息发布的延迟有何要求？
# a
从数据采集到信息发布的延迟必须控制在5秒以内。

# q
两块PCB（A和B）各自的核心功能是什么？
# a
PCB A：以esp32为主控，采集GPS信息并通过WiFi发送到云服务器。  
PCB B：以esp32为主控，通过WiFi向云服务器查询拥堵情况和到站时间，并控制屏幕显示相关信息。

# q
云服务器根据GPS和站点信息计算出的拥堵程度是如何分级的？
# a
拥堵程度分为4个等级（1、2、3、4），依据是否处于高峰时段判定得出。

# q
用户如何对云服务器上的后台系统进行维护和程序替换？
# a
用户通过SSH连接登录到云服务器，完成程序替换。
