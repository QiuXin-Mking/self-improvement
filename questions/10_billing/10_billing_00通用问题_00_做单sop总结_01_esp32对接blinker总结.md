# q
对接ESP32与Blinker的标准流程是什么？
# a
1. 找到ESP32母版代码文件（如 `18_fa_03_安防.ino`），将其拷贝到Arduino IDE中。
2. 在Blinker App中获取密钥，并将密钥拷贝到代码相应位置。
3. 检查并配置WiFi参数，通常使用默认家庭WiFi。
4. 根据实际需求确定所需的数字模组、滑块模组、按键控制模组数量。
5. 编写完代码后，将代码交给AI检查。
6. 通过ESP32母版进行测试，测试中需使用ESP32和CH340模块。

# q
ESP32对接Blinker时，如何获取要使用的密钥？
# a
在Blinker App的框架中找到密钥，然后将其拷贝到Arduino代码中。

# q
代码编写完成后，如何进行初步检查？
# a
将写好的代码交给AI进行检查，以发现潜在问题。

# q
ESP32母版测试需要哪些硬件？
# a
需要ESP32主板和CH340模块（通常用于USB转串口）进行测试。
