# q
数字地（DGND）和模拟地（AGND）在设计上有什么区别？
# a
数字地是数字信号的参考地，数字信号只有 0 和 1 两种电平，抗干扰能力相对较强；模拟地是模拟信号的参考地，模拟信号是随时间连续变化的，对干扰非常敏感，细微干扰就会表现为杂音或失真。

# q
为什么在电路设计中必须将数字地和模拟地分开？
# a
因为数字信号的开关动作会在地中产生地弹噪声，如果与模拟地混用，这些高频噪声会耦合到模拟信号中，导致模拟信号出现杂音、失真等问题，尤其经放大后影响更为严重。

# q
数字地与模拟地最终需要连接在一起时，常用哪两种连接方式？各有何优缺点？
# a
常用 0 欧姆电阻或磁珠连接。
- 0 欧姆电阻：能将两个地区分开，但仍会有部分数字高频谐波干扰引入模拟侧，不过比直接混用要好。
- 磁珠：能滤除部分数字信号的高频谐波，但磁珠存在一定的直流电阻，会在数字地与模拟地之间形成电位差。

# q
什么是一点接地和多点接地？
# a
一点接地：将同一性质（如同一类型信号）的地先各自集中连接在一起，然后通过一个公共点汇合，与其他地只在该点连接。
多点接地：所有地直接打孔连接到地层平面，地可以分布在电路板各处，彼此之间不需要单独再相互连接，全通过地层平面联通。
