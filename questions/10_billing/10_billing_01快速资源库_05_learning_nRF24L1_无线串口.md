# q
该无线串口模块在CH340 USB转串口模块下，5V和3.3V供电是否都能正常工作？
# a
测试结果表明，5V和3.3V供电均可正常收发信息。

# q
测试中发现该无线串口模块支持哪些波特率？9600波特率是否可用？
# a
测试中115200波特率可用，9600波特率不可用（直接乱码）。结论：使用波特率115200即可。

# q
在测试文档中，用于查询和设置模块参数的AT命令示例是什么？
# a
示例命令为：
```

AT?
```
