# q
如何使用 AWS CLI 创建一个新的 S3 桶？
# a
```bash
aws s3 mb s3://my-test-bucket-20240619
```

# q
如何查看当前 AWS CLI 会话所使用的身份信息？
# a
```bash
aws sts get-caller-identity
```
该命令返回 UserId、Account、Arn，例如 `assumed-role/voclabs/user3909028=HAOBO_ZHAO`，表明当前身份是通过 STS 临时令牌假设的 IAM 角色，而非根用户。

# q
在 S3 桶中如何创建一个类似“目录”的前缀对象？
# a
使用 `aws s3api put-object` 指定以 `/` 结尾的 Key：
```bash
aws s3api put-object --bucket buc-250516 --key thumbnails/
```

# q
如何为 S3 桶配置对象创建事件触发的 Lambda 通知（支持多个 Lambda）？
# a
使用 `aws s3api put-bucket-notification-configuration`，在 `LambdaFunctionConfigurations` 中设置事件和 ARN：
```bash
aws s3api put-bucket-notification-configuration \
    --bucket myapp-images-bucket \
    --notification-configuration '{
        "LambdaFunctionConfigurations": [
            {
                "LambdaFunctionArn": "你的Lambda-1-ARN",
                "Events": ["s3:ObjectCreated:*"]
            },
            {
                "LambdaFunctionArn": "你的Lambda-2-ARN",
                "Events": ["s3:ObjectCreated:*"]
            }
        ]
    }'
```
