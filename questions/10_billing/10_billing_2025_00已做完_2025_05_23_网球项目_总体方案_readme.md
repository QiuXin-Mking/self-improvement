# q
MQTT网球项目的公网IP和端口是什么？
# a
公网IP：101.201.53.141，端口：1883

# q
MQTT主题的格式是什么？
# a
主题格式为：`wq/v1.0/` + 电话号码后4位

# q
MQTT消息使用什么格式，包含哪些字段？
# a
消息为JSON格式，包含字段：
- motion：字符串变量，3位宽，不足补0，定义域0-100
- speed：整型变量，3位宽，不足补0，定义域0-100
- user_id：用户ID
- user_name：用户名称

# q
动作代码 `motion 001` 和 `motion 002` 分别对应什么操作？
# a
- motion 001：还原动作，speed由当前滑块决定
- motion 002：捡球动作，speed由当前滑块决定
- motion 003：急停动作，speed由当前滑块决定
