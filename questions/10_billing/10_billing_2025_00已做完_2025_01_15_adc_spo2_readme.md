# q
脉搏血氧仪测量 SpO2 的原理是什么？利用了血红蛋白的哪些光吸收特性？
# a
利用氧合血红蛋白（HbO2）与去氧血红蛋白（Hb）对不同波长光的吸收差异。HbO2 对红外光（约 940 nm）吸收更多，Hb 对红光（约 660 nm）吸收更多。通过发射这两波长光并测量经组织透射/反射后的光强度，分析脉搏波动引起的脉动吸收变化，计算血氧饱和度。

# q
SpO2 计算中用到的比率 R 是如何定义的？
# a
\[ R = \frac{(AC_{red}/DC_{red})}{(AC_{IR}/DC_{IR})} \]
其中 \( AC \) 代表光吸收波形的交流分量（脉动部分，主要由动脉血搏动引起），\( DC \) 代表直流分量（非脉动部分，反映组织、静脉血等恒定吸收）。

# q
一种常见的 SpO2 线性近似经验公式是什么？
# a
\[ SpO2 = 110 - 25R \]
该公式中的 110 和 25 是通过实验测量拟合得到的常数，不同设备可能使用不同的校准常数，实际应用中的准确公式依赖于具体校准。

# q
什么是 CPLD？
# a
CPLD（Complex Programmable Logic Device，复杂可编程逻辑器件）是一种可编程逻辑器件，用于实现数字电路设计，其规模和功能介于简单可编程逻辑器件（SPLD）和现场可编程门阵列（FPGA）之间，具有可重复编程、逻辑资源较丰富等特点。

# q
UART 接收器如何可靠地检测起始位？
# a
当检测到线路从空闲高电平变为低电平时，接收器按波特率的倍数（常见为 16 倍）对信号进行多次采样（例如 16 次中的多数判断），以确认这是一个真正的起始位而不是噪声干扰。
