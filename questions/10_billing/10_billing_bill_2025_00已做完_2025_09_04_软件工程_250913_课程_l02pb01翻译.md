# q
在gRPC天气预报服务的proto定义中，LocationDate消息包含哪些字段？
# a
```proto
message LocationDate {
  string city = 1;
  string country = 2;
  int32 day = 3;
}
```

# q
如何在proto文件中定义服务端流式RPC？
# a
在service的rpc方法中，返回类型前添加`stream`关键字。例如：
```proto
rpc GetCityWeatherMultipleDays(LocationDatePeriod) returns (stream CityWeatherData);
```

# q
在服务端实现简单RPC（如`getCityWeatherSingleDay`）时，应如何操作`StreamObserver`？
# a
获取数据后调用`responseObserver.onNext(data)`发送单条响应，然后调用`responseObserver.onCompleted()`通知客户端RPC完成。

# q
在多日流式查询的服务端实现中，如何逐日发送天气数据？
# a
通过循环从起始日期遍历到结束日期，每次生成当日的`CityWeatherData`并调用`responseObserver.onNext(data)`发送；循环结束后调用`responseObserver.onCompleted()`结束流。示例代码：
```java
for (LocalDate date = req.getStartDate(); !date.isAfter(req.getEndDate()); date = date.plusDays(1)) {
    CityWeatherData data = generateDailyData(date);
    responseObserver.onNext(data);
}
responseObserver.onCompleted();
```
