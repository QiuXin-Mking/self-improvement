# q
DX-WF24 模块如何设置为 MQTT 透传模式和 STA + MQTT 单发通讯？
# a
使用 AT 命令：
```
AT+SIMPLEMODE=3,4
```
该命令启用简单模式，参数 `3,4` 对应 MQTT 透传模式 + STA + MQTT 单发通讯。

# q
如何通过 AT 命令让 DX-WF24 连接指定的 Wi-Fi 路由器？
# a
使用 `AT+CWJAP` 命令，格式为：
```
AT+CWJAP=<SSID>,<password>
```
例如：
```
AT+CWJAP=Xiaomi_388B,qiuxin15970759167
```

# q
连接 EMQX MQTT 服务器时需要配置哪些参数，对应的 AT 命令是什么？
# a
需要依次执行以下 AT 命令（每条命令需追加 `\r\n`）：
1. 设置客户端 ID：`AT+MQTTLONGCLIENTID=mqtt_1`
2. 设置用户名：`AT+MQTTLONGUSERNAME=MQTT`
3. 设置密码：`AT+MQTTLONGPASSWORD=123456`
4. 连接服务器：`AT+MQTTCONN=8.129.97.33,1883,0`

# q
DX-WF24 模块如何向主题 `WF24` 发布一条 6 字节的负载 `123456`？
# a
使用 `AT+MQTTPUBRAW` 命令：
```
AT+MQTTPUBRAW=WF24,6,0,0\r\n
123456\r\n
```
参数说明：`WF24` 为订阅主题，`6` 为发送字节长度（不计 `\r\n`），后续 `0,0` 为 QoS 和 retain 标志，紧跟负载数据并以 `\r\n` 结束。

# q
在本地 EMQX 服务器上如何订阅 `WF24` 主题以测试模块上报？
# a
使用 mosquitto_sub 工具：
```
mosquitto_sub -h localhost -t WF24
```
如果服务器不在本地，将 `localhost` 替换为实际 IP 地址。
