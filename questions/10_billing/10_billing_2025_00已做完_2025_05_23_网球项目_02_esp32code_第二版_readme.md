# q
网球项目中“一键捡球”命令通过MQTT发送时，对应哪个motion值？动作顺序如何？
# a
motion值为1，表示前进。板子执行时会先发送前进指令到y轴，再发送到x轴，每个轴持续n秒后停止。

# q
急停功能对应的motion值是多少？发送MQTT消息的步骤是什么？
# a
motion值为2。步骤为：先向y轴发送停止指令，再向x轴发送停止指令，Topic均为 `wq/v1.0/9167`。

# q
复位（复原）命令对应的motion值是什么？与前进有何不同？
# a
motion值为3，表示后退。执行时先向y轴发送后退n秒后停止的指令，再向x轴发送相同指令。

# q
ESP32端接收到MQTT消息后，根据motion编号执行什么动作？
# a
motion 1：前进；motion 2：停止；motion 3：后退。
