# q
这款家居安防系统可以检测哪几类异常事件？
# a
系统可以检测五类异常：入侵检测（通过红外传感器，有异物报警）、火灾检测（有火报警）、天然气泄漏（超过上限报警）、室内温度过高（超过上限报警）以及室内湿度过高（超过上限报警）。

# q
当检测到异常时，系统通过哪些方式进行本地和远程报警？
# a
本地报警使用蜂鸣器和 LED；超过上限的报警信息会推送到 APP 的监控界面上，实现远程报警通知主人。

# q
APP 提供哪些实时监控数据显示？
# a
APP 监控界面实时显示当前室内温度、湿度、天然气检测数值、火灾状态（是否发生火灾）以及入侵状态（是否有人入侵）。

# q
用户可以通过 APP 调整哪些安全阈值？
# a
用户可以调整温度上限、湿度上限和天然气浓度上限，实现自定义报警阈值。
