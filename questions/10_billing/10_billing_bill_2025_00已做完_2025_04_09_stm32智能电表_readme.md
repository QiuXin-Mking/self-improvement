# q
STM32智能电表项目主要由哪些硬件组成？
# a
STM32 作为主控，OLED 作为显示屏，使用按键控制（按键1用于切换页面）。

# q
按键1在系统中的功能是什么？
# a
按键1用于切换不同的显示页面。

# q
项目中的三个页面分别显示哪些量程的电压和电流？
# a
页面1：0‑3.3V 量程；页面2：0‑5V 量程；页面3：0‑10V 量程。三个页面均显示电压、电流并计算功率。

# q
项目开发过程中量程范围发生了什么变化？
# a
最初设计为页面1‑0‑5V、页面2‑0‑10V、页面3‑0‑30V，后来调整为页面1‑0‑3.3V、页面2‑0‑5V、页面3‑0‑10V。

# q
硬件设计过程中出现了什么问题？
# a
封装画错了，布线画错了。
