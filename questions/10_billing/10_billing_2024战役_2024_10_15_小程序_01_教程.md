# q
如何使用 Docker 部署一个内存限制为 512MB 的 MySQL 容器？
# a
```bash
docker run -d \
  --name mysql \
  --memory=512m \
  -e MYSQL_ROOT_PASSWORD=15970759167 \
  -p 3306:3306 \
  -v /var/lib/mysql:/var/lib/mysql \
  -v /var/run/mysql:/var/run/mysqld \
  mysql
```
- `-d`：后台运行
- `--memory=512m`：限制容器最大内存为 512MB
- `-e MYSQL_ROOT_PASSWORD=...`：设置 root 密码
- `-p 3306:3306`：映射端口
- `-v`：挂载数据卷持久化数据

# q
如何连接本地 MySQL 并修改 root 用户的认证插件？
# a
使用 mysql 客户端连接：
```bash
mysql -h 127.0.0.1 -P 3306 -u root -p
```
进入 MySQL 后执行：
```sql
ALTER USER 'root'@'%' IDENTIFIED WITH 'mysql_native_password' BY '15970759167';
FLUSH PRIVILEGES;
```
其中 `'root'@'%'` 表示允许任意主机连接，`mysql_native_password` 是兼容性较好的认证插件。

# q
如何在 SQLite 中创建一个包含自增主键、设备名和温度字段的数据表？
# a
```sql
CREATE TABLE dev_tem(
    id INT AUTO_INCREMENT PRIMARY KEY,
    dev_name VARCHAR(255) NOT NULL,
    temperature DECIMAL(5,2),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```
注意：SQLite 中 `AUTO_INCREMENT` 应替换为 `AUTOINCREMENT`，但此处为教程原文，建议使用 `INTEGER PRIMARY KEY AUTOINCREMENT`。

# q
如何使用 curl 向 Flask 后端发送 JSON 格式的传感器数据？
# a
```bash
curl -X POST http://121.41.87.58:443/upload \
     -H "Content-Type: application/json" \
     -d '{"dev_name": "Sensor1", "temperature": 23.5}'
```
- `-X POST`：指定请求方法为 POST
- `-H "Content-Type: application/json"`：设置请求头
- `-d`：发送的 JSON 数据体
该命令向 `/upload` 端点提交设备名称和温度值。

# q
如何使用 mosquitto 工具测试 MQTT 消息发布与订阅？
# a
**发布消息**：
```bash
mosquitto_pub -h localhost -t test/topic -m "Hello, MQTT"
```
**订阅消息**：
```bash
mosquitto_sub -h localhost -t test/topic
```
- `-h`：指定 MQTT 服务器地址
- `-t`：主题名称
- `-m`：要发布的消息内容
