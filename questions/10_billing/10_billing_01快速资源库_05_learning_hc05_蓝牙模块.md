# q
如何将 HC05 蓝牙模块配置为从机模式以被手机识别？
# a
使用命令 `AT+ROLE=0` 并回车。设置后模块工作在从机角色，可被手机等主设备搜索到。

# q
在 HC05 模块中，如何让主机绑定指定从机的地址？
# a
先通过 `AT+ADDR?` 查询从机地址（例如 `98D3:31:F69E09`），然后在主机模块执行 `AT+BIND=98D3,31,F69E09` 并配合 `AT+CMODE=0` 固定连接模式。

# q
如何查询 HC05 模块的当前工作角色？
# a
发送 `AT+ROLE?`，返回 `0` 表示从机，`1` 表示主机。

# q
如何修改 HC05 模块的蓝牙设备名称？
# a
使用命令 `AT+NAME=<名称>`，例如 `AT+NAME=JSXQ` 将名称设为“JSXQ”。
