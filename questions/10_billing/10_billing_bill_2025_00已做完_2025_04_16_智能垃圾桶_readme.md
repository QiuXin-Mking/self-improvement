# q
智能垃圾桶通过什么传感器检测垃圾是否满溢？
# a
通过红外对射管监测垃圾桶内垃圾是否满溢。

# q
智能垃圾桶在自动模式下，有人经过时会触发什么动作？
# a
有人经过时驱动步进电机开盖，5秒后自动关闭，同时通过蓝牙发送开盖和关盖状态到手机。

# q
智能垃圾桶有哪些工作模式？
# a
四种模式：
- 自动模式：满溢检测、自动开盖、蓝牙通知；
- 手动模式：按键手动开盖或关盖；
- 消毒模式：按键手动开/关继电器，控制消毒灯；
- 语音控制模式：语音识别厨余垃圾、可回收垃圾、有害垃圾、其他垃圾，分别启动对应继电器或舵机。

# q
智能垃圾桶在垃圾满溢时如何通知用户？
# a
通过蓝牙模块发送短信到手机，同时满溢状态亮红灯，未满时亮绿灯。
