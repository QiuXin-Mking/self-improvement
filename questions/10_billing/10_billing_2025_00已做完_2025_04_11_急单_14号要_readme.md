# q
该项目使用的硬件有哪些？
# a
使用一块 STM32 微控制器、一块 OLED 显示屏和 4 个按键。

# q
如何在不同页面之间切换？
# a
通过按键1进行页面切换，依次在页面1~7之间循环切换。

# q
页面1、页面2、页面3分别显示什么传感器数据？
# a
- 页面1：检测温湿度并显示在 OLED 上；
- 页面2：检测噪音强度并显示在 OLED 上；
- 页面3：检测光照强度并显示在 OLED 上。

# q
在页面4中，灯光颜色如何随温度变化？
# a
温度高时显示冷光，温度低时显示缓光。

# q
根据页面7的描述，光照强度如何影响灯光颜色？
# a
光强大时显示冷光，光强小时显示暖光。
