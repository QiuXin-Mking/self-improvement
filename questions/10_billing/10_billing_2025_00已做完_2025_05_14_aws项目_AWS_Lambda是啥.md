# q
什么是 AWS Lambda？
# a
AWS Lambda 是亚马逊云服务提供的一种无服务器计算服务，允许用户上传代码并设置触发条件，AWS 自动在云端运行该代码，无需管理服务器、操作系统和扩缩容等运维工作。

# q
AWS Lambda 的核心特点有哪些？
# a
- 免运维：无需管理或维护服务器，只需专注编写代码。
- 自动伸缩：根据请求数量自动增减执行实例，实现“用多少算多少”。
- 按用量付费：仅为代码实际运行时间付费，无预付或闲置费用。
- 易于集成：可轻松响应 S3、DynamoDB、API Gateway、SNS 等 AWS 服务事件。

# q
AWS Lambda 的常见应用场景包括哪些？
# a
- 图片上传后自动缩放或添加水印。
- 通过 API Gateway 无服务器地实现 API。
- 数据库或日志变动触发通知。
- 自动处理备份、文件分析和 ETL 任务。

# q
简述 AWS Lambda 函数的基本执行流程。
# a
1. 编写函数代码（支持 Python、Node.js、Java、Go 等）。
2. 部署到 AWS Lambda。
3. 设置触发事件（例如 S3 新文件上传或 API 被访问）。
4. 事件发生时 Lambda 自动执行代码。
5. AWS 自动分配资源、运行、监控，并在执行结束后释放资源。
