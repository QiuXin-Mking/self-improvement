# q
树莓派的主机名是什么？如何使用主机名进行ping测试？
# a
主机名为 raspberrypi。使用命令 `ping raspberrypi -4` 可强制通过IPv4进行连通性测试。

# q
MLX90640和MLX90614传感器的数据流向及显示架构是怎样的？ESP32输出的数据格式是什么？
# a
传感器数据流架构：
```
mlx90640 --(IIC)--> esp32 --(UART)--> raspberrypi --> show
mlx90614 --(IIC)--> esp32 --(UART)--> raspberrypi --> show
csi --> raspberrypi --> show
```
ESP32通过UART输出的JSON格式示例：
```json
{
    "t":23.45,
    "m":[1, 2, 3]
}
```
