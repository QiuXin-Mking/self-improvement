# q
该项目的硬件组成包括哪些核心元器件？
# a
- 1个STM32微控制器
- 1个OLED显示屏
- 4个按键（按键1用于切换页面）
- 传感器：温湿度传感器、噪音传感器、光照传感器
- 可调色温的LED灯（冷光/暖光/缓光）

# q
通过哪个按键切换不同功能页面？
# a
按键1，循环切换7个页面。

# q
页面1~3分别显示什么传感器数据？
# a
- 页面1：检测并显示温湿度
- 页面2：检测并显示噪音强度
- 页面3：检测并显示光照强度

# q
页面4~7中LED灯的色温控制逻辑分别是什么？
# a
- 页面4：温度高时冷光，温度低时缓光
- 页面5：温度高时冷光，温度低时暖光
- 页面6：噪音大时冷光，噪音小时暖光
- 页面7：光强大时冷光，光强小时暖光
