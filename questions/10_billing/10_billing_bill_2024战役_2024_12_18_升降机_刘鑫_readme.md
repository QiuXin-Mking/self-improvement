# q
该系统的硬件通信架构是怎样的？
# a
显示屏 <--> 小主机 <--> ESP32/STM32 单片机 <--> 一堆传感器 + ADC。传感器和 ADC 数据经单片机处理后发送给小主机，小主机负责驱动屏幕显示。

# q
硬件中使用了哪些传感器？分别用于什么检测？
# a
- 6 个红外传感器：检测设备收折、设备打开、左侧摆臂抬起、右侧摆臂抬起、设备收折上行、设备收折下行。
- 2 个陀螺仪 MPU6050：检测加速度变化，实现上下行翘板碰撞（右型）和上下行侧壁碰撞（右型）检测。
- 电子秤模块：安装在隔层内，用于超载报警。
- ADC + 降压模块组合：用于电量显示、充电异常与电池亏电检测，以及配合红外检测电推杆损坏。
