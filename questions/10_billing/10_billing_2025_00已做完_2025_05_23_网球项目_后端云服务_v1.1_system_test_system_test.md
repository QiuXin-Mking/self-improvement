# q
如何使用 `curl` 命令测试 FastAPI 服务的用户注册接口？
# a
向 `/register` 端点发送 POST 请求，并在请求体中传递 JSON 格式的用户名和密码。命令示例：
```bash
curl -X POST "http://127.0.0.1:80/register" -H "Content-Type: application/json" -d '{"username": "alice", "password": "mypassword"}'
```

# q
如何使用 `curl` 命令测试 FastAPI 服务的用户登录接口？
# a
向 `/login` 端点发送 POST 请求，携带与注册时相同的 JSON 凭证。命令示例：
```bash
curl -X POST "http://127.0.0.1:80/login" -H "Content-Type: application/json" -d '{"username": "alice", "password": "mypassword"}'
```
