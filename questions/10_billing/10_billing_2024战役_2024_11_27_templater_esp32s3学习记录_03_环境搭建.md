# q
在 Arduino IDE 中配置 ESP32 支持时，需要在 Sketchbook location 下创建哪个目录来存放硬件包？
# a
需要创建 `hardware` 目录，即完整路径为 `C:\Users\Administrator\Documents\Arduino\hardware`，ESP32 核心文件应放在该目录下（如 `hardware\esp32\esp32`）。

# q
esp32 官方 Arduino 核心库的 GitHub 仓库地址是什么？
# a
https://github.com/espressif/arduino-esp32

# q
在 Arduino IDE 中针对 ESP32-S3 核心板开发时，应选择哪款板子？
# a
选择 `ESP32S3 Dev Module`
