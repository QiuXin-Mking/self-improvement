# q
如何使用 Docker 运行一个 MySQL 容器，并完成端口映射、数据持久化和内存限制？
# a
使用以下命令启动容器：
```bash
docker run -d \
  --name mysql \
  --memory=512m \
  -e MYSQL_ROOT_PASSWORD=15970759167 \
  -p 3306:3306 \
  -v /var/lib/mysql:/var/lib/mysql \
  -v /var/run/mysql:/var/run/mysqld \
  mysql
```
- `--name mysql`：容器名称为 mysql
- `--memory=512m`：限制内存使用 512MB
- `-e MYSQL_ROOT_PASSWORD=...`：设置 root 密码
- `-p 3306:3306`：将容器 3306 端口映射到宿主机 3306 端口
- `-v /var/lib/mysql:/var/lib/mysql`：挂载数据目录持久化数据库文件
- `-v /var/run/mysql:/var/run/mysqld`：挂载运行目录

# q
如何用 mosquitto 客户端在本地测试 MQTT 消息的发布与订阅？
# a
安装：
```bash
yum install -y mosquitto mosquitto-clients
```
启动服务：
```bash
sudo systemctl start mosquitto
sudo systemctl enable mosquitto
```
发布消息：
```bash
mosquitto_pub -h localhost -t test/topic -m "Hello, MQTT"
```
订阅消息：
```bash
mosquitto_sub -h localhost -t test/topic
```

# q
如何通过 curl 向 Flask Web 服务上传传感器数据（POST 请求）？
# a
使用如下命令：
```bash
curl -X POST http://121.41.87.58:3389/upload \
  -H "Content-Type: application/json" \
  -d '{
    "dev_name": "Sensor1",
    "temperature": 22.5
  }'
```
- 目标路由 `/upload` 接收 JSON 数据
- JSON 字段：设备名 `dev_name` 和温度值 `temperature`
- 服务处理后将数据存入 SQLite3 数据库
