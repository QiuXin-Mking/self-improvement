# q
无源蜂鸣器和有源蜂鸣器中的“源”指的是什么？
# a
指**震荡源**，而非电源。

# q
无源蜂鸣器的驱动要求是什么？
# a
无源蜂鸣器内部不带震荡源，无法用直流信号驱动；必须用 **2K~5K 的方波** 驱动才能鸣叫，声音频率可控。

# q
有源蜂鸣器与无源蜂鸣器在控制上的主要区别是什么？
# a
- **有源蜂鸣器**：内部带震荡源，通电即响，只需一个高低电平即可控制发声，程序控制方便。
- **无源蜂鸣器**：内部无震荡源，必须用方波驱动，无法仅靠高低电平发声。

# q
该无源蜂鸣器模块的工作电压范围和驱动三极管型号是什么？
# a
工作电压：**3.3V - 5V**，采用 **9012 三极管** 驱动。
