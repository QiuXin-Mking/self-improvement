# q
部署FastAPI网球助手应用需要安装哪些Python依赖包？
# a
```bash
pip install fastapi uvicorn paho-mqtt pymysql
```

# q
如果pip安装依赖时遇到代理错误（如ProxyError连接被拒绝）应该如何解决？
# a
清除当前会话的所有代理环境变量：
```bash
unset http_proxy
unset https_proxy
unset HTTP_PROXY
unset HTTPS_PROXY
```

# q
在服务器上创建并激活Python虚拟环境的基本步骤是什么？
# a
安装venv（如需要）：
```bash
apt install python3.10-venv -y
```
创建虚拟环境：
```bash
python3 -m venv tennis-env
```
激活环境：
```bash
source tennis-env/bin/activate
```
之后在环境中用pip安装依赖。

# q
如何查看网球助手服务的运行状态？
# a
可以使用以下两个命令之一：
```bash
systemctl status tennis_assistant
```
或
```bash
systemctl status tennis-assistant.service
```
