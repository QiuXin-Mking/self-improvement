# q
该智能公交站点信息发布系统的实时定位精度要求是多少？位置信息更新频率是多少？
# a
公交车辆 GPS 定位精度应达到 **10 米以内**，系统每隔 **30 秒** 更新一次公交车辆的位置信息。

# q
系统从数据采集到信息发布的延迟需要控制在多少秒以内？
# a
从数据采集到信息发布的延迟需控制在 **5 秒以内**。

# q
该方案中云服务器如何根据 GPS 信息计算拥堵程度？分为几个等级？
# a
云服务器根据当前是否处于高峰时段来判定拥堵程度，拥堵程度分为 **1、2、3、4 四个等级**。

# q
该方案中公交站点显示功能由哪块硬件板实现？它如何获取到站信息？
# a
由 **PCB B 板**（采用 ESP32 主控）实现。它通过 WiFi 向云服务器**查询**拥堵情况和预计到站时间，并控制屏幕进行显示。
