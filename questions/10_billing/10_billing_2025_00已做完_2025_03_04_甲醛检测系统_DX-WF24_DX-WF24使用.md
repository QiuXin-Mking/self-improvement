# q
DX-WF24模块的默认串口通信参数是什么？
# a
模块串口默认参数：`115200bps/8/n/1`（波特率 115200、8 位数据位、无校验、1 位停止位）。

# q
如何将DX-WF24设置为MQTT透传模式并准备连接Wi-Fi？
# a
发送AT指令：`AT+SIMPLEMODE=3,4`，这将设置模块为MQTT透传模式和STA+MQTT单发通讯。

# q
DX-WF24模块连接Wi-Fi路由器的AT指令格式是什么？
# a
使用`AT+CWJAP=路由器名称,路由器密码`，例如：
```
AT+CWJAP=Xiaomi_388B,qiuxin15970759167
```

# q
如何配置DX-WF24连接MQTT服务器并通过指定主题上报数据？
# a
1. 设置客户端ID：`AT+MQTTLONGCLIENTID=mqtt_1`
2. 设置用户名：`AT+MQTTLONGUSERNAME=MQTT`
3. 设置密码：`AT+MQTTLONGPASSWORD=123456`
4. 连接服务器：`AT+MQTTCONN=8.129.97.33,1883,0`（IP、端口、0表示无SSL）
5. 上报数据：`AT+MQTTPUBRAW=WF24,6,0,0`，后跟数据内容（如`123456\r\n`）
   - 参数依次为：主题、发送字节长度（不含`\n`）、保留参数、标志位。

# q
使用AT+BLUEISEND=1开启蓝牙发送时遇到ERROR=102，可能原因是什么？
# a
该命令需要先通过“大夏龙雀”APP成功连接对应模块后才能使用，否则返回错误（状态或模式错误）。ERROR=102通常提示操作时机不正确。
