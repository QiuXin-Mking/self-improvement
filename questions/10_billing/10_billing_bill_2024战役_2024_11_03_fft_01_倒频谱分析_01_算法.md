# q
在MATLAB中，如何将信号的频谱零频率分量移到中心？
# a
使用 `fftshift` 函数。例如：
```matlab
X = fft(x);
X = fftshift(X);
```

# q
在绘制频谱图时，常见的归一化幅度操作如何实现？
# a
先计算幅度谱 `abs(X)`，再除以幅度最大值进行归一化：
```matlab
spectrum = abs(X);
spectrum = spectrum / max(spectrum);
```

# q
在C语言给出的DFT实现中，如何从复数频谱计算对应频率分量的幅度和实际频率？
# a
幅度计算公式为：
```c
double amplitude = sqrt(spectrum[k].real * spectrum[k].real + spectrum[k].imag * spectrum[k].imag) / N;
```
对应频率（Hz）为：
```c
double frequency = k * fs / N;
```
其中 `N` 为采样点数，`fs` 为采样频率，`k` 为频谱索引（仅需处理前 `N/2` 个点）。

# q
给出的C语言DFT函数使用了什么算法？其时间复杂度如何？
# a
该实现为 DFT 的直接计算（按定义求和），使用两层 `for` 循环，时间复杂度为 O(N²)。
