# q
项目对比的两种虚拟化/容器化技术方案是什么？
# a
虚拟化方案采用 KVM + QEMU，容器化方案采用 Docker + Kubernetes（单节点集群）。

# q
实验中使用哪些压力测试工具来采集性能指标？
# a
使用 stress‑ng 生成 CPU、内存、磁盘 I/O 等混合负载，同时配合 iostat（来自 sysstat 工具集）实时监控磁盘 I/O 与 CPU 统计。

# q
实验统一采用哪个操作系统？
# a
Ubuntu 22.04 LTS（长期支持版）。

# q
实验中用于模拟真实边缘计算业务负载的应用是什么？
# a
OpenDroneMap（ODM），一套开源的无人机影像处理工具包，用于执行点云重建等计算密集型任务。

# q
实验所使用的两台物理服务器硬件配置是什么？
# a
每台服务器均配备 8 核 CPU、8 GB 内存、SSD 存储，通过千兆以太网卡互联，确保网络延迟低于 1 ms。
