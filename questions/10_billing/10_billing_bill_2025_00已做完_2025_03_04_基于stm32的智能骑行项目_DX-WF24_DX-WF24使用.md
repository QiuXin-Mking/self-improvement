# q
DX-WF24 模块的默认通讯波特率是多少？
# a
115200

# q
将 DX-WF24 设置为 STA 模式并启用 MQTT 单发通讯的 AT 命令是什么？
# a
```
AT+SIMPLEMODE=3,4
```

# q
连接 EMQX MQTT 服务器前，需要依次执行哪些 AT 命令完成 WiFi 连接与身份验证？
# a
1. 连接 WiFi（替换为实际 SSID 和密码）：
   ```
   AT+CWJAP=Xiaomi_388B,qiuxin15970759167
   ```
2. 设置客户端 ID：
   ```
   AT+MQTTLONGCLIENTID=mqtt_1
   ```
3. 设置用户名：
   ```
   AT+MQTTLONGUSERNAME=MQTT
   ```
4. 设置密码：
   ```
   AT+MQTTLONGPASSWORD=123456
   ```
5. 连接 MQTT 服务器（IP、端口、0）：
   ```
   AT+MQTTCONN=8.129.97.33,1883,0
   ```

# q
DX-WF24 向主题 “WF24” 发送 6 字节数据 “123456” 的 AT 命令格式是什么？
# a
先发送发布指令（主题,长度,0,0）：
```
AT+MQTTPUBRAW=WF24,6,0,0
```
然后紧接着发送数据（长度不包含结尾的 \r\n）：
```
123456
```

# q
在本地 MQTT Broker 上订阅 “WF24” 主题的 mosquitto_sub 命令是什么？
# a
```
mosquitto_sub -h localhost -t WF24
```
