# q
在网球项目需求中，通过 MQTT 接口转发的消息包含哪些动作和参数？
# a
转发消息包括：
- `speed`：取值范围 0-100
- `action: pick`
- `action: stop`
- `action: reset`

# q
网球项目中范兴堑负责的云服务器需要实现哪些核心功能？
# a
- 使用 Go/Python 开发云服务，实现用户记录、用户鉴权、服务代理
- 通过 MQTT 接口转发 `speed`、`pick`、`stop`、`reset` 等消息
- 实现计费功能或至少在数据库做好记录
- 提供单元测试（UT）及自测结果

# q
机器人电控工程师招聘要求中需要掌握哪些工业总线通信协议？
# a
需要实现 CAN、EtherCAT、Modbus 等工业总线通信协议，其中 EtherCAT/CAN 总线经验优先。

# q
机器人电控工程师的硬性技能要求包括哪些？
# a
- 熟悉 C/C++ 嵌入式开发，有 STM32/ARM/DSP 项目经验
- 掌握 PID 控制、电机驱动（如伺服驱动器配置）
- 了解 EtherCAT/CAN 总线优先
- 有机器人运动控制、工业自动化（如 PLC）相关经验者加分

# q
机器人电控工程师招聘中的加分项有哪些？
# a
- 参与过机器人比赛（如 RoboMaster、FRC、电子设计大赛）或相关项目
- 熟悉 Python/Matlab 用于算法仿真
- 了解实时操作系统（RT-Thread、FreeRTOS）
