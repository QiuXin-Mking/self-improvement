# q
如何使用 apt 安装 stress-ng 压力测试工具？
# a
```bash
sudo apt update
sudo apt install stress-ng
```

# q
如何同时启动 4 个 CPU、2 个 I/O 和 2 个虚拟内存压力测试，每个内存工作负载分配 512MB，总时长 60 秒？
# a
```bash
stress-ng --cpu 4 --io 2 --vm 2 --vm-bytes 512M --timeout 60s
```

# q
stress-ng 输出中的 Bogo Ops 指标表示什么，有哪些相关时间指标？
# a
Bogo Ops 表示压力测试期间完成的操作数量，提供系统性能的粗略估计，但不是标准化的性能基准。输出中还包括 real time（实际运行时间）、usr time（CPU 执行用户代码时间）和 sys time（CPU 管理和硬件交互时间）。
