# q
项目中使用的主控芯片型号是什么？
# a
stm32f103c8t6 核心板

# q
OLED 的 I2C 接口 SCL 和 SDA 分别连接到 STM32 的哪两个引脚？
# a
SCL 连接到 B11，SDA 连接到 B10

# q
电机驱动模块 TB6612 的 AIN1、AIN2、BIN1、BIN2、PWM1、PWM2 分别对应 STM32 的哪些引脚？
# a
AIN1 → A11, AIN2 → A12, BIN1 → A15, BIN2 → B3, PWM1 → B4, PWM2 → B5

# q
红外模块的头部和尾部输入信号分别连接至 STM32 的哪两个引脚？
# a
头部 IN1 → A5，尾部 IN2 → A0

# q
四个按键 KEY1～KEY4 分别与 STM32 的哪些引脚相连？
# a
KEY1 → B1, KEY2 → B0, KEY3 → A7, KEY4 → A6
