# q
在Windows上使用pip安装包时遇到错误提示“This error might have occurred since this system does not have Windows Long Path support enabled”，应如何解决？
# a
需要启用Windows长路径支持，具体方法可参考官方说明：https://pip.pypa.io/warnings/enable-long-paths

# q
当YOLOv5模型导出.pt文件失败时，实施方案中采用了什么替代方案？
# a
放弃生成.pt文件，转而导出ONNX格式模型，并在树莓派上使用ONNX Runtime运行推理。

# q
在树莓派上创建并激活Python虚拟环境的命令是什么？
# a
```bash
python3 -m venv onnx_env
source onnx_env/bin/activate
```

# q
在树莓派上使用清华镜像源安装Python包的通用命令格式是什么？
# a
`pip install 包名 -i https://pypi.tuna.tsinghua.edu.cn/simple`

# q
在树莓派上安装onnxruntime时报错“No matching distribution found for onnxruntime”，文档中提到的进一步尝试方向是什么？
# a
可能需要在32位环境上运行，参考博客：https://blog.csdn.net/weixin_45233413/article/details/125198496
