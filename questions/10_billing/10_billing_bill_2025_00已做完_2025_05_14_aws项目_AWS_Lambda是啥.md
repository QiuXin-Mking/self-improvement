# q
什么是AWS Lambda？
# a
AWS Lambda是亚马逊云服务提供的无服务器计算服务，允许用户上传代码并设置触发条件，AWS自动在云上运行这段代码，无需管理服务器、操作系统或伸缩等运维工作。

# q
AWS Lambda的主要特点有哪些？
# a
主要特点包括：免运维（无需管理服务器）、自动伸缩（根据请求数量自动扩缩实例）、按用量付费（只为代码实际运行时间付费）、易于与其它AWS服务集成（如S3、DynamoDB、API Gateway、SNS等）。

# q
AWS Lambda的常见应用场景有哪些？
# a
常见场景：图片上传后自动缩放或加水印、通过API Gateway实现无服务器API、数据库/日志变动触发通知、自动处理备份、文件分析和ETL等事件驱动的实时小型任务。
