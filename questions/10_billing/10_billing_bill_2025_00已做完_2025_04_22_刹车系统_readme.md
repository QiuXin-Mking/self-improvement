# q
该系统用于距离测量的传感器是什么？
# a
使用超声波传感器，明确不采用霍尔传感器。

# q
安全距离报警的触发条件是什么，如何配置安全阈值？
# a
当测量距离小于设定的安全距离阈值时，蜂鸣器鸣叫且 LED 灯亮。在安全距离测试页面（Page2）中，按键 2 可增加安全阈值，按键 3 可减少安全阈值，按键 4 可将当前测得的距离直接设为安全阈值。

# q
速度报警如何触发，如何设置速度阈值？
# a
当实际速度大于设定的速度阈值时，蜂鸣器鸣叫且 LED 灯亮。在速度显示页面（Page4）中，按键 2 增加速度阈值，按键 3 减少速度阈值，按键 4 可将当前相关数值（如基于距离和时间算出的速度）设为速度阈值。

# q
按键 1 的主要功能是什么？
# a
按键 1 用于在多个页面间切换，包括按键测试页、安全距离测试页、速度测试页和速度显示页。
