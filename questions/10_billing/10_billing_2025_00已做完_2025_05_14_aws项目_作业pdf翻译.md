# q
在该AWS项目中，无服务器组件（Lambda函数）是如何被触发的？
# a
当有新图片上传至指定的S3桶时，会通过S3的ObjectCreated事件自动触发两个Lambda函数（标注函数和缩略图生成器）。

# q
Web应用组件必须支持哪些核心功能？
# a
1. 提供HTML表单上传图片，将图片存储到S3，并将文件名、时间戳等元数据保存到RDS (MySQL) 数据库。
2. 展示已上传图片的列表、缩略图以及从RDS/S3获取的标注/描述。

# q
自动扩展测试需要提交哪些证据来证明ASG和ALB的正确行为？
# a
需要提供EC2控制台截图、CloudWatch指标和ALB监控数据，以证明：
- 扩展出（Scale Out）：因负载增加，ASG启动了新的EC2实例。
- 扩展入（Scale In）：负载降低后，ASG自动终止了多余的实例。
- 负载均衡：ALB成功将请求分发到了多个EC2实例。

# q
标注Lambda函数的具体任务是什么？
# a
从S3获取上传的图片，调用Gemini API生成图片的描述/标注，然后将生成的结果存储到RDS数据库中。

# q
作业要求的架构图中需要展示哪些关键组件与交互？
# a
- **Web应用架构图**：必须包含EC2 Auto Scaling Group、Application Load Balancer、S3、RDS，以及网络层组件（VPC、子网、安全组、IAM角色、堡垒机）。
- **无服务器架构图**：必须包含Lambda函数、事件源（S3）、外部API（Gemini）、下游目标（RDS/S3），并明确标注S3事件触发Lambda的机制以及它们对共享资源（RDS/S3）的访问。
