# q
当使用 `pip install` 时出现代理连接拒绝的错误，应该如何快速解决？
# a
清除当前 shell 中的代理环境变量：
```bash
unset http_proxy
unset https_proxy
unset HTTP_PROXY
unset HTTPS_PROXY
```

# q
如何将本地的 `tennis_assistant` 项目部署到远程服务器 `101.201.53.141` 的 `/root` 目录下？
# a
```bash
scp -r tennis_assistant root@101.201.53.141:/root
```

# q
在服务器上如何创建一个名为 `tennis-env` 的 Python 虚拟环境，并安装项目所需的依赖包？
# a
```bash
apt install python3.10-venv -y
python3 -m venv tennis-env
source tennis-env/bin/activate
pip install fastapi uvicorn paho-mqtt pymysql
```
