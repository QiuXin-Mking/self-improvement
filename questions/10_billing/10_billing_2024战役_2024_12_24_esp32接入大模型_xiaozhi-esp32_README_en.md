# q
What development framework is the XiaoZhi AI Chatbot project based on?
# a
It is developed based on Espressif's ESP-IDF.

# q
Which speech recognition model does the project use for multilingual support (Mandarin, Cantonese, English, Japanese, Korean)?
# a
SenseVoice

# q
Which large language models (LLMs) are integrated into XiaoZhi AI Chatbot?
# a
Qwen2.5 72B and Doubao (豆包) API

# q
What technology is used for offline voice wake-up in this project?
# a
ESP-SR (Espressif's speech recognition framework)

# q
Which voice print recognition model is used to identify who is calling the AI's name?
# a
3D Speaker
