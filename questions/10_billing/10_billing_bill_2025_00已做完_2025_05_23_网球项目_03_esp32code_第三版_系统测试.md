# q
如何使用 mosquitto_pub 向指定 MQTT 代理发布主题消息？
# a
使用如下格式：
```
mosquitto_pub -h <代理地址> -p <端口> -t "<主题>" -m '<消息内容>'
```
例如：
```
mosquitto_pub -h 101.201.53.141 -p 1883 -t "wq/v1.0/9167" -m '{"motion":"1", "speed":100, "user_id":"U100", "user_name":"Test"}'
```

# q
该测试中控制网球设备的 MQTT 消息负载（JSON）包含哪些字段？
# a
字段包括：
- `motion`：运动方向（如"1"、"2"、"3"）
- `speed`：运动速度（数值）
- `user_id`：用户标识
- `user_name`：用户名称

# q
根据测试记录，speed 参数设为 70 和 90 时设备分别表现如何？
# a
- speed 为 70 时：“可以跑得起来”
- speed 为 90 时：“快得很”

# q
测试中 motion 字段值为 "1" 且 speed 为 80 时设备出现了什么现象？
# a
“在转动，很慢”
