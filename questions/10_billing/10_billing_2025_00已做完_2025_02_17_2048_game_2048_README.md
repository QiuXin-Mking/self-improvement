# q
2048 游戏是基于哪些作品启发的？
# a
2048 是 1024 的小型克隆，基于 Saming 的 2048（也是一个克隆），并间接受到 Threes 的启发。

# q
2048 项目当前由谁维护？
# a
该仓库的维护者是 Anna Harren 和 sigod。

# q
2048 项目使用什么许可证？
# a
2048 使用 MIT 许可证。

# q
如何为 2048 项目做出贡献？
# a
欢迎修改和改进。可以 fork 仓库并开启 pull request。请在一个特定的分支上进行更改，并请求合并到 `master` 分支。提交 PR 前请确保游戏完全正常工作。更多信息见 contributing guide。
