# q
该智能门锁系统支持哪几种开锁方式？
# a
支持指纹匹配、六位密码（可设两个）、RFID刷卡以及手机蓝牙解锁。

# q
与AS608指纹模块通信的串口波特率是通过哪个宏定义设置的？默认值是多少？
# a
通过宏 `usart2_baund` 设置，默认值为 `57600`，初始化时作为参数传入 `usart2_init()`。

# q
在代码中，两个解锁密码从STM32内部Flash的哪些地址读取？
# a
密码1从地址 `0X08020004` 读取，密码2从地址 `0X08090004` 读取，均使用 `STMFLASH_Read()` 函数，读取长度为 2 个 uint32_t。

# q
RFID RC522模块的SPI引脚与STM32的对应连接是怎样的？
# a
RST → A1, IRQ → B0, MISO → B1, MOSI → A5, SCK → A6, SDA → A7。

# q
系统中用于驱动步进电机的电机型号和驱动板是什么？
# a
步进电机型号为 28BYJ4，驱动模块为 ULN2003。
