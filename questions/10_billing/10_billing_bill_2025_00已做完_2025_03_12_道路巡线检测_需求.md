# q
主控芯片和显示模块的选型及理由是什么？
# a
主控芯片为 STM32，显示模块选用 OLED 128×64，理由是亮度高、具有科技感、资料丰富，且便于论文撰写。

# q
为什么项目中推荐使用 USB 供电而不是 AC220V？
# a
AC220V 存在触电和电弧风险，对新手尤其危险，因此推荐使用更安全的 USB 供电。

# q
车道线检测使用哪种传感器？如何在实际环境中模拟车道线？
# a
使用 OpenMV 进行车道线检测，在实际环境中用黑色电工胶布贴在地上模拟一条大黑线作为车道线。

# q
如何模拟方向盘操作并检测方向盘抖动？
# a
采用舵机左右转动模拟方向盘修正操作，通过 MPU6050 陀螺仪检测方向盘是否抖动并将结果在 OLED 屏上显示。

# q
异常报警通过哪些器件实现？
# a
通过蜂鸣器和红色 LED 进行声光报警指示。
