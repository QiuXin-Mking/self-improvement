# q
智能大棚系统如何根据环境温度自动调控？
# a
系统通过 DHT11 模块获取环境温度，若温度低于设定的阈值，则通过继电器执行加热动作，并将实时参数同步显示在 Web UI 和 OLED 屏幕上。

该项目中使用的 MQTT Broker 配置信息是什么？
# a
MQTT Broker IP 为 `121.41.87.58`，端口 `1883`，发布/订阅的主题为 `template1`。

# q
该系统的 Web UI 如何启动？
# a
进入目录 `/var/www/my_project`，先激活虚拟环境 `source /root/cpp_project/python_project/venv/bin/activate`，然后执行 `python3 test1.py` 启动 Flask 云平台。
