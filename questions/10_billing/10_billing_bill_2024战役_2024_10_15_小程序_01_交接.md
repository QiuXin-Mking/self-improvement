# q
使用 Docker 运行 MySQL 容器时，如何以限制内存 512MB 并挂载数据目录的方式启动？
# a
```bash
docker run -d \
  --name mysql \
  --memory=512m \
  -e MYSQL_ROOT_PASSWORD=<密码> \
  -p 3306:3306 \
  -v /var/lib/mysql:/var/lib/mysql \
  -v /var/run/mysql:/var/run/mysqld \
  mysql
```
关键参数：`-d` 后台运行，`--memory` 限制内存，`-e` 设置 root 密码，`-p` 映射端口，`-v` 挂载数据卷保证数据持久化。

# q
在命令行下如何用 curl 向服务器发送 JSON 格式的 POST 请求上传传感器数据？
# a
```bash
curl -X POST http://121.41.87.58:3389/upload \
  -H "Content-Type: application/json" \
  -d '{
    "dev_name": "Sensor1",
    "temperature": 22.5
}'
```
`-X POST` 指定请求方法，`-H` 设置请求头表明 JSON 格式，`-d` 后跟 JSON 数据。

# q
如何使用 mosquitto 客户端在命令行发布和订阅 MQTT 消息？
# a
发布消息：
```bash
mosquitto_pub -h localhost -t test/topic -m "消息内容"
```
订阅消息：
```bash
mosquitto_sub -h localhost -t test/topic
```
`-h` 指定 MQTT 代理地址，`-t` 指定主题，`-m` 指定消息载荷。

# q
在 SQLite3 中如何创建带有自增主键和时间戳的表，并插入与查询数据？
# a
创建表：
```sql
CREATE TABLE dev_tem(
    id INT AUTO_INCREMENT PRIMARY KEY,
    dev_name VARCHAR(255) NOT NULL,
    temperature DECIMAL(5,2),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```
插入数据：
```sql
INSERT INTO dev_tem (dev_name, temperature) VALUES ('Device 1', 23.58);
```
查询最新 10 条记录：
```sql
SELECT temperature, created_at FROM dev_tem
WHERE dev_name = "Sensor1"
ORDER BY created_at DESC
LIMIT 10;
```
SQLite 中虽无 `AUTO_INCREMENT`，但使用 `INTEGER PRIMARY KEY` 会自动自增，上述语句为兼容概念说明。

# q
如何进入正在运行的 Docker 容器的内部 bash 环境？
# a
```bash
docker exec -it <容器名或ID> bash
```
例如：
```bash
docker exec -it my-mysql bash
```
