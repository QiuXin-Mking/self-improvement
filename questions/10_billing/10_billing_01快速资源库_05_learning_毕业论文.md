# q
毕业论文第一章（绪论）通常需要包含哪些内容？
# a
需要说明学术背景（通过知网查阅相关论文）、落地应用场景（分析现有问题及你解决的问题）、展示学术意义（纵向课题，结合文献指出不足并说明你的改进）和商业价值（横向课题，对比同类项目、阐述商业前景与可行性验证）。

# q
第二章的核心任务是什么？
# a
进行方案对比，确定最终方案，涉及电机、步进电机、电磁铁等执行器选型。

# q
第三章硬件部分一般包括哪些内容？
# a
介绍各硬件模块（如STM32、UART2、LCD1602、步进电机），附图片与资料；贴原理图并叙述电路连接；讲解相关协议与配置，如GPIO模式（推挽输出）、NVIC中断分组2、UART/USART、IIC、SPI协议。

# q
第四章软件部分如何组织？
# a
分层描述程序逻辑：驱动层、应用代码层、用户代码层（STM32）或用户代码层（51）；画出Visio流程图（形状用正方形、文字加粗），并进行文字论述。
