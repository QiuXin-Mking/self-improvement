# q
这个STM32小电视项目使用了哪些主要硬件模块？
# a
- 主控：普中STM32F103C8T6最小系统板
- 显示：0.96寸OLED（IIC接口）
- 网络：ESP8266模块（通过第三方API获取时间、天气等信息）
- 姿态：MPU6050陀螺仪模块（IIC接口，AD0）
- 环境传感：光敏传感器、DHT11温湿度传感器
- 输入：4个独立按键

# q
delay_init() 函数在这个工程中的作用是什么？
# a
delay_init() 用于初始化系统滴答定时器（SysTick），使得 delay_ms() 等延时函数能够正常工作。

# q
项目中配置的 NVIC 中断分组是怎样的？UART 波特率设为多少？
# a
NVIC 中断分组配置为第2组：2位抢占优先级，2位响应优先级。UART 通信波特率设置为115200。

# q
下列调试代码段实现了什么功能？
```c
len = USART2_RX_STA & 0x3fff;
printf("\r\n send command \r\n\r\n");
for (t = 0; t < len; t++)
{
    USART_SendData(USART1, USART2_RX_BUF[t]);
    while (USART_GetFlagStatus(USART1, USART_FLAG_TC) != SET);
}
printf(" %d\r\n", t);
```
# a
这段代码的作用是：从 USART2 的接收缓冲区中取出已接收的数据长度（低14位），然后通过 USART1 逐字节发送出去，最后打印发送的总字节数。通常用于将 ESP8266 返回的数据转发到串口调试助手显示。
