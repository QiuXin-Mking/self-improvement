# q
该智能家居系统实现了哪些核心功能？
# a
人机交互（4个按键 + OLED显示）、联网交互（ESP32）、监控房间温湿度（DHT11）、窗帘拉开和关闭（舵机）、灯光的开关（LED）、空调的通断（继电器）、光强调节（光照传感器）、来人灯开（人体红外传感器）。

# q
系统硬件分为哪些标准部分和拓展部分？
# a
标准硬件：ESP32、STM32、ST-Link、按键、OLED、Type-C、拨动开关；
拓展硬件：LED、舵机、继电器、DHT11、光照传感器、人体红外传感器。

# q
ESP Link面板中，温度、湿度、光强度的数值使用哪种组件显示？
# a
使用 `num` 组件显示。

# q
ESP Link面板中，用于调节温度阈值、湿度阈值和光阈值的组件是什么？
# a
使用 `slider` 组件。

# q
ESP Link面板中，窗帘、灯光和空调的开关控制采用什么组件类型？
# a
采用 `button` 组件，每个功能包含开和关两个按钮。
