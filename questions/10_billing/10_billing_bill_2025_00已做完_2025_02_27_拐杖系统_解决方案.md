# q
如何使用yum安装Mosquitto MQTT服务端并启动它？
# a
```bash
yum install -y mosquitto mosquitto-clients
```
配置文件位于 `/etc/mosquitto/mosquitto.conf`。启动和设置开机自启的命令：
```bash
sudo systemctl start mosquitto
sudo systemctl enable mosquitto
systemctl status mosquitto
```
测试订阅的命令示例：
```bash
mosquitto_sub -h localhost -t test/topic
```

# q
在CentOS中如何配置Docker使用阿里云镜像加速器？
# a
创建或编辑 `/etc/docker/daemon.json` 并添加以下内容：
```json
{
  "registry-mirrors": ["https://1umjrwoj.mirror.aliyuncs.com"]
}
```
然后重新加载配置并重启Docker：
```bash
sudo systemctl daemon-reload
sudo systemctl restart docker
```

# q
如何为Node.js项目编写Dockerfile并构建运行？
# a
`Dockerfile` 内容示例：
```dockerfile
FROM node:latest
WORKDIR /usr/src/app
COPY package*.json ./
RUN npm install
COPY . .
EXPOSE 3000
CMD ["node", "app.js"]
```
构建镜像：
```bash
docker build -t my-node-app .
```
运行容器（映射宿主80端口到容器3000端口）：
```bash
docker run -d -p 80:3000 --name my-node-container my-node-app
```

# q
如何初始化一个Node.js项目并安装Express？
# a
```bash
npm init -y
npm install express
```
这会生成 `package.json` 并安装Express框架到 `node_modules` 目录。
