# q
如何使用 AWS CLI 查询指定安全组的入站规则？
# a
```bash
aws ec2 describe-security-groups --group-ids sg-0783450f5d3042229 --query "SecurityGroups[0].IpPermissions"
```

# q
如何使用 AWS CLI 为安全组添加允许来自任意 IP 的 SSH、HTTP 和 HTTPS 入站规则？
# a
```bash
# 放通 22 端口（SSH）
aws ec2 authorize-security-group-ingress --group-id sg-0783450f5d3042229 --protocol tcp --port 22 --cidr 0.0.0.0/0

# 放通 80 端口（HTTP）
aws ec2 authorize-security-group-ingress --group-id sg-0783450f5d3042229 --protocol tcp --port 80 --cidr 0.0.0.0/0

# 放通 443 端口（HTTPS）
aws ec2 authorize-security-group-ingress --group-id sg-0783450f5d3042229 --protocol tcp --port 443 --cidr 0.0.0.0/0
```

# q
如何使用 AWS CLI 创建密钥对并保存私钥文件？
# a
```bash
aws ec2 create-key-pair --key-name my-key --query 'KeyMaterial' --output text > my-key.pem
chmod 600 my-key.pem
```

# q
如何使用 AWS CLI 基于指定 AMI、实例类型、密钥对和安全组创建 EC2 实例？
# a
```bash
AMI_ID=ami-0c02fb55956c7d316
aws ec2 run-instances \
  --image-id $AMI_ID \
  --count 1 \
  --instance-type t2.micro \
  --key-name my-key \
  --security-group-ids sg-0783450f5d3042229 \
  --associate-public-ip-address \
  --query 'Instances[0].InstanceId'
```

# q
如何使用 AWS CLI 查询 EC2 实例的详细信息（如状态、IP、子网、VPC）？
# a
```bash
aws ec2 describe-instances --instance-ids i-070816f44138c3b56
```
该命令返回 JSON，包含实例状态（如 `"State": {"Code": 16, "Name": "running"}`）、公有/私有 IP、子网 ID、VPC ID 等信息。
