# q
系统硬件主要由哪些组件构成？
# a
硬件包括4个按键、一个OLED显示屏，以及LED报警灯、蜂鸣器、水泵、通风风扇、补光灯等执行器和传感器。

# q
该系统如何实现远程监控和控制？
# a
通过云服务器进行通信：在云服务器上搭建网站，使用Python Flask框架后端与HTML网页前端，实现远程监控和控制硬件。

# q
页面1中空气温湿度的报警逻辑是什么？
# a
检测空气温湿度，当温度超过上限或湿度超过上限时，通过LED和蜂鸣器发出报警。

# q
页面2的土壤湿度控制如何工作？
# a
检测土壤湿度，当湿度低于设定下限时，自动启动水泵进行灌溉。

# q
页面7的语音控制支持哪些具体命令？
# a
支持语音控制通风风扇、补光灯和水泵，示例命令如“打开风扇”、“打开补光灯”等。
