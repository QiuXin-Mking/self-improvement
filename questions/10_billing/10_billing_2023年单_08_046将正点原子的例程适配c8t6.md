# q
将正点原子例程从ZET6适配到C8T6时，需要修改哪两个C/C++编译宏定义？
# a
需要将宏定义 `STM32F10X_HD` 改为 `STM32F10X_MD`，并同时保留 `USE_STDPERIPH_DRIVER`，最终定义为：
```
STM32F10X_MD,USE_STDPERIPH_DRIVER
```

# q
ZET6例程改为C8T6后，启动文件的更换路径在哪里？
# a
将启动文件 `startup_stm32f10x_hd.s` 更换为 `startup_stm32f10x_md.s`，文件路径为：
```
E:\003_smart_car\004 32\000_stm32f103光盘资料\STM32F103C8T6 STM32开发板小系统板单片机核心板 学习板实验板\STM32F103C8小系统\STM32F103C8T6核心板程序资料\STM32F103C8T6核心板测试程序(RTC)\Libraries\CMSIS\CM3\DeviceSupport\ST\STM32F10x\startup\arm
```

# q
适配C8T6时，除了宏定义和启动文件，还需要修改什么芯片相关配置？
# a
还需要修改芯片的Flash大小配置，从ZET6对应的高容量Flash改为C8T6对应的中等容量Flash配置（具体修改需在工程选项的Target或Linker中调整）。
