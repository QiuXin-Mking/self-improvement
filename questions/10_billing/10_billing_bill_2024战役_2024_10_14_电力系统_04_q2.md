# q
电力系统可靠性指标EENS的全称及含义是什么？
# a
EENS 全称为 Expected Energy Not Supplied（期望未供应电量），是用于评估电力系统可靠性的指标，表示在统计期间内因容量不足而期望未能供应的电能总量。

# q
本案例中系统A和系统B的装机容量、机组参数及峰值负荷分别是多少？
# a
- 系统A：4台30MW发电机组，总容量120MW，单台强制停运率10%，峰值负荷100MW。
- 系统B：8台15MW发电机组，总容量120MW，单台强制停运率8%，峰值负荷100MW。
- 两条联络线，每条16MW，总联络容量32MW。

# q
根据给定的故障率和修复时间，如何计算联络线的可用率A？
# a
已知故障率 λ = 5 次/年（转换为天：λ = 5/365 ≈ 0.013698 次/天），修复率 μ = 1/24 次/天（平均修复时间24小时）。
可用率按稳态公式计算：
```
A = μ / (λ + μ) = (1/24) / (0.013698 + 1/24)
  = 0.041667 / 0.055365 ≈ 0.98649
```

# q
当联络线100%可靠时，根据文中的计算，各系统是否会出现负荷损失？
# a
不会。系统A期望可用容量通过 `0.9^4 × 120MW ≈ 78.73MW` 计算，加上联络线容量32MW后总可用容量约110.73MW，高于100MW峰值负荷；系统B类似计算得到约110.96MW，同样高于峰值负荷，因此无负荷损失。
