# q
步进电机单四拍工作方式的励磁相序（十六进制）是什么？
# a
```c
0x01, 0x02, 0x04, 0x08
```
对应二进制：0001、0010、0100、1000，每次仅一相导通。

# q
步进电机双四拍工作方式的励磁相序（十六进制）序列是什么？
# a
```c
0x01, 0x03, 0x02, 0x06, 0x04, 0x0C, 0x08, 0x09
```
该序列在单相励磁之间插入相邻两相同时导通的节拍，可提高转矩和平稳性。

# q
在C语言中，如何用位运算表示步进电机第0～3相的励磁信号？
# a
使用左移运算：
```c
(1 << 0) // 0x01，第0相
(1 << 1) // 0x02，第1相
(1 << 2) // 0x04，第2相
(1 << 3) // 0x08，第3相
```
