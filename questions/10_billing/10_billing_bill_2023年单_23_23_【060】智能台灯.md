# q
智能台灯项目使用什么主控芯片？
# a
使用 STM32 作为主控模块，实现整个系统的任务控制。

# q
ASR 语音模块支持哪些具体的语音控制指令？
# a
ASR 语音模块可识别以下词条并执行操作：
- 打开台灯
- 关闭台灯
- 台灯调到一档
- 台灯调到二档
- 台灯调到三档

# q
声音传感器和光照传感器如何联合实现台灯的自动控制？
# a
自动控制逻辑为：
- 当环境 **无光且有声音** 时，控制台灯 **打开**。
- 当环境 **无声且无光** 时，控制台灯 **关闭**。

# q
HC-05 蓝牙模块在系统中的功能是什么？
# a
通过蓝牙与手机连接，实现在手机上控制台灯的开/关及一/二/三档调节。

# q
OLED 模块在智能台灯中承担什么角色？
# a
提供人机交互界面，将系统状态和参数显示在 OLED 屏幕上。
