# q
智能大棚节水系统通过什么方式切换不同的功能页面？
# a
通过按键1进行页面切换，共有4个页面（page 1～4）。

# q
系统在页面1中如何根据土壤湿度触发报警？
# a
页面1检测土壤湿度，OLED显示土壤湿度和上限阈值。当土壤湿度超过设定阈值时，蜂鸣器报警且LED亮。

# q
空气温湿度监测页面如何实现报警与控制？
# a
页面2检测空气温湿度，OLED显示温湿度及上限阈值。若空气温湿度超过上限阈值，则触发蜂鸣器报警并点亮LED。

# q
水泵的控制方式有哪些？
# a
有两种控制方式：
- 页面3：通过键盘按键本地控制水泵；
- 页面4：通过远程App控制水泵。

# q
远程App需要实现哪些监控和控制功能？
# a
根据二次需求，App需实现：
- 显示空气温度和湿度；
- 设置温度上限和湿度上限；
- 控制水泵开关。
