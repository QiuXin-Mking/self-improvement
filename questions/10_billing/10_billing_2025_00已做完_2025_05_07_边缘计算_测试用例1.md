# q
如何使用 stress-ng 对 CPU 进行压力测试？
# a
使用 `stress-ng --cpu` 指定工作线程数，`--timeout` 设定运行时长。例如，启动 4 个 CPU 密集型线程运行 30 秒：
```bash
stress-ng --cpu 4 --timeout 30s
```

# q
如何使用 stress-ng 对内存进行压力测试？
# a
使用 `stress-ng --vm` 指定压测进程数，`--vm-bytes` 设定每个进程分配的内存量，`--timeout` 设定运行时长。例如，启动 2 个虚拟内存压测进程，每个分配 512 MB，运行 60 秒：
```bash
stress-ng --vm 2 --vm-bytes 512M --timeout 60s
```
