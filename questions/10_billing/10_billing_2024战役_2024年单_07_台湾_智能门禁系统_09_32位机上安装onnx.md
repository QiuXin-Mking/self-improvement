# q
如何查看树莓派的系统架构？
# a
使用命令 `uname -a`，输出中的 `armv7l` 表示 32 位 ARM 架构。

# q
如何检查树莓派系统是 32 位还是 64 位？
# a
使用命令 `getconf LONG_BIT`，输出为 `32` 则表示 32 位系统，输出为 `64` 则表示 64 位系统。

# q
如何查看树莓派的 Debian 版本号及对应的代号？
# a
使用 `cat /etc/debian_version` 查看版本号，然后对照以下映射：
- 11.x → Bullseye
- 10.x → Buster
- 9.x → Stretch

# q
在 32 位树莓派上安装 onnxruntime 1.7.2, Python 3.7 的具体命令是什么？
# a
先进入包含 .whl 文件的目录，然后执行：
```bash
pip3 install onnxruntime-1.7.2-cp37-cp37m-linux_armv7l.whl
```

# q
哪里可以找到为树莓派预编译的 onnxruntime 包？
# a
可以从 GitHub 仓库获取：
```bash
git clone https://github.com/nknytk/built-onnxruntime-for-raspberrypi-linux.git
```
或访问：https://github.com/nknytk/built-onnxruntime-for-raspberrypi-linux
