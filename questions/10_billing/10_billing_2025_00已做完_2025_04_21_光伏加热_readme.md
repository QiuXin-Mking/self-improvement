# q
该系统基于ESP32实现了哪些自动控制及报警逻辑？
# a
- 温度超过阈值 → 停止加热，LED灯亮、蜂鸣器响
- 湿度超过阈值 → 打开风扇，蜂鸣器响
- 光照低于阈值 → 灯亮，蜂鸣器响
- 烟雾超过阈值 → 打开风扇通风，蜂鸣器响
- CO₂超过阈值 → 打开风扇通风，蜂鸣器响

# q
手机APP支持哪些功能？
# a
- 显示传感器数据：温度、湿度、光照强度、烟雾浓度、CO₂浓度
- 远程设置各传感器报警阈值
- 手动控制：蜂鸣器、加热、风扇、LED灯
- 定时加热（可设定秒数，如10s/20s/30s后开启或关闭）
- 监控光伏板参数：蓄电池电压、电流、电量和光伏板光强

# q
系统硬件主要包含哪些组件？
# a
ESP32主控、继电器、温湿度传感器、光照传感器、烟雾传感器、CO₂传感器、加热模块、风扇模块、蜂鸣器、LED灯模块、按键、电压/电流/功率检测模块。

# q
按键控制继电器的功能如何实现蓄电池充放电切换？
# a
按键控制继电器触点切换：继电器公共端接蓄电池，常闭端接光伏充电回路，常开端接放电端。按下按键切换继电器状态，实现光伏充电与蓄电池放电的切换。
