# q
智能作息系统的核心功能流程是什么？
# a
系统基于单片机时基计时进行年历计算，每分钟递增时与预设作息时间比较，若相等则触发相应控制动作。具体演示为每10秒播报上课，20秒播报下课，30秒播报上课，循环执行。

# q
该系统如何通过灯光区分上课和下课状态？
# a
上课时点亮红灯，下课时点亮绿灯。

# q
用户如何修改 OLED 上显示的教室名称？
# a
通过手机蓝牙 APP 与蓝牙模块 WF24 通信，实时修改每次显示的教室名称。

# q
该智能作息系统由哪些主要硬件组成？
# a
硬件包括 OLED 显示屏、STM32 单片机、蓝牙模块 WF24、JQ8900 语音播报模块、红灯和绿灯。
