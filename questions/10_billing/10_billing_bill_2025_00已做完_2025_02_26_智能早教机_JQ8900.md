# q
JQ8900语音模块的默认串口波特率是多少？
# a
9600 bps

# q
JQ8900播放第1个音频文件的串口指令是什么？
# a
```c
pr_temp[0] = 0xAA;
pr_temp[1] = 0x07;
pr_temp[2] = 0x02;
pr_temp[3] = 0x00;
pr_temp[4] = 0x01;
pr_temp[5] = 0xB4;
```
即十六进制序列：`AA 07 02 00 01 B4`

# q
JQ8900播放第2个音频文件的串口指令校验字节是多少？
# a
校验字节为 `0xB5`，完整指令序列：`AA 07 02 00 02 B5`
