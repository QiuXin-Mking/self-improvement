# q
在Matlab中，如何对信号频谱进行归一化处理并绘制幅度图？
# a
计算信号的FFT后，使用`abs(X)`获取幅度谱，然后除以最大值`spectrum = spectrum / max(spectrum)`实现归一化，最后用`plot(f, spectrum)`绘制频率-幅度图。

# q
用C语言实现DFT时，复数结构体如何定义？
# a
使用包含`double real`和`double imag`两个成员的结构体来表示复数：
```c
typedef struct {
    double real;
    double imag;
} Complex;
```

# q
在DFT的C实现中，如何计算频谱每个频率分量的实部和虚部？
# a
对于每个频率索引`k`，累加信号与复指数旋转因子的乘积：
```c
double angle = 2 * M_PI * k * n / N;
spectrum[k].real += signal[n] * cos(angle);
spectrum[k].imag -= signal[n] * sin(angle);
```
其中`N`为采样点数，`n`为时间索引。

# q
在Matlab频谱分析中，如何构造频率轴并将零频率分量移到中心？

使用`fftshift(X)`将零频移到频谱中心，频率轴由`f = (-(length(x)/2):(length(x)/2-1)) / (length(x)/Fs)`计算得到。
