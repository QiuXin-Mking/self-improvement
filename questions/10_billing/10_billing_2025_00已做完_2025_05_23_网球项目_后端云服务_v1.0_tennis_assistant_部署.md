# q
部署FastAPI项目需要安装哪些Python依赖包？
# a
需要安装 fastapi、uvicorn、paho-mqtt 和 pymysql，命令如下：
```bash
pip install fastapi uvicorn paho-mqtt pymysql
```

# q
当 pip install 因代理连接失败时（如报 `ProxyError: Cannot connect to proxy.`），如何清除代理环境变量？
# a
依次取消 http 和 https 代理变量：
```bash
unset http_proxy
unset https_proxy
unset HTTP_PROXY
unset HTTPS_PROXY
```

# q
如何在服务器上创建一个新的 Python 虚拟环境（以 tennis-env 为例）并激活？
# a
先安装 Python 3 的 venv 支持，然后创建并激活：
```bash
apt install python3.10-venv -y
python3 -m venv tennis-env
source tennis-env/bin/activate
```
