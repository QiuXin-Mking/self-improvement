# q
如何解决 pip install 时因代理配置导致的 Connection refused 错误？
# a
使用以下命令清除环境变量中的代理设置：
```bash
unset http_proxy
unset https_proxy
unset HTTP_PROXY
unset HTTPS_PROXY
```

# q
如何在服务器上为 Python 项目创建独立的虚拟环境并安装所需依赖？
# a
```bash
apt install python3.10-venv -y
python3 -m venv tennis-env
source tennis-env/bin/activate
pip install fastapi uvicorn paho-mqtt pymysql
```

# q
如何将本地 `tennis_assistant` 目录完整复制到远程服务器的 `/root` 路径下？
# a
```bash
scp -r tennis_assistant root@101.201.53.141:/root
```

# q
部署完成后，如何查看网球助手服务的运行状态？
# a
使用 systemctl 命令检查服务状态：
```bash
systemctl status tennis_assistant
# 或
systemctl status tennis-assistant.service
```
