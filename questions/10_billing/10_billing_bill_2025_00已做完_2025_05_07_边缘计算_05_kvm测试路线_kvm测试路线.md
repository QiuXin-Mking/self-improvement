# q
如何使用 virsh 命令查询某个 KVM 虚拟机的 VNC 监听端口？
# a
使用 `virsh vncdisplay <虚拟机名称>` 命令，输出格式类似 `:0` 或 `:1`。例如：
```
virsh vncdisplay my-vm
```

# q
使用 `virsh domifaddr` 无法获取到虚拟机 IP，可能是什么原因？如何解决？
# a
可能是因为虚拟机内缺少 `qemu-guest-agent` 软件包。解决方法：
1. 在宿主机上下载并传输到虚拟机：
   ```
   yumdownloader --resolve qemu-guest-agent
   scp -r ... ...
   ```
2. 进入虚拟机安装：
   ```
   sudo rpm -ivh qemu-guest-agent-*.rpm
   ```
3. 确认服务状态：
   ```
   systemctl status qemu-guest-agent
   ```

# q
在 KVM 虚拟机中如何手动加载 virtio 相关内核模块？
# a
使用 `modprobe` 命令加载模块：
```
modprobe virtio
modprobe virtio_pci
modprobe virtio_ring
modprobe virtio_console
```
