# q
智能早教机项目需要实现哪三个主要学习界面？
# a
- 英语听力界面：语音播放单词，屏幕显示4个单词，用按键选择，正确或错误均有显示。
- 数学学习界面：加减计算，4个页面，每页一个题目和4个答案，正确或错误均有显示。
- 古诗听写界面：古诗跟读，只播放古诗。

# q
项目中对正确和错误操作的视觉与声音反馈是如何设计的？
# a
- 正确：绿灯亮起，播放奖励声音。
- 错误：红灯亮起，播放错误声音。

# q
方案1中，ESP32S3与SD卡通信的默认SPI引脚连接是怎样的？
# a
SD卡 -> ESP32:
- VCC -> 5V
- GND -> GND
- SS (片选) -> IO5 (可通过初始化函数自定义)
- SCK -> IO18
- MISO -> IO19
- MOSI -> IO23

# q
方案1中，MAX98357A音频功放与ESP32的I2S引脚连接是什么？
# a
- VIN 供电
- GND 接地
- GAIN 增益控制（接地）
- SD 关断控制（接VIN）
- DIN (数据输入) -> IO25
- BCLK (位时钟) -> IO26
- LRC (左右声道时钟) -> IO27
初始化代码示例：`amplifier.initI2S(/*_bclk=*/GPIO_NUM_25, /*_lrclk=*/GPIO_NUM_26, /*_din=*/GPIO_NUM_27)`

# q
方案2中JQ8900语音模块的串口和按键的引脚分配是怎样的？
# a
JQ8900:
- BUSY: 播放/空闲指示
- RX: IO43
- TX: IO44

按键:
- K1 -> IO39
- K2 -> IO40
- K3 -> IO41
- K4 -> IO42
