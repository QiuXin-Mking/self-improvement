# q
卷帘门控制系统的防夹保护如何实现？
# a
通过红外对射或压力传感器检测障碍物，触发门体急停或反转。

# q
卷帘门的限位精准定位采用什么方案？
# a
采用霍尔传感器配合磁铁，实现上下极限位置的精确检测。

# q
超声波传感器在卷帘门控制中的作用是什么？
# a
实时检测障碍物距离，达到阈值时触发蜂鸣器鸣叫与LED闪烁报警，并在OLED上显示异常信息。

# q
步进电机开门/关门的控制逻辑是怎样的？
# a
按键2开门对应步进电机正转，按键3关门对应步进电机反转。若红外传感器检测到障碍物则急停，同时蜂鸣器鸣叫、LED闪烁；障碍物移除后，开门动作继续运转至下霍尔传感器检测到磁铁位置，关门动作继续运转至上霍尔传感器检测到磁铁位置。

# q
系统如何显示环境信息？
# a
实时采集温湿度（通过DHT11）和光照强度，数据动态显示在OLED屏幕上。
