# q
如何克隆2048游戏仓库并部署到Nginx的默认网站目录？
# a
使用 ```git clone https://github.com/gabrielecirulli/2048.git``` 克隆仓库，然后通过 ```sudo cp -r 2048 /var/www/html/``` 将游戏目录复制到Nginx文档根目录。

# q
如何递归设置2048游戏目录的所有权为Nginx用户？
# a
执行命令：```sudo chown -R nginx:nginx /var/www/html/2048```

# q
部署后需要执行哪些命令来验证Nginx配置并重启服务？
# a
运行 ```nginx -t``` 测试配置文件语法，然后执行 ```sudo systemctl restart nginx``` 重启Nginx服务。
