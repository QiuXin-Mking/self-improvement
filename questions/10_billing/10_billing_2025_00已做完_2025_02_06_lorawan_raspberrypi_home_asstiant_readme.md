# q
Home Assistant 的官方网站是什么？
# a
https://www.home-assistant.io/installation/

# q
在树莓派上安装 Home Assistant 的官方指南链接有哪些？
# a
- https://www.home-assistant.io/installation/raspberrypi-other
- https://www.home-assistant.io/installation/raspberrypi

# q
示例安装所针对的树莓派型号及使用的镜像文件是什么？
# a
适用于树莓派 4 的 64 位镜像：`haos_rpi4-64-14.2.img.xz`（版本 14.2）

# q
安装完成后，如何访问 Home Assistant？当前存在什么连接限制？
# a
- 访问地址：`http://homeassistant.local:8123/`
- 连接限制：当前无法通过无线连接，只能使用有线连接访问
