# q
在ESP32 Arduino环境中，如何定义串口2默认使用的RX和TX引脚？
# a
通过 `#define` 宏指定引脚号，示例：
```cpp
#define UART2_RX_PIN 16  // 串口2接收引脚
#define UART2_TX_PIN 17  // 串口2发送引脚
```

# q
如何创建一个使用引脚可配置的串口2对象？
# a
使用 `HardwareSerial` 类，传入串口编号2创建全局对象：
```cpp
HardwareSerial Serial2(2);
```

# q
在 `setup()` 中初始化串口2需要调用哪个方法，常用参数是什么？
# a
调用 `Serial2.begin()`，指定波特率、串口配置和自定义引脚，示例：
```cpp
Serial2.begin(115200, SERIAL_8N1, UART2_RX_PIN, UART2_TX_PIN);
```

# q
如何让ESP32的串口2每隔1秒发送一条周期性消息？
# a
在 `loop()` 中使用 `delay(1000)` 并调用 `Serial2.println()`：
```cpp
void loop() {
  delay(1000);
  Serial2.println("这是来自ESP32串口2的周期性消息。");
}
```
