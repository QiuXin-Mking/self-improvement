# q
卷帘门控制系统中，防夹保护功能通过哪些传感器实现，触发后门体执行什么动作？
# a
防夹保护使用红外对射或压力传感器检测障碍物，触发后门体急停或反转。

# q
如何实现手动/自动模式切换？
# a
通过按键指令进行模式切换。

# q
远程控制功能支持哪些操作？
# a
遥控器无线控制门体的开闭和状态查询。

# q
障碍物距离检测与分级报警是如何实现的？
# a
使用超声波传感器实时测距，当距离达到阈值时，触发近距蜂鸣器报警和LED闪烁。

# q
环境监测与显示功能具体包含哪些参数，显示在什么设备上？
# a
实时采集温湿度（使用DHT11）和光照强度（使用光照传感器），并通过LCD或OLED屏幕动态显示。
