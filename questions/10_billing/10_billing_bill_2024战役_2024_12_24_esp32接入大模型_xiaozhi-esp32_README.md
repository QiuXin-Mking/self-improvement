# q
小智 AI 聊天机器人项目基于什么开发框架？
# a
基于乐鑫的 ESP-IDF 进行开发。

# q
小智 AI 聊天机器人支持哪些语言识别？
# a
支持国语、粤语、英语、日语、韩语 5 种语言识别，使用 SenseVoice 实现。

# q
小智 AI 聊天机器人使用了哪些主要 AI 能力？
# a
- 离线语音唤醒：ESP-SR
- 语音识别（ASR）：SenseVoice（支持5种语言）
- 声纹识别：3D Speaker
- 大模型 TTS：火山引擎 或 CosyVoice
- 大模型 LLM：Qwen2.5 72B 或 豆包 API

# q
小智 AI 聊天机器人用什么方式实现流式语音对话？
# a
通过 WebSocket 或 UDP 协议实现流式语音对话。

# q
项目推荐新手首次烧录固件的方式是什么？
# a
先使用免开发环境烧录的固件，不用搭建 IDF 开发环境，固件连接作者提供的测试服。推荐参考《Flash烧录固件（无IDF开发环境）》教程。
