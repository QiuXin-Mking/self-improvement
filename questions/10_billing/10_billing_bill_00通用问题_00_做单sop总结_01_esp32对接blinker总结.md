# q
ESP32对接Blinker时，如何找到可用的母版代码？
# a
在项目目录中找到 `18_fa_03_安防.ino` 文件，该文件即为 ESP32 母版代码。

# q
ESP32对接Blinker时，如何正确获取并配置密钥？
# a
在 App 的 Blinker 框架中找到对应的密钥，将其拷贝到 ESP32 代码中的密钥配置位置。

# q
ESP32对接Blinker时，若WiFi连接失败，应优先检查什么？
# a
检查 WiFi 参数配置，一般默认为家庭 WiFi，确认 SSID 和密码是否正确。

# q
ESP32对接Blinker做单前，需要确认哪些控制模组的数量？
# a
需要确认数字模组、滑块模组和按键控制模组的数量，确保代码中配置与实际需求一致。
