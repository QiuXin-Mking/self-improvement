# q
该单片机停车系统使用了哪些硬件外设？
# a
使用了 oled1 和 oled2 两块显示屏，4 个按键，以及串口 1 用于通信。

# q
车辆入库时 OLED1 的显示与处理流程是什么？
# a
OLED1 初始显示剩余车位 300 个。当红外被遮挡时，显示“need input 车牌”；按键输入车牌后，记录车牌与当前时间，剩余车位减 1。

# q
车辆出库时 OLED2 的计费与处理流程是什么？
# a
出口红外被遮挡时，OLED2 显示“need input 车牌”；输入车牌后，显示停车秒数和费用，提示“请缴费”；缴费完成后启动（抬杆放行）。

# q
文档中提到的车位容量与存储方式是什么？
# a
实际停车位为 10 个，使用一个长度为 10 的 int 数组存储车牌信息。
