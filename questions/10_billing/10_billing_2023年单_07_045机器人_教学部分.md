# q
贝雷帽（Beret）板载RTC使用的振荡器是什么型号和工作频率？
# a
贝雷帽采用 SiTime SIT1532AC-J5-DCC-32.768E MEMS 振荡器，工作频率为 32.768 kHz，用于产生 OSC32 时钟，同时驱动 STM32 的 RTC 和 ICM-42688-P IMU。

# q
在 VBAT 模式下按下复位按钮会触发哪个引脚，其作用是什么？
# a
当贝雷帽处于 VBAT 模式（低功耗睡眠，仅由 VL-1220 纽扣电池供电）时，按下复位按钮会将 PE6 引脚上的 RTC_TAMP3 通道拉低，作为唤醒事件之一。

# q
STM RTC 模块提供了哪几种方式将电路板从 VBAT 模式唤醒？
# a
唤醒方式包括：RTC_TAMP3 引脚被拉低（例如复位按钮）、RTC 定时唤醒、或通过 LPUART 通道接收唤醒信号。唤醒时利用 PC13 引脚的 RTC_OUT1 输出控制理想二极管控制器使能，进而打开主功率 MOSFET，自动恢复 5V 和 3.3V Vreg 以及 STM32 的供电。

# q
贝雷帽上 RTC_OUT1 通道（PC13 引脚）的具体作用是什么？
# a
RTC_OUT1 通道连接到理想二极管控制器的使能引脚，该控制器再驱动主功率 MOSFET。当 RTC 产生唤醒信号时，RTC_OUT1 会使能理想二极管控制器，从而打开 MOSFET，恢复板上 5V 和 3.3V 电源轨及 STM32 运行。

# q
STM32 RTC 的基本工作原理是什么？
# a
RTC 是一个连续计数的 32 位计数器，通常由外部 32.768 kHz LSE 晶振经预分频后提供时钟，包括 RTC_CNT（计数器）、RTC_ALR（闹钟）和 RTC_CR 等寄存器，最大计数值为 2^32-1。后备寄存器（BKP）用于保存数据。
