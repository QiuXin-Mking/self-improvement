# q
树莓派中用于查看 GPIO 引脚布局和状态的命令是什么？
# a
使用 `gpio readall` 命令可以查看当前所有 GPIO 引脚的编号、模式和电平状态。

# q
如何查看树莓派上安装的操作系统版本信息？
# a
通过执行 `cat /etc/os-release` 可以查看树莓派操作系统的发行版名称、版本号等详细信息。

# q
树莓派 3 的板载串口（UART）为何不稳定，并常与蓝牙功能冲突？
# a
树莓派 3 的 CPU 内部有两个串口：硬件串口（PL011 UART）和迷你串口（mini-uart）。硬件串口被蓝牙模块占用，而分配给 GPIO 的串口是迷你串口。迷你串口没有独立的时钟源，依赖可变的内核频率，导致其波特率不稳定，因此无法正常使用，且板载串口与蓝牙不能同时使用。

# q
在树莓派上使用外部串口设备（如 /dev/ttyAMA0）时，如何快速赋予读写权限？
# a
执行 `sudo chmod 777 /dev/ttyAMA0` 可以赋予所有用户对该串口设备的读写执行权限。
