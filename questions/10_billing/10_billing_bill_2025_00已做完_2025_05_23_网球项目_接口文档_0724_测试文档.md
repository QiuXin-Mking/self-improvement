# q
测试中通过MQTT向网球设备发送控制指令的Topic格式是什么？
# a
基础Topic为 `wq/v1.0/9167`，也可使用子Topic `wq/v1.0/9167/x`。

# q
控制消息的JSON负载包含哪些字段，各表示什么含义？
# a
负载字段包括：
- `motion`：运动模式，如 `"1"` 被注释为“正”（正转）。
- `speed`：速度值，整数范围推测为0–100。
- `user_id`：用户标识，如 `"U100"`。
- `user_name`：用户名，如 `"Test"`。

示例：
```
mosquitto_pub -h 101.201.53.141 -p 1883 -t "wq/v1.0/9167" -m '{"motion":"1", "speed":100, "user_id":"U100", "user_name":"Test"}'
```

# q
根据测试注释，speed参数的几个关键经验值是什么？
# a
- `speed:70`：可以正常运转。
- `speed:80`：转动较慢。
- `speed:90`：速度很快。
- `speed:0`：停止。
