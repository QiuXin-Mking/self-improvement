# q
JQ8900 语音模块发送“上一首”播放命令的字节序列是什么？
# a
```text
0xAA, 0x06, 0x00, 0xB0
```

# q
JQ8900 语音模块发送“下一首”播放命令的字节序列是什么？
# a
```text
0xAA, 0x06, 0x00, 0xB0
```
（注：文档中“下一首”与“上一首”命令相同）

# q
通过 Arduino Serial2 向 JQ8900 发送播放第 3 首曲目的指令，对应的命令序列和代码示例是什么？
# a
命令字节序列：  
```text
0xAA, 0x07, 0x02, 0x00, 0x03, 0xB6
```
代码示例：
```cpp
Serial2.write(0xAA);
Serial2.write(0x07);
Serial2.write(0x02);
Serial2.write(0x00);
Serial2.write(0x03);
Serial2.write(0xB6);
```

# q
使用 JQ8900 模块播放 U 盘中的指定曲目时，U 盘内的文件命名有何要求？
# a
U 盘上的音频文件必须使用 5 位数进行命名（例如 `00001.mp3`），指令中指定的曲目编号对应文件名的数字部分，如播放第 3 首即指向 `00003`。
