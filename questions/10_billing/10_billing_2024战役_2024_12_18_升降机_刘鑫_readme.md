# q
升降机系统中，显示屏、小主机、单片机和传感器之间的硬件连接架构是什么？
# a
显示屏 <--> 小主机(J6412) <--> ESP32/STM32 <--> 一堆传感器 + ADC

# q
该升降机项目上云采用的通信架构是什么？
# a
UI <--- mqtt <--- web服务器 <--- 上云

# q
现阶段升降机设备（不含上云）的报价是多少？
# a
1万5千元（15000）

# q
升降机中6个红外传感器分别用于检测哪些状态？
# a
设备收折、设备打开、左侧摆臂抬起、右侧摆臂抬起、设备收折上行、设备收折下行

# q
MPU6050陀螺仪通过加速度变化检测哪些碰撞事件？
# a
上行翘板碰撞（右型）、下行翘板碰撞（右型）、上行侧壁碰撞（右型）、下行侧壁碰撞（右型）
