# q
LTC4150的INT引脚每个下降沿脉冲代表多少电荷和电量？
# a
每个INT下降沿脉冲代表 0.614439 库仑（C）的电荷，即 0.1707 毫安时（mAh）。计数变量 `num` 记录脉冲个数。

# q
LTC4150模块的POL引脚如何指示充放电状态？
# a
POL引脚为高电平时表示电流流入电池（充电状态），低电平时表示电流流出电池（放电状态）。

# q
如何将模块的默认供电电压从5V改为3.3V？
# a
短接焊盘 SJ2 和 SJ3，即可将供电电压切换为 3.3V。

# q
SJ1焊盘的作用是什么？
# a
SJ1默认断开；短接后，可以对INT引脚的中断信号自动清除（自动清中断）。
