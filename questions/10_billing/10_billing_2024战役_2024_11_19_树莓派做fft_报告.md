# q
在这个频谱分析仪项目中使用的ADC芯片型号是什么？
# a
MCP3208（文档中“修改后”部分纠正了最初错误写的 MC3208）

# q
MCP3208与树莓派的SPI接口引脚应如何连接？
# a
- DIN (Data In) → 树莓派 MOSI (Pin 19, GPIO10)
- DOUT (Data Out) → 树莓派 MISO (Pin 21, GPIO9)
- CLK (Clock) → 树莓派 SCLK (Pin 23, GPIO11)
- CS (Chip Select) → 树莓派任一指定为片选的 GPIO 引脚
- V_DD 接 3.3V 电源，GND 接地
- V_REF 决定输入电压范围，通常也接 3.3V

# q
设置ADC采样率时需要遵循什么准则？为什么？
# a
需要遵循奈奎斯特准则（Nyquist criterion），以保证采样后的数字信号能够准确复现原始模拟信号的频率成分。

# q
在这个项目中，FFT（快速傅里叶变换）被用于实现什么功能？
# a
将时域采样信号转换为频域表示，分析信号的频率分量，并可用于生成频谱图像或保存频域数据至文件。
