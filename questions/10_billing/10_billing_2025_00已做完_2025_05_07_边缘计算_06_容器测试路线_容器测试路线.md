# q
如何将本地 Docker 镜像导出为 tar 归档文件？
# a
使用 `docker save` 命令，例如：
```bash
docker save -o ubuntu-latest.tar ubuntu:latest
```
该命令会将 `ubuntu:latest` 镜像保存为 `ubuntu-latest.tar` 文件。

# q
如何从 tar 归档文件加载 Docker 镜像？
# a
使用 `docker load` 命令，例如：
```bash
docker load -i ubuntu-latest.tar
```
该命令会导入 tar 文件中的镜像到本地 Docker 镜像库。

# q
如何运行一个持续运行的后台容器，避免启动后立即退出？
# a
可以使用 `tail -f /dev/null` 作为保持运行的命令，例如：
```bash
docker run -d ubuntu:latest tail -f /dev/null
```
也可以先安装必要工具再保持运行：
```bash
docker run -d ubuntu:latest sh -c "apt update && apt install -y coreutils && tail -f /dev/null"
```

# q
如何基于正在运行的容器创建新镜像？
# a
先对容器进行修改（如安装软件），然后使用 `docker commit` 命令，例如：
```bash
docker commit 48e59cefaf69 ubuntu:1.0
```
这会将容器 ID `48e59cefaf69` 的当前状态保存为新镜像 `ubuntu:1.0`，之后可以用 `docker save` 导出该镜像。
```bash
docker save -o ubuntu:1.0.tar ubuntu:1.0
```
