# q
该刹车系统使用什么类型的传感器进行测距？
# a
使用光电传感器（广电），明确不用霍尔传感器。

# q
系统共有几个功能页面？请列出各页面名称和主要功能。
# a
共有4个页面：
- page1：按键测试，测试按键是否有效。
- page2：安全距离测试，小于安全距离则报警（蜂鸣器叫、LED亮）；按键2/3增减安全阈值，按键4将当前距离设为安全阈值方便测试。
- page3：速度测试，显示时间、距离、速度。
- page4：速度限制，显示速度和速度限制；按键2/3增减速度阈值，按键4将当前距离设为速度阈值；大于速度阈值报警（蜂鸣器叫、LED亮）。

# q
系统使用哪个主控MCU？输出设备有哪些？
# a
主控为STM32。输出设备包括OLED显示屏、蜂鸣器和LED。

# q
如何快速将当前测量距离设置为安全阈值？
# a
在page2（安全距离测试）页面，按下按键4即可将当前距离写入安全阈值。

# q
速度阈值报警的触发条件是什么？
# a
在page4页面，当实测速度大于设定的速度阈值时，蜂鸣器鸣叫且LED亮起，触发报警。
