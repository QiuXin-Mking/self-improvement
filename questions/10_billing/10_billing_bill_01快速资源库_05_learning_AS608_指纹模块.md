# q
AS608 指纹模块必须使用什么电压供电？
# a
只能用 3.3V 电源，不能使用 5V。

# q
AS608 指纹模块的 VCC、TX、RX、GND 引脚分别对应什么颜色的杜邦线？
# a
- VCC → 红色
- TX  → 黄色  
- RX  → 白色  
- GND → 黑色
