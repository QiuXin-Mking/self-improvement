# q
如何查看树莓派的 GPIO 引脚状态？
# a
使用命令 `gpio readall`。

# q
在树莓派上操作串口时，如何获取设备访问权限？（至少列出两种方法）
# a
第一种：`sudo chmod 777 /dev/ttyAMA0` 赋予所有用户读写执行权限；第二种：用 `sudo` 运行程序，例如 `sudo ./a.out`。

# q
树莓派 3 中为什么板载串口和蓝牙不能同时使用？
# a
树莓派 CPU 内部有两个串口：硬件串口（PL011 UART）和迷你串口（mini-uart）。在树莓派 3 上，硬件串口被蓝牙模块占用，仅将没有时钟源的迷你串口分配给 GPIO 串口；由于内核频率本身会变化，迷你串口的速率不稳定，导致无法正常使用。
