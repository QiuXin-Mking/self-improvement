# q
如何在 Thonny IDE 中为 ESP32 烧录 MicroPython 固件？
# a
1. 点击 Thonny 右下角解释器区域，选择 configure interpreter  
2. 进入 interpreter 选项卡  
3. 点击 Install or update Micropython  
4. 在 Port 中选择对应端口，在 Firmware 中选择固件文件（如 `esp32-20220618-v1.19.1.bin`）  
5. 点击安装即可烧录
