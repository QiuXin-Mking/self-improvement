# q
STM32最小系统通常包含哪些基本电路与外设？
# a
STM32最小系统一般包括晶振电路、上拉和下拉电阻配置、下载引脚（SWD/JTAG）以及串口（UART）。

# q
硬件IIC与软件IIC的主要区别是什么？
# a
硬件IIC通过配置寄存器实现，精度高；软件IIC由开发者自行编写时序，精度较低但实现灵活、好用。

# q
ULN2003A达林顿管用于弱电控制强电时，其输入与输出的逻辑关系是什么？
# a
输入1时输出0，输入0时输出1。通过该反相逻辑可实现0‑3.3V到0‑5V的电平转换，实现弱电控制强电。

# q
IIC写操作的典型流程是什么？根据文中代码，给出对应的写入字节函数实现。
# a
流程：主设备发送起始条件 → 发送从机地址（写位0）→ 从设备回应ACK → 主设备发送寄存器地址 → 从设备回应ACK → 主设备发送数据字节 → 从设备回应ACK → 主设备发送停止条件。
代码示例：
```c
void OLED_I2C_WriteByte(uint8_t addr, uint8_t data)
{
    OLED_IIC_Start();
    OLED_IIC_Send_Byte(OLED_ADDRESS);
    OLED_IIC_Wait_Ack();
    OLED_IIC_Send_Byte(addr);
    OLED_IIC_Wait_Ack();
    OLED_IIC_Send_Byte(data);
    OLED_IIC_Wait_Ack();
    OLED_IIC_Stop();	
}
```

# q
使用STM32 LL库初始化USART1时，如何配置波特率、数据宽度、停止位等关键参数？
# a
关键参数配置示例：
```c
USART_InitStruct.BaudRate = 115200;
USART_InitStruct.DataWidth = LL_USART_DATAWIDTH_8B;
USART_InitStruct.StopBits = LL_USART_STOPBITS_1;
USART_InitStruct.Parity = LL_USART_PARITY_NONE;
USART_InitStruct.TransferDirection = LL_USART_DIRECTION_TX_RX;
USART_InitStruct.HardwareFlowControl = LL_USART_HWCONTROL_NONE;
USART_InitStruct.OverSampling = LL_USART_OVERSAMPLING_16;
LL_USART_Init(USART1, &USART_InitStruct);
LL_USART_ConfigAsyncMode(USART1);
LL_USART_Enable(USART1);
```
