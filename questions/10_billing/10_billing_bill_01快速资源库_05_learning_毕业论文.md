# q
毕业论文第一章（绪论）需要包含哪些核心内容？
# a
第一章应包含：学术背景说明（通过知网查相关论文）、落地背景/应用场景（如智能门禁、智能广告，阐述现存问题、问题严重性及你解决的问题）、纵向课题（展示学术成就，查同类型研究生论文，指出其不足并说明自身意义）和横向课题（展示商业价值，找同类型项目，指出不足，讲商业前景、需求与可行性验证，避免唯我独尊）。

# q
毕业论文的硬件章节（第三章）通常怎么写？
# a
第三章主要介绍硬件模块，包括：贴模块实物图并讲解、整理各模块资料（如STM32、UART2、LCD1602、步进电机）、给出原理图并描述电路连接，并说明相关技术细节（如STM32的8种GPIO模式、推挽输出、NVIC中断分组、UART/USART、IIC、SPI协议等）。

# q
毕业论文软件章节（第四章）的写作要点有哪些？
# a
第四章介绍软件逻辑，需分层次描述（驱动层、应用代码层、用户代码层），并重点讲解用户代码层（因人而异）。必须使用Visio绘制加粗文字的正方形流程图，并配以文字论述。

# q
毕业论文的软硬件联调与测试章节（第五章）应如何展示工作量？
# a
第五章要体现软硬件联调的难点和工作量，多贴测试实物图、展示核心代码片段，以凸显实现过程中的付出和不易。

# q
撰写工科毕业论文常用的软件工具有哪些？
# a
常用软件包括：VSCode、Keil5 MDK（STM32开发）、Altium Designer（原理图与PCB设计）、Keil5（51单片机开发）和Proteus 8.9（仿真）。
