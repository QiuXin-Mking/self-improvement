# q
如何用希尔伯特变换计算信号的包络？
# a
先对信号进行希尔伯特变换得到解析信号，再取解析信号的绝对值即为包络。在代码中，`analytical_signal = hilbert(filtered_signal)`，随后 `envelope = np.abs(analytical_signal)`。

# q
这段代码中如何设计用于提取载波附近成分的带通滤波器？
# a
使用 `scipy.signal.butter` 设计 1 阶巴特沃斯带通滤波器，参数为截止频率 [95, 105] Hz（采样率 fs=1000 Hz）。具体实现：先计算奈奎斯特频率 `nyquist = 0.5 * fs`，归一化截止频率 `low = 95 / nyquist`，`high = 105 / nyquist`，然后调用 `b, a = butter(1, [low, high], btype='band')`，最后用 `filtfilt(b, a, signal)` 进行零相位滤波。

# q
包络谱是如何计算得到的？
# a
对包络信号 `envelope` 执行快速傅里叶变换（FFT）得到频谱，取绝对值作为幅度谱，并只保留前半部分频谱（正频率部分）；频率轴则通过 `np.fft.fftfreq(len(envelope), 1/fs)` 获得，同样取前半部分。代码：
```
envelope_spectrum = np.fft.fft(envelope)
envelope_spectrum_magnitude = np.abs(envelope_spectrum)[:len(envelope) // 2]
frequencies = np.fft.fftfreq(len(envelope), 1/fs)[:len(envelope) // 2]
```
