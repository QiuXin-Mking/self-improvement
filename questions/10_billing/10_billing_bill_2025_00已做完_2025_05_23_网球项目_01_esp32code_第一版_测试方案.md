# q
如何使用 mosquitto_pub 向指定 MQTT 服务器发布消息？
# a
使用 `mosquitto_pub` 命令，指定主机地址 `-h`、端口 `-p`、主题 `-t` 和消息内容 `-m`。示例：
```
mosquitto_pub -h 101.201.53.141 -p 1883 -t "wq/v1.0/9167" -m '{"motion":"1", "speed":50, "user_id":"U100", "user_name":"Test"}'
```

# q
示例中 MQTT 消息的主题格式是什么？
# a
主题为 `wq/v1.0/9167`，看起来采用 `产品/版本/设备ID` 的结构。

# q
消息 payload 中包含哪些关键字段？
# a
JSON 格式，包含 `motion`（动作编号）、`speed`（速度）、`user_id` 和 `user_name`。例如 `{"motion":"1", "speed":50, "user_id":"U100", "user_name":"Test"}`。
