# q
该光伏发电加热系统集成了哪些环境传感器？
# a
系统集成了温湿度传感器、光照传感器、烟雾传感器和 CO₂ 传感器，用于实时采集温度、湿度、光强、烟雾浓度和 CO₂ 浓度数据。

# q
当温度超过设定阈值时，系统会执行哪些联动动作？
# a
温度超过上限时，加热片停止加热，同时 LED1 灯亮起，蜂鸣器鸣叫报警。

# q
在哪些条件下系统会自动开启风扇进行通风？
# a
当湿度超过上限、烟雾浓度超过阈值或 CO₂ 浓度超过上限时，均会触发风扇开启通风，同时伴随蜂鸣器报警。

# q
手机 APP 支持哪些控制和监测功能？
# a
APP 可实时显示温度、湿度、光强、烟雾、CO₂ 数据；支持远程设置各类传感器阈值；可手动控制蜂鸣器、加热、风扇、LED；具备定时加热功能，可设定若干秒后自动开启或关闭加热；还能监测光伏板的电压、电流、电量等电气参数。
