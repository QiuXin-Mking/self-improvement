# q
单片机停车系统使用了哪些主要外设？
# a
系统包含两个 OLED 显示屏（oled1 作为入口显示屏，oled2 作为出口显示屏）、四个按键以及串口 1。

# q
入口的车辆处理流程是什么？
# a
当入口红外传感器被挡住时，oled1 显示 `need input 车牌`。输入车牌后系统记录入场时间，并将剩余车位数减 1。

# q
出口的车辆处理流程是什么？
# a
当出口红外传感器被挡住时，oled2 显示 `need input 车牌`。输入车牌后显示停车秒数和费用，并提示缴费；缴费后道闸启动放行。

# q
系统使用什么数据结构存储停车记录？最多支持多少条？
# a
使用一个包含 10 个 int 的数组存储，实际最多支持 10 条停车记录。
