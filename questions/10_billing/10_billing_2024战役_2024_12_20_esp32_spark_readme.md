# q
3D打印机出现拉面条现象时，如何通过调平解决？
# a
1. 保证 Z 轴在 0 刻度。
2. 调节 Z 轴补偿，直到 A4 纸卡在喷头与平台之间无法移动。
3. 分别调整平台左上、右上、左下、右下 4 个角的弹簧，使平台下移。
4. 确保 5 个点（四角 + 中心）在同一水平面。
