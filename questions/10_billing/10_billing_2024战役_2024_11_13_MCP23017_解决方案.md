# q
I²C 协议使用哪两根信号线，它们的作用分别是什么？
# a
- **SDA（串行数据线）**：双向，用于传输数据，所有设备连接到同一条线上。
- **SCL（串行时钟线）**：由主设备生成时钟信号，控制数据传输速率。

# q
I²C 通信的启动条件和停止条件分别如何定义？
# a
- **启动条件**：SCL 为高电平时，SDA 从高电平变为低电平。
- **停止条件**：SCL 为高电平时，SDA 从低电平变为高电平。

# q
I²C 总线常见的速度模式有哪些？对应的频率是多少？
# a
- 标准模式：100 kHz
- 快速模式：400 kHz
- 高速模式：3.4 MHz
（还有快速模式+ 1 MHz 等）

# q
I²C 总线的 SDA 和 SCL 线为何需要上拉电阻？
# a
上拉电阻将 SDA 和 SCL 线拉到高电平，使总线在空闲时保持确定的高电平状态，因为 I²C 设备只能拉低线路或释放线路（开漏输出），需外部电阻配合实现高电平。

# q
如何使用示波器判断 I²C 总线是否工作在标准模式（100 kHz）？
# a
1. 将示波器探头连接到 SCL 线上，地线接公共地。
2. 设置时基为 10 µs/div 或 20 µs/div。
3. 在总线通信时观察 SCL 方波波形。
4. 测量信号频率（或周期，频率 = 1/周期）：若频率接近 100 kHz，则为标准模式；若接近 400 kHz 则为快速模式等。
