# q
在ESP32的Arduino代码中，如何定义串口2的接收和发送引脚？
# a
通过宏定义指定引脚号，例如：
```cpp
#define UART2_RX_PIN 16  // 串口2接收引脚
#define UART2_TX_PIN 17  // 串口2发送引脚
```
然后在初始化 `Serial2.begin()` 时传入这些引脚。

# q
如何创建并初始化ESP32的串口2对象（波特率115200，数据位8，无校验，1停止位）？
# a
```cpp
HardwareSerial Serial2(2); // 使用串口编号2
void setup() {
  Serial2.begin(115200, SERIAL_8N1, UART2_RX_PIN, UART2_TX_PIN);
}
```

# q
在ESP32上使用串口2发送一行数据的典型方法是什么？
# a
使用 `Serial2.println()` 方法，例如：
```cpp
Serial2.println("Hello from Serial2!");
```

# q
在ESP32上如何调整串口2的发送和接收缓冲区大小？
# a
可在初始化后调用：
```cpp
Serial2.setTxBufferSize(256);
Serial2.setRxBufferSize(256);
```
