# q
如何查询 Auto Scaling 组 webapp-asg 内的 EC2 实例列表？
# a
使用以下命令：
```bash
aws autoscaling describe-auto-scaling-groups \
   --auto-scaling-group-names webapp-asg \
   --query 'AutoScalingGroups[0].Instances'
```

# q
如何获取特定 EC2 实例的公网 IP 地址？
# a
使用 `describe-instances` 命令，指定实例 ID 并查询 `PublicIpAddress`，输出为纯文本：
```bash
aws ec2 describe-instances \
  --instance-ids i-04f389d687b74f455 \
  --query 'Reservations[0].Instances[0].PublicIpAddress' \
  --output text
```
示例输出：`174.129.104.86`

如何通过 SSH 登录到 EC2 实例？
# a


# q
如何在 Auto Scaling 组中更新启动模板并触发实例滚动替换？
# a
分三步完成：
1. 创建启动模板的新版本（例如基于版本 5）：
```bash
aws ec2 create-launch-template-version \
  --launch-template-id lt-0cf76dc39448990ab \
  --source-version 5 \
  --launch-template-data "UserData=$(base64 < userdata.sh | tr -d '\n')"
```

2. 更新 ASG 使用新版本（例如版本 6）：
```bash
aws autoscaling update-auto-scaling-group \
  --auto-scaling-group-name webapp-asg \
  --launch-template "LaunchTemplateId=lt-0cf76dc39448990ab,Version=6"
```

3. 启动实例刷新：
```bash
aws autoscaling start-instance-refresh \
  --auto-scaling-group-name webapp-asg
```
