# q
如何使用 mosquitto_pub 向远程 MQTT Broker 发布一条 JSON 消息？
# a
```bash
mosquitto_pub -h 101.201.53.141 -p 1883 -t "wq/v1.0/9167" -m '{"motion":"050", "speed":50, "user_id":"U100", "user_name":"Test"}'
```
-h 指定 Broker 地址，-p 指定端口（1883 为默认 MQTT 端口），-t 为主题，-m 为消息内容。

# q
示例中发布的 MQTT 消息体的 JSON 结构包含哪些字段？
# a
```json
{"motion":"050", "speed":50, "user_id":"U100", "user_name":"Test"}
```
字段含义：motion 动作码（如 "050"），speed 速度值（整数），user_id 用户ID，user_name 用户名。

# q
如果 MQTT Broker 部署在本机，发布命令应如何修改？
# a
将 -h 参数改为 localhost，例如：
```bash
mosquitto_pub -h localhost -p 1883 -t "wq/v1.0/9167" -m '{"motion":"050", "speed":50, "user_id":"U100", "user_name":"Test"}'
```
