# q
智能门禁系统的门锁执行机构选用了什么方案？具体模块及工作原理是什么？
# a
选用电磁铁方案，使用 Keyes 直流吸盘式电磁铁模块。关门时模块通电产生吸力，开门时断电吸力消失。

# q
本系统的人脸辨识功能用到了哪些算法或框架？各自完成什么任务？
# a
- 检测图片中是否有人脸：使用 OpenCV Face。
- 人脸分类（区分户主/非户主）：使用 YOLOv5，实施难度较大。

# q
主机与安保人员端的通讯使用哪个模块？从机端有哪些外设？
# a
通讯使用 nRF24L01 无线串口模块，主机与从机之间进行数据传输。从机端外设包括：
- 低电平触发蜂鸣器
- LED
- 复位按键

# q
系统主机端用到了哪些核心硬件？
# a
- 树莓派 4b（8G 版本）
- STM32F103C8T6
- AS608 光学指纹识别模块
- Raspberry Pi CSI 摄像头（500 万像素，15cm 软排线）
- Keyes 直流吸盘式电磁铁模块
- 32G SD 卡、读卡器、type-c 供电线

# q
指纹识别失败后的处理策略是什么？
# a
当指纹识别连续失败三次后，系统会自动录制视频。
