# q
这段代码实现了什么信号处理流程？
# a
代码实现了一个基于希尔伯特变换的包络谱分析流程：首先生成一个载波频率 100 Hz、调制频率 5 Hz、调制指数 0.5 的调幅信号，然后通过窄带带通滤波器（95‑105 Hz）提取载波附近频段，接着利用希尔伯特变换求取包络，最后对包络做 FFT 获得包络谱，并在 0‑50 Hz 范围内显示调制特征。

# q
窄带带通滤波器是如何设计并应用的？
# a
使用 `scipy.signal.butter` 设计一阶巴特沃斯带通滤波器，给定采样频率 `fs=1000 Hz`，低截止 `lowcut=95 Hz`，高截止 `highcut=105 Hz`，归一化截止频率为 `low = 95/500 = 0.19`，`high = 105/500 = 0.21`。滤波器系数 `b, a` 再通过 `filtfilt` 进行零相位滤波，得到 `filtered_signal`。

# q
如何利用希尔伯特变换提取信号的包络？
# a
调用 `scipy.signal.hilbert` 对滤波后的信号进行希尔伯特变换，得到解析信号 `analytical_signal`，然后取其绝对值 `envelope = np.abs(analytical_signal)`，即得到信号的包络。

# q
包络谱的计算步骤和关注频段是什么？
# a
对包络信号做快速傅里叶变换：`envelope_spectrum = np.fft.fft(envelope)`，取幅度谱正频率部分 `np.abs(envelope_spectrum)[:len(envelope)//2]`，频率轴由 `np.fft.fftfreq(len(envelope), 1/fs)[:len(envelope)//2]` 给出。绘图时通过 `plt.xlim(0, 50)` 将注意力集中在 0‑50 Hz 的低频调制信息，从而清晰观察 5 Hz 的调制频率。
