# q
2048 这个游戏项目是克隆自哪个游戏的？
# a
2048 是 [1024](https://play.google.com/store/apps/details?id=com.veewo.a1024) 的小型克隆，基于 [Saming's 2048](http://saming.fr/p/2048/)（同样是一个克隆），间接受到 [Threes](https://asherv.com/threes/) 的启发。

# q
2048 游戏采用什么开源许可证？
# a
2048 采用 MIT 许可证，许可证文件地址为：https://github.com/gabrielecirulli/2048/blob/master/LICENSE.txt

# q
向 2048 项目贡献代码时，应该向哪个分支发起 Pull Request？
# a
应该向 `master` 分支发起 Pull Request，并且建议在特定分支中进行修改，确保游戏完全正常运行后再发送 PR。
