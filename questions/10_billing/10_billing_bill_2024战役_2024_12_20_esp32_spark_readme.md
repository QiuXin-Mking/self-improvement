# q
3D 打印机出现拉面条一样的现象时如何通过调平解决？
# a
1. 保证Z轴在0刻度。  
2. 调节到零点，调整Z轴补偿，使喷嘴卡住底面，A4纸抽不动。  
3. 依次调整左上、右上、左下、右下四个点，通过弹簧缩紧让平台下移。  
4. 确保这五个点（中心加四角）在同一水平面。

# q
文档记录的 3D 打印机喷嘴和热床参考温度是多少？

热床（机床）温度：20°C（不加热），且不使用平台附着。  
打印机型号为 Ender-3 Max Neo。
