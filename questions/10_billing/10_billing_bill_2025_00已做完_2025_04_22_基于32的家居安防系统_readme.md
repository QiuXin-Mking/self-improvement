# q
该家居安防系统包含哪些类型的异常检测传感器？
# a
入侵检测（红外传感器）、火灾检测、天然气检测（浓度）、室内温度、室内湿度。其中火灾和入侵为离散异常检测，温度、湿度、天然气为数值上限超限检测。

# q
当检测到异常时，系统采用哪些本地报警方式？
# a
本地报警使用蜂鸣器 + LED 进行声光提示。

# q
系统除了本地报警外，还具备什么远程通知能力？
# a
每一个数据异常时，系统会远程报警，将报警信息推送到主人手机 App 的监控界面上。

# q
用户可以通过 App 实现哪些主要功能？（至少列出 5 项）
# a
1. 显示实时温度
2. 显示实时湿度
3. 显示实时天然气检测数值
4. 显示是否发生火灾
5. 显示是否有人入侵
6. 调整温度报警上限
7. 调整湿度报警上限
8. 调整天然气报警上限
