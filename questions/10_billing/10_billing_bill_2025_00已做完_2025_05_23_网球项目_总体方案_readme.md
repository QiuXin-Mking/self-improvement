# q
该网球项目使用的MQTT broker连接参数是什么？
# a
MQTT broker的公网IP为 `101.201.53.141`，端口为 `1883`，主题（Topic）格式为 `wq/v1.0/{电话后4位}`，其中 `{电话后4位}` 需要替换为对应用户电话号码的后四位数字。

# q
消息体采用什么格式？包含哪些字段及其技术约束？
# a
消息体为JSON格式，包含以下字段：
- `motion`：字符串类型，动作代码，取值范围 `0-100`，固定3位宽，不足前面补零（例如 `001`）；
- `speed`：整数类型，速度值，取值范围 `0-100`，固定3位宽，不足补零；
- `user_id`：用户ID；
- `user_name`：用户名称。

# q
`motion` 字段的三组动作代码分别对应什么操作？
# a
- `"001"` → 还原动作；
- `"002"` → 捡球动作；
- `"003"` → 急停动作；
`speed` 的取值由UI滑块决定。
