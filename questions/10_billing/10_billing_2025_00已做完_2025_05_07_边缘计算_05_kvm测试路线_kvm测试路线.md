# q
如何使用 virsh 命令查看 KVM 虚拟机的 VNC 端口？
# a
```bash
virsh list --all         # 获取虚拟机 ID 或名称，例如 my-vm
virsh vncdisplay my-vm   # 输出类似 :0 或 :1
```

# q
如果 `virsh domifaddr my-vm` 无法获取虚拟机的 IP 地址，可能是什么原因？如何解决？
# a
可能是缺少 `qemu-guest-agent` 包。解决方法：
```bash
yumdownloader --resolve qemu-guest-agent
scp -r ...               # 将下载的 RPM 包复制到虚拟机
sudo rpm -ivh qemu-guest-agent-*.rpm
```

# q
如何检查 `qemu-guest-agent` 服务状态？如果需要手动加载 virtio 相关模块，应执行哪些命令？
# a
服务状态检查：
```bash
systemctl status qemu-guest-agent
```
手动加载模块：
```bash
modprobe virtio
modprobe virtio_pci
modprobe virtio_ring
modprobe virtio_console
```
