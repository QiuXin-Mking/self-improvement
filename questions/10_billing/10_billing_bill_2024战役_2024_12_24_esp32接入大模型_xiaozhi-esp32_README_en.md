# q
What is the primary purpose of the XiaoZhi AI Chatbot project?
# a
The project is an open-source educational initiative to help people get started with AI hardware development and understand how to integrate large language models into physical devices.

# q
Which hardware modules does XiaoZhi support for network connectivity?
# a
It supports Wi-Fi and ML307 Cat.1 4G for network connectivity.

# q
What speech-related AI capabilities are integrated into XiaoZhi?
# a
Offline voice wake-up via ESP-SR, streaming voice dialogue over WebSocket or UDP, 5-language recognition (Mandarin, Cantonese, English, Japanese, Korean) using SenseVoice, voice print recognition with 3D Speaker, and large model TTS (Volcengine or CosyVoice).

# q
Which development environment is recommended for building the XiaoZhi firmware?
# a
Cursor or VSCode with the ESP-IDF plugin (SDK version 5.3 or above), on Linux if possible for faster compilation and fewer driver issues.
