# q
Home Assistant 在树莓派上使用的安装镜像文件名是什么？
# a
镜像文件为 `haos_rpi4-64-14.2.img.xz`，存放路径 `C:\Users\Administrator\Downloads\`。

# q
推荐使用哪款烧录软件将镜像写入 SD 卡？
# a
使用 `imager_1.8.5.exe` 进行镜像烧录。

# q
安装完成后，如何通过浏览器访问 Home Assistant？
# a
访问地址为 `http://homeassistant.local:8123/`，需确保设备处于同一局域网。

# q
在树莓派上部署 Home Assistant 后，网络连接有什么限制？
# a
目前无法通过无线连接，只能使用有线网络连接。
