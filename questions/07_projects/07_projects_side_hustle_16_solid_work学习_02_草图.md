# q
在SolidWorks草图中，如何绘制一条水平直线并使其完全约束？
# a
使用直线工具，点击原点，点击另一个点，按ESC退出。然后在PropertyManager中添加几何关系，选择“水平”，直线变为黑色并显示水平标识。

# q
如何删除草图实体的几何关系？
# a
点击几何关系标识（如水平标识），右键选择“删除”。

# q
如何将草图直线转换为构造线？
# a
选中直线，在PropertyManager的选项中勾选“构造线”，直线变为点划线。

# q
完全约束的草图实体在SolidWorks中显示为什么颜色？
# a
完全约束的草图实体显示为黑色，例如完全约束的矩形或圆形。

# q
在SolidWorks草图中，智能尺寸工具的快速激活方式是什么？
# a
按住鼠标右键并向上拖动，即可激活智能尺寸工具。
