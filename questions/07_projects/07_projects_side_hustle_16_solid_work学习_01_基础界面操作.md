# q
SolidWorks 中有哪三种主要的文件类型？
# a
零件（.SLDPRT）、装配体（.SLDASM）、工程图（.SLDDRW）

# q
在 SolidWorks 中新建一个零件后，保存时默认的文件扩展名是什么？
# a
.SLDPRT（例如 test1.SLDPRT）

# q
在 SolidWorks 中绘制一个长度为 100 的正方体基本流程是怎样的？
# a
1. 新建零件，进入草图绘制，选择一个基准面（如前视基准面）。
2. 使用矩形命令中的“中心矩形”绘制矩形，通过智能尺寸标注长宽均为 100mm。
3. 退出草图，选择特征中的“拉伸凸台/基体”，设置拉伸深度为 100mm，确定后生成正方体。

# q
在 SolidWorks 绘制的正方体设计中，如何观察模型的各个方向？
# a
使用“视图定向”按钮（通常位于工具栏或视图相关区域），可快速切换到前视、上视、右视等标准视图。

# q
SolidWorks 界面中左侧的设计树可以做什么？
# a
设计树显示模型的结构历史，可从中选择特征（如凸台拉伸）或草图，方便编辑或查看建模步骤。
