# q
什么是货币宽松政策？
# a
货币宽松政策是中央银行通过增加市场货币供应量来刺激经济增长、稳定物价、实现充分就业并保持金融市场稳定的一种宏观经济调控手段。

# q
货币宽松政策的主要手段有哪些？
# a
主要手段包括：
- 降低存款准备金率
- 降低再贴现率
- 公开市场操作（央行购买证券释放资金）
- 降低基准利率

# q
货币宽松政策可能带来哪些负面影响？
# a
可能导致通货膨胀风险上升、资产价格泡沫，以及一旦政策逆转可能引发的市场动荡。

# q
通常在什么情况下实施货币宽松政策？
# a
通常在经济增长放缓、通货紧缩或金融危机时实施，同时需考虑国内外经济形势、金融市场状况和通胀压力等因素。
