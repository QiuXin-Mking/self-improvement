# q
USDT是什么？它的核心锚定机制是怎样的？
# a
USDT（Tether USD）是由Tether公司发行的、与美元1:1锚定的加密货币。其核心机制是：每发行1枚USDT，Tether声称会在银行账户中存入1美元等值的储备资产（如美元、美国国债、商业票据等），以维持其价值与美元挂钩。

# q
在加密货币市场中，USDT的主要作用是什么？能举一个实际的避险例子吗？
# a
USDT的主要作用是为高波动的加密货币市场提供价格稳定性，让用户在行情剧烈波动时可将资产兑换为USDT来锁定价值。例如，在2023年硅谷银行倒闭期间，USDT曾短暂脱钩至0.97美元，但仍被市场视为相对稳定的避险选择。
