# q
ESP-IDF 开发框架主要由哪三个核心组件构成？
# a
ESP-IDF 包含：用于编译 ESP32 代码的 Toolchain（工具链）、基于 CMake 和 Ninja 的 Build tools（构建工具），以及包含 API（软件库和源代码）与操作 Toolchain 所需脚本的 ESP-IDF 本体。

# q
在 ESP-IDF 中，使用什么构建系统来构建完整的 ESP32 应用程序？
# a
使用 CMake 和 Ninja 作为构建工具。

# q
ESP-IDF 本身提供了哪两类内容？
# a
提供了 ESP32 的 API（软件库和源代码），以及操作 Toolchain 的脚本。
