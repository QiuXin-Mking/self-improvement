# q
在中国电信网络下进行内网穿透时，运营商提供的是哪种公网地址？
# a
中国电信提供的是 IPv6 公网地址，而非 IPv4 公网地址。

# q
光猫改桥接模式需要哪些关键步骤和凭据？

1. 登录光猫管理员账号：用户名 `telecomadmin`，密码 `D#E#pz4#`。

# q
路由器通过 PPPoE 拨号后，开启 IPv6 的正确顺序是什么？如何验证连通性？
# a
1. 先用电信宽带账号完成 IPv4 的 PPPoE 拨号和登录。
2. 登录成功后再开启路由器的 IPv6 功能，切勿提前开启。
3. 验证时访问测试网址 `http://ipv4.testipv6.cn/`，完成出向和入向测试。

# q
虚拟机和物理主机在 IPv6 网络中的安全组差异是什么？
# a
- 虚拟机通常可以自动获取 IPv6 地址，且不受宿主机安全组（防火墙）的阻碍，入站访问更容易。
- 实体电脑自带的系统安全组默认会阻止外部入站连接，直接使用 IPv6 时可能需要手动调整防火墙规则。
