# q
如何将中国电信光猫的路由模式改为桥接模式？
# a
1. 登录光猫后台，进入“网络”→“宽带设置”。  
2. 在连接名称中选择`1_INTERNET_R_VID_41`。  
3. 业务类型选“上网”，连接类型改为“Bridge”（或“桥接”）。  
4. 点击“保存/应用”。

# q
在 Windows 上用什么命令查看完整的网络配置信息（包括 IPv6 地址）？
# a
在命令提示符中执行：
```cmd
ipconfig /all
```

# q
如何测试本机是否能正常访问 IPv6 网络？
# a
使用浏览器打开 https://test-ipv6.com 进行连通性检测。
