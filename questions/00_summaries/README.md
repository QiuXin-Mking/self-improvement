# q
这个知识库覆盖了哪些存储核心技术？
# a
覆盖 Ceph 全链路（RADOS/RBD/RGW/CephFS）、Lustre 并行文件系统、对象存储（S3/AWS/Gofys/STS）、文件系统（JuiceFS/FUSE/EXT4 等），以及 RocksDB 嵌入式存储引擎。

# q
根据 README，排查 Ceph 故障的“三板斧”是什么？
# a
日志分析、strace 系统调用跟踪、`dump_historic_ops` 历史操作 dump。

# q
在 README 给出的目录结构中，问题案例库位于哪个目录，包含多少个生产问题？
# a
位于 `05_problems/cases/`，包含 71 个按组件分类的生产问题。

# q
根据 README 的“按角色使用”表，复习面试应从哪个摘要文件开始？

从 `_summary_career.md` 开始，包含八股文和算法。
