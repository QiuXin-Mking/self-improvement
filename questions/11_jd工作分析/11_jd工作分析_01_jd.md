# q
存储运维岗位的核心职责包括哪些方面？
# a
核心职责包括：
1. 负责企业云存储、分布式存储日常运维（资源规划、容量管理、性能调优，保障稳定可靠）；
2. 统筹数据备份、归档、容灾体系搭建与维护（制定备份策略，定期开展备份校验、灾难恢复演练）；
3. 排查存储及备份系统故障（处理IO异常、链路中断、存储报错等，输出复盘与优化方案）；
4. 监控存储集群状态，优化架构，跟进版本升级、补丁更新，提升可用性与安全性；
5. 梳理运维文档，规范流程，配合业务完成资源交付、扩容及技术支撑。

# q
该存储运维岗位对工作经验和技术背景有何硬性要求？
# a
要求本科及以上学历，计算机相关专业；5年及以上分布式存储、集中式存储与备份运维经验；熟悉主流分布式存储和企业级存储架构，精通数据备份、复制、容灾技术原理；掌握Linux基础运维命令，具备独立排查存储故障的能力；有云存储集群运维经验者优先。
