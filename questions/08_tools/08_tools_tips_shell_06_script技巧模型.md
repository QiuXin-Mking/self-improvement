# q
check_ipaddr函数如何验证IP地址的格式和每个字节的范围？
# a
函数首先使用正则表达式 `^[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}$` 检查IP地址是否符合“数字.数字.数字.数字”的格式，若不匹配则报错退出。然后通过awk以点号分割IP地址，依次检查四个数字是否大于255或小于0，若超出范围则报错退出，全部通过则返回0。

# q
脚本中如何根据传入参数数量决定是否检查网关IP？
# a
通过判断 `$#`（参数个数）是否等于4来决定：若为4，则调用 `check_ipaddr $GATEWAY` 对网关地址进行校验。参数少于4时不检查网关，因为网关是可选的。

# q
脚本中 `call_help` 函数展示了什么内容，使用了哪个特殊变量？
# a
`call_help` 函数展示了脚本的用途（网络配置）和使用方法，包含 `-i` 和 `-b` 选项的说明以及示例用法。其中用到了特殊变量 `$0` 来动态获取脚本名称，以便在帮助信息中显示实际调用命令。
