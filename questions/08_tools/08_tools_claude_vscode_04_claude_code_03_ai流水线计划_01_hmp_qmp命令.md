# q
如何制作 hmp_app 调用关系表格？
# a
需要找出所有 `hmp_app` 的调用位置，制作一个 markdown 表格，包含以下三列：  
1. HMP命令  
2. 调用 `hmp_app` 函数的文件名称  
3. 调用 `hmp_app` 函数的父函数名称  
表格需尽可能详细，并记录在 `hmp调用关系.md` 文件中。

# q
HMP 专家需要分析并制作什么样的表格？
# a
HMP 专家需要根据 QEMU 的函数对应关系，找出 HMP 命令所有可能的输出和 QMP 命令所有可能的输出，并按场景划分清晰。生成的 markdown 表格包含三列：  
1. HMP命令  
2. 场景  
3. HMP输出打印  
表格需尽可能详细，记录在 `.md` 文件中。

# q
HMP 和 QMP 联合分析的综合表格需要包含哪些列？
# a

1. HMP命令  
2. 场景  
3. HMP输出打印  
4. QMP输出打印
