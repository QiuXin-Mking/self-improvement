# q
如何使用sed删除文件中的所有空行？
# a
```bash
sed '/^$/d' 文件名
```
例如：`sed '/^$/d' file`

# q
如何使用sed在指定行后追加内容并直接修改文件？
# a
```bash
sed -i "行号a追加内容" 文件名
```
例如：`sed -i "1ahello" 1.txt` 在第一行后添加"hello"并保存修改。

# q
如何使用sed将匹配特定模式的行写入另一个文件？
# a
```bash
sed -n '/模式/w 目标文件' 源文件
```
例如：`sed -n '/UUID/w ifcfg-enp1' ifcfg-enp1.bak` 会将包含UUID的行写入ifcfg-enp1（覆盖原内容）。

# q
如何使用sed删除从匹配某模式的行到文件末尾的所有行？
# a
```bash
sed '/模式/,$d' 文件名
```
例如：`sed '/ab/,$d' 1.txt` 会删除从第一次匹配"ab"的行开始直到文件结尾的所有行。

# q
sed命令中 `-i` 选项的作用是什么？
# a
`-i` 选项用于直接修改文件内容；不加 `-i` 时，sed 仅将修改后的结果输出到标准输出（预览），不会实际更改原文件。
