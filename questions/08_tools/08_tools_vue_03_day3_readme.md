# q
Vue 的单文件组件是什么？
# a
单文件组件（Single-File Component，SFC）是将模板、脚本和样式等封装在一个 `.vue` 文件中的组件格式。

# q
Vue 单文件组件的最小模板由哪些部分构成？
# a
由 `<script setup>` 脚本部分和 `<template>` 模板部分组成。

# q
如何在 Vue 的 `<script setup>` 中导入 `ref`？
# a
```js
import {ref} from 'vue'
```

# q
Vue 的 `reactive` 函数可以用于哪些数据结构？
# a
可以用于对象、数组、Map、Set 等复杂类型，将它们转换为响应式数据。
