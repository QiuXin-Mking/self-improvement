# q
在 Bash 脚本中如何定义一个加法函数，并调用它传入参数？
# a
使用 `function_name() { ... }` 语法定义函数，在函数内通过 `$1`, `$2` 获取参数，最后在脚本里直接调用函数名并传入参数。示例：
```bash
#!/bin/bash
add(){
    a=$1;
    b=$2;
    sum=`expr $a + $b`;
    echo "the sum is ${sum}"
}
add $1 $2
```
执行 `./add.sh 1 2` 将输出 `the sum is 3`。

# q
在 Bash 函数中进行变量赋值时，需要注意什么语法规则？
# a
等号两边不能有空格。例如 `a=$1` 是正确的，`a = $1` 会报错。

# q
使用 `expr` 进行算术运算时，有哪些关键的语法要求？
# a
- 运算符两边必须有空格，如 `expr $a + $b` 不能写成 `expr $a+$b`。
- 整个 `expr` 表达式必须用反引号（或 `$()`）包裹，以将计算结果赋值给变量，如 `sum=\`expr $a + $b\`` 或 `sum=$(expr $a + $b)`。
