# q
正则表达式中的 `\b` 有什么作用？
# a
`\b` 是单词边界锚点，用于匹配单词的开头或结尾位置。例如，`\bt\w+\b` 匹配以字母 "t" 开头的完整单词。

# q
字符类 `[0-9]` 和 `[a-fA-F]` 分别匹配什么内容？
# a
`[0-9]` 匹配从 0 到 9 的单个数字字符；`[a-fA-F]` 匹配从 a 到 f 或 A 到 F 中的任意一个英文字母。

# q
如何写出匹配类似 `mdbs-1.1.18-Linux.rpm` 文件名的正则表达式？
# a
使用模式 `mdbs-[0-9]+\.[0-9]+\.[a-zA-Z0-9]+-Linux\.rpm`，其中 `[0-9]+` 匹配版本号中的数字序列，`\.` 转义点号，`[a-zA-Z0-9]+` 匹配由字母和数字组成的构建标识。
