# q
在 vi 的命令模式下，如何快速跳转到文件的最后一行？
# a
按 `Shift + G`（即 `G`）跳转到最后一行。

# q
在 vi 中，如何在命令模式下向前和向后搜索字符串？
# a
向前（向下）搜索：按 `/` 后输入搜索词并按回车。  
向后（向上）搜索：按 `?` 后输入搜索词并按回车。  
查找下一个匹配：按 `n`，反向查找下一个：按 `N`。  
关闭搜索高亮：命令模式下输入 `:nohlsearch` 或 `:set nohlsearch`（可简写 `:noh` 或 `:set noh`）。

# q
在 vi 中，如何复制、粘贴和删除一整行？
# a
在命令模式下：  
- 复制一行：将光标移至目标行，按 `yy`；复制 n 行：按 `nyy`（例如 `3yy` 复制 3 行）。  
- 粘贴：将光标移至目标位置，按 `p`（粘贴到下一行）或 `P`（粘贴到上一行）。  
- 删除一行：按 `dd`；删除 n 行：按 `ndd`。

# q
在 vi 中，如何显示和取消行号？
# a
命令模式下输入 `:set nu` 或 `:set number` 显示行号，输入 `:set nonu` 或 `:set nonumber` 取消行号。

# q
在 vi 中，如何跳转到文件的第 66 行？
# a
在命令模式下使用 `66G`（即输入 66 后按 `Shift + G`），或者在命令行模式输入 `:66` 然后回车。
