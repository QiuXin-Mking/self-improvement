# q
如何在一台主机上启动 iperf3 服务端？
# a
执行命令：
```
iperf3 -s
```

# q
如何使用 iperf3 测试两台主机之间的网络带宽？
# a
在服务端主机运行 `iperf3 -s`，在客户端主机运行：
```
iperf3 -c <服务器IP>
```
例如：
```
iperf3 -c 172.22.251.104
```

# q
如何使用 iperf3 进行限速的 UDP 带宽测试？
# a
服务端仍使用 `iperf3 -s`，客户端使用 `-u` 指定 UDP 模式，`-b` 限制带宽。例如限制为 1 Mbits/sec：
```
iperf3 -c 172.22.251.104 -u -b 1M
```
