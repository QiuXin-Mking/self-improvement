# q
如何在Excel中检查单元格A5是否在A29:A48范围内，并返回对应行J列的值？
# a
使用以下公式（放入F5）：
```excel
=IFERROR(INDEX($J$29:$J$48, MATCH(A5, $A$29:$A$48, 0)), "")
```
- `MATCH(A5, $A$29:$A$48, 0)` 查找A5在A29:A48中的位置
- `INDEX($J$29:$J$48, 位置)` 返回该位置对应的J列值
- `IFERROR` 处理未找到的情况，返回空字符串

# q
公式 `=IFERROR(INDEX($J$29:$J$48, MATCH(A5, $A$29:$A$48, 0)), "")` 中，`IFERROR` 的作用是什么？
# a
当 `MATCH` 未找到匹配项时，避免单元格显示错误值（如 `#N/A`），而是返回空字符串 `""`，保持表格整洁。

# q
在 `=IFERROR(INDEX($J$29:$J$48, MATCH(A5, $A$29:$A$48, 0)), "")` 中，哪些引用使用了绝对引用？为什么？
# a
- 查找范围 `$A$29:$A$48` 和返回值范围 `$J$29:$J$48` 使用绝对引用
- 目的是当公式向下或向右填充时，锁定这些区域不偏移，确保始终在正确的范围内查找
- `A5` 为相对引用，会根据填充位置自动调整要查找的值
