# q
IPv6地址的长度和首选表示法是什么？
# a
IPv6地址长度为128位，首选标识法为 `x:x:x:x:x:x:x:x`，其中每个x是8个16位的十六进制值。

# q
IPv6地址的短格式表示法有哪些规则？
# a
- 省略前导零：如 `1050:0000:0000:0000:0005:0600:300c:326b` 可写作 `1050:0:0:0:5:600:300c:326b`。
- 双冒号：用 `::` 替代连续的全零段，整个地址中只能使用一次。例如 `ff06:0:0:0:0:0:0:c3` 可写作 `ff06::c3`。

# q
配置IPv6时通常进入哪个目录查看网卡信息？
# a
```bash
cd /etc/sysconfig/network-scripts/
ip a
```
查看后端网卡配置。
