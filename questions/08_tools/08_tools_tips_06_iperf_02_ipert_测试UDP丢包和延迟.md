# q
iperf3测试UDP丢包和延迟时，用哪个参数指定使用UDP协议？
# a
使用 `-u` 参数。

# q
iperf3测试UDP时，如何指定目标带宽？
# a
使用 `-b` 参数，例如 `-b 100M` 表示限制带宽为 100 Mbps。

# q
iperf3输出中的“Jitter”代表什么？
# a
Jitter 代表抖动时间，即传输延迟的变化量（通常以毫秒为单位）。

# q
iperf3 UDP测试结果中，“Lost/Total Datagrams”列的含义是什么？
# a
显示丢失的数据报数量与总的数据报数量，以及丢包率（百分比）。例如 `0/1380 (0%)` 表示未丢包。

# q
如何设置iperf3报告的间隔时间？
# a
使用 `-i` 参数，后跟秒数。例如 `-i 3` 表示每3秒输出一次报告。
