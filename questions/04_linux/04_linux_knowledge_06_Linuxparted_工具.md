# q
parted工具是什么？
# a
parted 是一个用于硬盘分区或调整分区大小的工具，支持创建、清除、调整、移动和复制多种文件系统分区，包括 ext2、ext3、linux-swap、FAT、FAT32、reiserfs、HFS，并能检测 jfs、ntfs、ufs 和 xfs 分区。常用于为新操作系统创建空间、重新分配硬盘使用情况以及数据迁移。

# q
parted工具支持操作或检测哪些文件系统类型？
# a
支持创建、调整、移动和复制的文件系统：ext2、ext3、linux-swap、FAT、FAT32、reiserfs、HFS（苹果系统）；支持检测的文件系统：jfs、ntfs、ufs、xfs。
