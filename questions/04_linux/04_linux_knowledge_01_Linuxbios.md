# q
BIOS是什么？
# a
BIOS（Basic Input Output System，基本输入输出系统）是一组固化在计算机主板ROM芯片上的程序，保存着计算机最重要的基本输入输出程序、开机自检程序、系统自启动程序，并可从CMOS中读写系统设置信息，为计算机提供最底层的硬件设置和控制。

# q
BIOS的主要功能是什么？
# a
1. 提供最底层、最直接的硬件设置和控制。
2. 完成开机自检（POST）。
3. 引导系统自启动程序。
4. 从CMOS读写系统配置。
5. 向作业系统提供系统参数，隐藏硬件变化，程序通过BIOS功能而非直接控制硬件（但现代操作系统通常会忽略该抽象层直接控制硬件）。

# q
BIOS与CMOS的关系是什么？
# a
BIOS程序本身存储在主板的ROM芯片中，而CMOS是一块可读写的RAM芯片，用于保存系统设置的具体信息。BIOS可以从CMOS中读取或写入这些配置数据。
