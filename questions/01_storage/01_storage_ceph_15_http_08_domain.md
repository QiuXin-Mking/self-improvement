# q
什么是 domain（域名）？
# a
域名（domain）是互联网上的一个标识，由一串字符组成，用于识别和定位网络上的计算机或资源。用户访问网站时输入的网址的主要部分就是域名，它在技术上通过 DNS（域名系统）解析为 IP 地址，从而建立网络连接。

# q
什么是 subdomain（子域名）？
# a
子域名（subdomain）是域名系统等级中属于更高一层域的域，用于区分一个主域名下的不同服务或功能。例如，在 `mail.example.com` 和 `calendar.example.com` 中，`mail` 和 `calendar` 都是 `example.com` 的子域。

# q
subdomain 与 domain 之间是什么关系？
# a
子域名是域名的一个组成部分，用于进一步细分和标识域名下的不同部分或服务。它们共同构成互联网上的地址系统，使用户可以方便地访问和定位网络资源。例如 `mail.example.com` 中，`example.com` 是域名，`mail` 是其子域名。
