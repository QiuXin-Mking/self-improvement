# q
性能测试环境中，网卡的默认路由应如何配置以避免网络冲突？
# a
管理网口（如eno2）设置 `DEFROUTE="yes"`，高性能数据网口（如ens5f1、ens5f2）设置 `DEFROUTE="no"`。只允许一个默认路由存在，防止多网卡默认路由导致网络异常。

# q
如何通过命令行确认某块网卡是否为25GE网卡？
# a
使用 `ethtool` 命令查询网卡，若输出中 `Speed` 显示为 `25000Mb/s`（如 `ethtool enp175s0f3` 所示），则表明该网卡是25GE网卡。

# q
25GE网卡在硬件外观上有什么典型特征？
# a
25GE网卡使用的光纤线缆头部通常为蓝色外观，而管理网口一般使用千兆网口。

# q
在分布式存储性能测试的网卡配置中，常见的路由设置坑是什么？
# a
如果在多个网卡（尤其是高性能数据网口）上配置了 `GATEWAY` 或开启了默认路由（`DEFROUTE=yes`），会导致路由冲突。正确做法是仅在一个节点的管理网口上保留 `GATEWAY` 并设置 `DEFROUTE=yes`，其他数据网口应设为 `DEFROUTE=no`。
