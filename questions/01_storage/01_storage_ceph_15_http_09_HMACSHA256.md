# q
HMAC-SHA256是什么
# a
HMAC-SHA256是一种基于哈希的消息认证码算法，它结合密钥和消息数据生成256位的哈希值，用于验证消息的完整性和真实性。在C++中通常通过HMACSHA256类实现。

# q
HMACSHA256类中Update和Final方法的作用是什么
# a
Update方法用于分块向对象中添加数据，可多次调用，方便处理大文件或流式数据，对象内部会维护哈希计算状态。Final方法用于生成最终的HMAC-SHA256哈希值，调用后对象通常重置内部状态，以便重新开始新的哈希计算。

# q
初始化HMACSHA256类需要哪些关键数据
# a
必须提供密钥（Key），用于与消息数据结合生成唯一的认证哈希。部分实现还支持可选的初始向量（IV）以增强安全性，但并非所有库都强制要求。密钥需安全存储，避免泄露。
