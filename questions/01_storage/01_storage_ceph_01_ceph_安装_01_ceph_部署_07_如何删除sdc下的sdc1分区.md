# q
如何使用 fdisk 删除 /dev/sdc 磁盘上的 sdc1 分区？
# a
执行 `fdisk /dev/sdc` 进入交互式分区管理界面，然后按以下顺序操作：
1. 输入 `p` 并回车，显示当前分区表，确认要删除的分区号（例如 sdc1 对应的分区号是 `1`）
2. 输入 `d` 并回车，进入删除分区模式，输入分区号 `1` 并回车
3. 输入 `w` 并回车，将删除操作写入分区表并退出 fdisk

# q
fdisk 中 `p`、`d`、`w` 命令的作用分别是什么？
# a
- `p`：打印（显示）磁盘当前的分区表
- `d`：删除一个分区（需随后指定分区号）
- `w`：将修改后的分区表写入磁盘并退出 fdisk
