# q
如何在Python代码中快速设置pdb断点？
# a
在代码中插入 `import pdb; pdb.set_trace()`，当执行到该行时会暂停并进入pdb交互调试模式。

# q
pdb调试器是什么？
# a
pdb是Python内建的交互式源代码调试器，允许在代码中设置断点、查看变量、单步执行等。

# q
进入pdb交互模式后，常用的调试命令有哪些？
# a
常用命令：
- `l`：列出当前代码周围的几行
- `n`：执行下一行（不进入函数）
- `s`：执行下一行并进入函数
- `c`：继续执行到下一个断点或结束
- `p 变量名`：打印变量值
- `q`：退出pdb并结束程序

# q
如何用命令行直接启动pdb调试一个脚本？
# a
使用 `python -m pdb my_script.py`，程序将在脚本第一行暂停并进入pdb交互模式。
