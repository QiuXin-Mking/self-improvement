# q
超聚变服务器性能测试中，操作系统应安装在哪个磁盘上？为什么？
# a
操作系统 CentOS 7.8 必须安装在 **sdm 盘**上，不能安装在 8T 或 NVMe 盘上，否则性能测试结果无效。

# q
后端节点 77 和 79 使用的后端网卡接口是什么？
# a
节点 77 和 79 的后端网卡口为 **ens5f2**，节点 78 的后端网卡口为 **ens6f2**（物理位置均为从左到右第三个网口）。

# q
在该性能测试环境中，业务网段与后端网段的 IP 配置和连通性规则是怎样的？
# a
- 前端节点仅配置业务网段（如 172.250.101.x），后端节点同时配置业务网段（172.250.77.x）和后端网段（192.168.77.x）。
- ping 连通性规则：只有双方都配置了同一网段（如都配了 172.250 网段）时才能相互 ping 通；如果一个节点配置了某网段而另一个没有，则无法 ping 通该网段的地址。

# q
后端节点的业务网段与后端网段分别对应的接口如何配置？
# a
后端节点的 f1 接口配置 **172.250** 网段（业务），f2 接口配置 **192.168** 网段（后端），以实现业务流量与后端存储流量的分离。
