# q
这段笔记描述了什么组网架构？
# a
描述了一个包含前端节点、后端节点、上下交换机以及业务网段（172.250, 172.251）和后端存储网段（192.168开头）的 MDBS 系统网络拓扑，使用了 bond4 组网配合 25GE/10GE 交换机。

# q
笔记中提到的业务网段有哪些？
# a
iSCSI 业务网段为 172.250 和 172.251（压力机业务网），管理网段为 172.22，后端存储网段使用 192.168 开头的地址。

# q
网络拓扑中交换机如何区分与命名？
# a
交换机分为“上交换机”和“下交换机”，并标注有交换机13到18，部分交换机对应 bond 上下联，例如“bond上交换机”和“bond下交换机”。

# q
节点网口的配置原则是什么？
# a
后端节点使用 0、2 网口配置后端存储网口（192.168 网段），前端节点使用 0、2 网口配置前端业务网口（172.250/251 网段），网卡分为芯片A和芯片B，每个芯片提供4个网口（0-3）。
