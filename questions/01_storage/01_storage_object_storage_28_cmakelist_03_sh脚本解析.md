# q
shell脚本中set -e的作用是什么？
# a
`set -e` 使脚本在任何命令返回非零退出状态（即执行失败）时立即退出，不再执行后续命令，用于捕获错误并防止错误累积。

# q
shell脚本中set -x的作用是什么？
# a
`set -x` 在每条命令执行前将其打印到标准错误输出（通常是终端），用于调试，帮助观察脚本实际执行的命令。

# q
在shell脚本开头使用 `set -ex` 有什么好处？
# a
`set -ex` 结合了 `-e` 和 `-x`：既能在命令失败时立即退出（`-e`），又能在执行前打印命令（`-x`），便于快速定位问题，提高脚本开发和调试效率。
