# q
缓存的核心原理是什么？
# a
缓存基于**局部性原理**（locality of reference），包括**时间局部性**（temporal locality：近期访问过的数据很可能被再次访问）和**空间局部性**（spatial locality：靠近已访问数据附近的数据很可能被访问），利用较小的快速存储来弥补各级存储设备之间的速度差异。

# q
缓存关联性中的映射策略有哪几种？
# a
1. **直接映射**（Direct Mapped）：每个内存地址固定映射到唯一的cache line。
2. **全相联**（Fully Associative）：任意内存地址可缓存到任意cache line。
3. **组相联**（N-Way Associative）：缓存按组等分，一个内存地址唯一确定一个组，数据可存放在组内的任意cache line，是前两种的折中。

# q
全写法和写回法这两种写策略有何区别？
# a
- **全写法**（Write-Through）：CPU写命中时，数据同时写入Cache和主存；替换时新块直接覆盖，无需写回。
- **写回法**（Write-Back）：CPU写命中时，只更新Cache内容，不立即写入主存，直到该块被替换时才写回；每个Cache行需要一个脏位标记是否被修改过。

# q
衡量缓存系统性能的关键指标是什么？
# a
**cache命中率**（hit rate）和**命中性能**（访问延迟），命中率越高、命中时延迟越低，系统性能越好。
