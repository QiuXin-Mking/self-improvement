# q
calloc 函数的原型是什么？参数含义是什么？
# a
```c
void* calloc(size_t num, size_t size);
```
`num` 表示要分配的元素个数，`size` 表示每个元素的字节大小。calloc 会在堆区申请 `num * size` 字节的连续内存空间，并返回指向该空间起始地址的指针。

# q
calloc 和 malloc 的主要区别有哪些？
# a
1. 参数方式不同：malloc 接受一个总字节数，calloc 接受元素个数和单个元素大小两个参数。  
2. calloc 会在返回地址之前，将所申请内存空间的每个字节都初始化为 0；malloc 不会进行初始化，内存内容不确定。

# q
C 程序运行时内存主要划分为哪些区域？堆区和栈区有何不同？
# a
主要区域：代码区、数据区、堆区、栈区。  
- 栈区：由编译器自动申请和释放，存放函数的局部变量、参数值和临时变量。  
- 堆区：需要程序员显式调用 malloc/calloc 等函数分配内存，使用完毕后必须通过 free 手动释放，否则会造成内存泄漏。

# q
使用 calloc 申请动态内存后必须做什么？
# a
必须调用 `free(p)` 释放所申请的内存空间，并将指针置为 `NULL`（如 `p = NULL;`），避免野指针和内存泄漏。
