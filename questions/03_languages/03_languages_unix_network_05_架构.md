# q
在客户端-服务器模型中，通讯由哪一方发起？
# a
通讯由客户端发起。

# q
TCP 和 UDP 的全称分别是什么？在 Web 场景中常用哪一种？
# a
TCP 是 Transmission Control Protocol，UDP 是 User Datagram Protocol。Web 客户与服务器之间通常使用 TCP 进行通信。

# q
局域网（LAN）的核心特征是什么？
# a
LAN 覆盖较小的地理区域（如单栋建筑或园区），使计算机、打印机等设备能够彼此通信和共享资源，通常通过以太网线缆或 Wi-Fi 无线技术连接。

# q
广域网（WAN）与局域网的主要区别是什么？
# a
WAN 覆盖更大的地理范围（可跨城市、国家或洲），用于连接多个 LAN 及其他网络，实现远距离通信和资源共享，互联网就是广域网的典型例子。
