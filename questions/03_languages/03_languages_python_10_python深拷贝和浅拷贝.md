# q
Python中浅拷贝和深拷贝的核心区别是什么？
# a
浅拷贝只复制对象的引用（指针），新旧对象共享同一块内存；深拷贝则创建一个完全独立的新对象，与原对象不共享内存，修改新对象不会影响原对象。

# q
上述Python代码中导致输出重复数据的根本原因是什么？
# a
`disk_rsp` 字典在循环外只创建了一次，每次循环中 `rsp.append(disk_rsp)` 添加的都是同一个字典对象的引用（浅拷贝），后续循环修改该字典会覆盖之前添加的内容，导致最终列表中所有元素都显示为最后一次循环的数据。
