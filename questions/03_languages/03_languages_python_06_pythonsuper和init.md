# q
Python中的super()是什么？它的核心作用是什么？
# a
super()返回一个特殊的代理对象，用于访问父类中的属性，严格按照继承关系（MRO）查找父类的方法和属性。

# q
在子类的__init__方法中使用super().__init__()的目的是什么？
# a
将父类的__init__构造方法继承过来并执行，避免子类的__init__覆盖父类的__init__，使子类实例既能初始化父类的属性，又能添加自己的属性，从而可以调用父类和子类的成员属性与方法。

# q
如果子类定义了__init__但不调用super().__init__()，会导致什么问题？
# a
父类的__init__方法会被完全覆盖，导致无法初始化父类的实例属性，子类实例也就无法访问父类中定义的成员属性和方法。

# q
Python2和Python3中super()的用法有什么区别？
# a
Python2中需要显式传递当前类和self：`super(自己的类, self).__init__()`，Python3中可以简写为`super().__init__()`，无需传递类和self。
