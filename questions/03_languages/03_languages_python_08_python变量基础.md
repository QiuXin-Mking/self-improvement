# q
Python中如何向字典动态添加新字段？
# a
使用方括号语法，如 `options["type"] = "edit_total"`。

# q
Python中定义一个空字典的常见方式是什么？
# a
使用 `dict()` 或 `{}`，例如 `disk_rsp = dict()`。

# q
在Python中如何调用系统命令行并执行脚本？
# a
使用 `os.system()` 函数，传入命令字符串，如 `os.system(path + 'snap_test.py')`。

# q
Python中将整数或对象转换为字符串使用哪个内置函数？
# a
使用 `str(x)` 函数，可将对象 `x` 转换为字符串。
