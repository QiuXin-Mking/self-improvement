# q
printf中 `%#lx` 的作用是什么？会输出什么格式？
# a
`#` 标志会为十六进制数添加 `0x` 前缀，`l` 表示参数为 `unsigned long int`。例如 `printf("%#lx", 0x41)` 输出 `0x41`。

# q
`%#o` 在 printf 中如何表现？与不加 # 有何区别？
# a
`#` 标志会强制输出八进制数时带前导 `0`。例如 `printf("%#o", 65)` 输出 `0101`。不加 `#` 时只会输出 `101`。

# q
`%lx`、`%x` 和 `%u` 分别用于格式化什么类型的整数？
# a
- `%lx`：格式化 `unsigned long int`（长无符号整型）为十六进制  
- `%x`：格式化 `unsigned int`（无符号整型）为十六进制  
- `%u`：格式化 `uint16_t` 或其他无符号整型为十进制
