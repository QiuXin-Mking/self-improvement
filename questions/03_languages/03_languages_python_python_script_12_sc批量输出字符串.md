# q
这个Python脚本的核心功能是什么？
# a
生成300个iSCSI IQN字符串，每个字符串由基础IQN（`iqn.1994-05.com.redhat:57769080a`）拼接三位数字后缀得到，后缀从000到299。

# q
脚本中生成三位数字后缀的原理是什么？
# a
使用 `suffix = str(1000 + i)[1:]`：循环变量 `i` 从0到299，`1000 + i` 得到1000~1299，转为字符串后取索引1至末尾（丢弃最高位'1'），从而获得固定三位数的字符串。`i<100` 时自动前补零（如 i=0 → "000"，i=25 → "025"），i≥100 时直接取后三位（如 i=299 → "299"）。

# q
生成的IQN字符串是如何存储与输出的？
# a
所有生成的IQN字符串依次追加到列表 `iqn_list` 中，循环结束后再通过一个独立的 `for` 循环遍历该列表，逐行打印每个IQN字符串。
