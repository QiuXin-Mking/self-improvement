# q
`__sync_sub_and_fetch`是什么？
# a
`__sync_sub_and_fetch`是 GCC 内置的原子操作函数，用于对指定变量执行不可分割的减法操作，并返回减法后的新值。该操作不会被其他线程或进程中断，可保证多线程环境下的数据安全。

# q
`__sync_sub_and_fetch`的核心原理是什么？
# a
该函数通过硬件支持的原子指令（或内存屏障）确保“读取、减法、写回”的整个过程为一个不可分割的整体，避免多线程同时操作时出现数据竞争，因此总是返回减法后的最终结果。

# q
`__sync_sub_and_fetch`的语法和参数含义是什么？
# a
语法：  
```c
type __sync_sub_and_fetch(type *ptr, type value);
```
- `type`：要操作的变量类型（仅限基本数据类型，如整数、指针等）；
- `ptr`：指向被操作变量的指针；
- `value`：要从变量中减去的值。  
函数原子地执行 `*ptr = *ptr - value`，并返回更新后的 `*ptr`。

# q
`__sync_sub_and_fetch`的使用有哪些限制？
# a
只能用于基本数据类型（如 `int`、`long`、指针等），不能用于结构体、数组等复杂类型。若需对复杂类型进行原子操作，可考虑使用 C++11 的原子操作库。
