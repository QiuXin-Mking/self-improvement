# q
在Go中如何动态构建一个二维切片？
# a
先声明一个空的二维切片，再使用 `append` 将一维切片作为行逐行添加进去。例如：
```go
values := [][]int{}
row1 := []int{1, 2, 3}
row2 := []int{4, 5, 6}
values = append(values, row1)
values = append(values, row2)
```
访问元素时使用 `values[行索引][列索引]`。

# q
如何声明并初始化一个固定大小的二维数组？
# a
使用 `var 数组名 = [行数][列数]类型{}` 或直接指定初始值：
```go
// 全零初始化
sites := [2][2]string{}
sites[0][0] = "Google"
// 初始化时赋值
a := [5][2]int{ {0,0}, {1,2}, {2,4}, {3,6}, {4,8} }
```

# q
如何遍历Go二维数组的所有元素？
# a
使用两层嵌套 `for` 循环，外层遍历行索引，内层遍历列索引：
```go
var a = [5][2]int{ {0,0}, {1,2}, {2,4}, {3,6}, {4,8} }
for i := 0; i < 5; i++ {
    for j := 0; j < 2; j++ {
        fmt.Printf("a[%d][%d] = %d\n", i, j, a[i][j])
    }
}
```
