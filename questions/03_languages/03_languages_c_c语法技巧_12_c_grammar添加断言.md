# q
测试中如何对函数返回值添加断言？
# a
在调用被测函数后，获取其返回值，使用 `assert()` 宏或测试框架的断言方法检查返回值是否符合预期。

# q
如何对 `atomic_read` 的结果添加断言？
# a
调用 `atomic_read` 获取原子变量的当前值，然后用断言验证其值是否正确，常用于检验并发访问下的数据一致性。

# q
如何断言一个元素已被成功删除？
# a
先按删除逻辑执行删除操作，然后通过查找该元素，断言查找结果表示元素不存在（如返回 `NULL`、`false` 或特定错误码）。

# q
如何通过“打桩”来添加断言？
# a
使用测试桩（stub）替换目标函数，让桩函数返回可控的值，从而在调用被测代码后对依赖该返回值的后续行为进行断言。
