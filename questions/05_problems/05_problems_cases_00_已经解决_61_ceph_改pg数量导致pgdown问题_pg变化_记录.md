# q
如何记录Ceph问题排查的触发时间点？
# a
使用 `date` 命令输出当前系统时间，例如：
```bash
[root@stg_ssd_4-101-219 ~]# date
Wed Oct 15 11:22:22 CST 2025
```
将输出保存或手动记录，用于定位问题发生的时间窗口。

# q
如何从Ceph OSD日志中过滤特定时间段的日志进行分析？
# a
使用 `fgrep` 命令按时间模式过滤日志内容并重定向到文件，例如：
```bash
fgrep "2025-10-15T11:2" ceph-osd.6.log > ceph-osd.6.log_10_15_11_2x
```
其中 `"2025-10-15T11:2"` 匹配当天 11:20~11:29 的所有日志行，`ceph-osd.6.log` 为原始 OSD 日志文件。
