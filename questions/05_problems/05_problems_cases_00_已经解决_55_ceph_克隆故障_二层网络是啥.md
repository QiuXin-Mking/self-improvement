# q
什么是二层网络？
# a
二层网络（Layer 2 Network）是指 OSI 模型中第二层——数据链路层（Data Link Layer）的网络，主要负责在直接相连的设备之间通过 MAC 地址（物理地址）识别和转发数据帧（Frame）。通常指同一个局域网（LAN）、同一个广播域，设备之间通过 MAC 地址直接通信，不需要经过路由器（不涉及 IP 地址路由）。

# q
二层网络和三层网络的主要区别是什么？
# a
二层网络使用 MAC 地址通信，工作在数据链路层，典型设备为交换机，通信不经过路由器，广播包会到达整个广播域且不能跨不同子网直接通信；三层网络使用 IP 地址通信，工作在网络层，典型设备为路由器，需要 IP 路由，广播被路由隔离，可以跨网络通信。
