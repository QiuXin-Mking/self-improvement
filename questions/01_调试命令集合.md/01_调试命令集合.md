# q
如何使用iscsiadm发现并登录iSCSI目标？
# a
```bash
iscsiadm -m discovery -t st -p xxx:3260 -l
```
其中 `xxx` 为目标IP，`-m discovery` 表示执行发现操作，`-t st` 指定类型为SendTargets，`-l` 表示发现后自动登录。

# q
如何使用etcdctl快速修改存储集群的心跳超时、节点隔离和磁盘离线重建时间？
# a
```bash
# 修改心跳丢失后触发重建的时间（秒）
etcdctl --endpoints=127.0.0.1:23790 put /ttl/heartbeat_off/rebuild 300
# 修改节点隔离判定时间（秒）
etcdctl --endpoints=127.0.0.1:23790 put /ttl/isolate 300
# 修改磁盘离线后触发重建的时间（秒）
etcdctl --endpoints=127.0.0.1:23790 put /ttl/disk_off/rebuild 60
# 查询当前TTL配置
etcdctl --endpoints=127.0.0.1:23790 get /ttl --prefix
```

# q
如何在Linux上挂载CIFS网络共享并同时限制文件和目录的可执行权限？
# a
```bash
mount -t cifs //172.17.8.69/001_gitlab /home/qiuxin -o file_mode=0644,dir_mode=0755,username=John,pass=5i6106100258
```
`file_mode=0644` 禁止文件的可执行权限，`dir_mode=0755` 允许目录的访问和执行权限。

# q
如何使用engine/client工具对远端存储卷执行1MB顺序读测试？
# a
```bash
/opt/macrosan/mdbs/bin/engine/client -i 192.168.179.77 -a 290306 -p 0 -l 1024 -o 32 -f /root/read2 -m 0 -v 1
```
参数说明：`-i` 目标管理IP，`-a` 卷ID（290306），`-p` 分区号，`-l 1024` 单次IO块大小1MB（1024KB），`-o 32` 并发IO深度32，`-f /root/read2` 结果输出文件，`-m 0` 读模式，`-v 1` 详细输出。

# q
如何在不重启系统的情况下删除SCSI块设备并重新扫描SCSI总线？
# a
```bash
# 删除指定SCSI设备（以sdb为例）
echo 1 > /sys/block/sdb/device/delete

# 重新扫描所有SCSI主机适配器上的设备变化
echo "- - -" > /sys/class/scsi_host/host0/scan
```
删除设备后，可通过重新扫描让系统发现新添加或变更的LUN。
