# 墨墨背单词 (MoMo) — 竞品分析

## 1. 产品概述

墨墨背单词是中国市场最受好评的英语词汇记忆应用。它以艾宾浩斯遗忘曲线为核心算法，提供极简、无广告的纯词汇记忆体验。产品定位极为专注：不做课程，不卖社群，不搞社交，甚至不做英语听读写——只做"背单词"这一件事，并把它做到极致。用户群体以大学生和考研人群为主，覆盖六级、考研、雅思、托福、GRE 等全部主流词库。

墨墨在用户口碑上长期碾压同类产品。无论是在知乎、小红书还是应用商店评论区，"纯净""良心""算法准"是最高频的关键词。这种口碑并非来自营销投入，而是源于产品本身几乎没有干扰项——没有开屏广告、没有弹窗促销、没有课程推销——用户打开就是为了背单词，背完就关掉。

## 2. 盈利模式

墨墨的盈利模式是整个中国教育工具类产品中最值得研究的一个案例：**单词配额制**。

具体规则如下：

- 新用户免费获得约 **600 个单词** 的永久背诵额度。
- 超过 600 个后需要购买配额：**88 元 = 8,800 个单词**。
- 还有更高档位的额度包，但核心就是这个"一次性购买、永久拥有"的模式。
- **无广告、无课程、无订阅、无会员体系、无增值服务。**

这个模式的精妙之处在于：

1. **切中刚需**：考研英语大纲词约 5,500 个。免费额度仅 600 个，覆盖不了核心需求。用户从注册第一天就清醒地知道"这个产品必须付费"。
2. **零决策摩擦**：88 元一次性付费，不是订阅，不会持续扣钱。中国用户对"永久买断"有极强的偏好，而对中国式的订阅制（连续包月自动续费）普遍反感。墨墨完美避开了这个信任雷区。
3. **付费即交付完整价值**：用户付费后拿到的是完整产品的使用权，而不是一个"还需要继续付钱才能用下去"的入口。这种一次交付的感觉和"课程产品"不同，更接近"工具产品"的购买心智——你买的是锤子，不是锤子课。
4. **低 ARPU 但极高转化率**：客单价 88 元看似不高，但对于一个几乎零边际成本（无内容采购、无人工服务、无广告投放）的工具来说，转化率接近 100%（对于需要背诵超过 600 词的用户群），综合 LTV 极其健康。
5. **收入可预测**：与中国移动互联网主流的"免费+广告"或"免费+订阅"模式相比，墨墨的模式让收入几乎完全由新增用户驱动，无需担心留存率波动导致收入崩盘。

简单计算：假设日新增 1,000 人，其中 70% 的目标背诵量超过 600 词（考研用户几乎全部超过），转化率 70%，客单价 88 元，则日均收入约 43,000 元，年营收约 1,570 万元。对于个人开发或小团队而言，这是非常优秀的商业模型。

## 3. 转化漏斗

墨墨的转化漏斗设计体现了产品设计的极高成熟度——它不是靠营销话术转化用户，而是让需求本身自然驱动付费。

```
用户注册
    ↓
选择词库（考研 5500 / 六级 2000 / 雅思 4000...）
    ↓
开始背诵，单词逐个解锁
    ↓
第 600 个单词时 → 触发配额耗尽提示
    ↓
弹窗："你还有 4,900 个单词未解锁 | 88 元解锁全部"
    ↓
支付转化
```

关键洞察：

- **"撞墙"时机精准**：对于考研用户，600 个单词通常在 2-7 天内就会耗尽（取决于每天背诵量）。此时用户已经体验到了产品的核心价值——算法真的有效，界面真的很干净，没有广告真的很舒服。用户在"墙"面前几乎没有犹豫——他们已经验证了产品，只是需要解锁更多。
- **自然的紧迫感**：不是倒计时、不是限时折扣，而是"你的进度卡住了"。用户已经开始了背诵计划，中断意味着之前的复习节奏白费。这种沉没成本驱动的紧迫感远比营销促销有效。
- **体验即转化**：600 个免费单词本身就是最佳的"试用期"。用户不需要看教程、不需要读说明——背完 600 个单词，用户自然知道这个产品值不值 88 元。转化是体验的自然结果，而不是销售的结果。

## 4. 用户体验

墨墨的 UI 极其克制，属于"反流量思维"的设计：屏幕极简，核心操作只有"认识 / 模糊 / 忘记"三个按钮，无任何干扰元素。所有设计服务于一个目的——让用户最快地进入心流状态，持续背单词。

特点：

- 无启动屏广告、无弹窗、无 Banner
- 首页就是今日待复习的单词列表
- 复习流程：显示单词 → 用户自我回忆 → 点击"认识/模糊/忘记"中一个 → 查看答案和记忆曲线 → 下一个
- 辅助信息（词根、例句、助记）**折叠隐藏**，需要时才展开——不主动暴露以降低认知负荷
- 每日背完目标数量后温和提醒，不制造焦虑

## 5. 核心竞争力

1. **无广告的纯净体验**：在中文互联网几乎"处处是广告"的环境下，墨墨的零广告承诺本身就是最大的差异化武器。用户对"不卖课不卖广告"的信任是多年如一日的产品克制换来的，竞争对手很难短期复制。
2. **算法准确度**：基于艾宾浩斯遗忘曲线的个性化排程。墨墨对"遗忘临界点"的把握被用户广泛认可——复习的时机"刚刚好"，不会过早（浪费时间）也不会过晚（已经忘了）。
3. **永久购买的安心感**：88 元买断，"我买的就是我的"。无需担心"不续费课就没了"。这种所有权心理在中国用户中尤为强烈。
4. **极高的用户信任**：产品存在多年，从未变更过商业模式或推出"割韭菜"功能。用户信任是墨墨最大的护城河。
5. **词库完整性**：覆盖几乎所有主流考试的词库，且持续更新。

## 6. 弱点

1. **品类极度狭窄**：只做英语词汇。用户考完研、考完雅思后基本就会流失。LTV 上限被品类锁死。
2. **无用户生成内容 (UGC)**：所有词库内容由团队维护，无法覆盖长尾需求（如医学英语、法律英语、编程英语等专业词汇）。
3. **无 AI 能力**：完全基于规则的艾宾浩斯算法，没有智能化的个性化优化能力。虽然"够用"，但在 AI 时代可能成为技术债务。
4. **无法扩展到非词汇领域**：产品架构深度绑定于"单词记忆"，无法轻松扩展到知识管理、技能学习等更广泛的学习场景。
5. **增长天花板清晰**：中国考研人数约 400 万/年，英语学习人群虽然大但并非无限。工具品类天然没有网络效应，增长纯靠口碑+应用商店分发。

## 7. 对 KnowLoop 的启示

墨墨是 KnowLoop 最重要的竞品参照物——不是因为我们在做背单词，而是因为它的**"纯工具 + 用量计量"盈利模型**与 KnowLoop 的定位高度共振。

核心启示：

1. **"使用权售卖" > "内容售卖"**：墨墨不卖"英语课"，卖的是"高效背单词的能力"。同理，KnowLoop 不应该卖"知识内容"，而应该卖"高效记忆知识的能力"。工具价值天生比内容价值更有议价空间，因为工具是"能力放大器"——用户可以往里装任何知识。
2. **免费额度 = 体验窗口 = 转化引擎**：600 个单词的免费额度既不是"太少了根本试不出来"，也不是"太多了不用付费"。KnowLoop 的知识点免费额度应该遵循同样的逻辑——让用户在体验到算法价值后自然撞墙。
3. **一次付费的信任资产**：中国用户对"永久买断"的偏好是真金白银验证过的。KnowLoop 如果采用订阅制，需要面对用户"持续被薅羊毛"的心理抵抗。考虑"一次性购买基础版本 + 可选增值功能"或"大容量一次性额度"的模型会更符合用户心理。
4. **无广告是付费的前提**：墨墨用户愿意付费的前提是"我知道这是在买纯净"。如果产品本身有广告，用户会觉得"我已经在忍受广告了，为什么还要付钱"。KnowLoop 如果要走付费路线（而非广告路线），必须从一开始就保持清洁。
5. **工具产品的留存 = 算法的实际效果**：墨墨留存好的原因不是运营手段，而是用户发现"用这个背真的记得住"。同样，KnowLoop 的核心留存指标只有一个：用户是否真的记住了他们想记住的东西。
