# KnowLoop 付费产品化差距分析

## 摘要

KnowLoop 的核心功能（间隔重复引擎、.md 导入、多租户、JWT 认证、Markdown 渲染）已就绪，但距离可付费出售还差 **8 个维度的关键功能**，估算 **4-6 个月的 solo 开发量**。

---

## 竞品盈利模式调研

### 竞品定价全景

| 产品 | 模式 | 价格 | 广告 | 估计收入 |
|------|------|------|------|----------|
| Anki (官方 iOS) | iOS 一次买断 | $24.99 | 无 | ~$9.6M/年 |
| Quizlet | Freemium + 广告 | $3-8/月 | 有（免费版） | $139M (2025) |
| RemNote | 分级订阅 | $6-18/月 | 无 | 未公开 |
| AnkiPro | 年订阅 | $29.99/年 | 无 | 未知 |
| **墨墨背单词** | 单词额度包 | ~88 RMB ($12) | **无** | 未公开 |
| **不背单词** | 终身一次买断 | ~149 RMB ($21) | **无** | 未公开 |
| 知米背单词 | 免费+多渠道 | 各异 | 有 | 未公开 |
| Noji | 订阅 | 未知 | 无 | $4M ARR |

### 两种成功的中国模式

**墨墨背单词（最纯粹）**：
- 免费 600 词上限 → 用完就买额度包（88 RMB / 8800 词）
- 零广告、零课程推销、零功能付费墙
- 核心 SRS 算法永远完整可用，只卡"词汇量"
- 考研用户需要 5500 词 → 注册当天就撞墙 → 转化率极高

**不背单词**：
- 免费版核心引擎完整，每日 3 次拼写测试限制
- 终身买断 149 RMB（中国人不喜欢订阅）
- 无广告，靠"买了就是你的"打动用户

### 中国市场定价敏感度

- 理想月订阅：**20-50 RMB**（约 $3-7/月）
- 可接受一次买断：**50-200 RMB**（约 $7-28）
- 超过 50% 用户不愿意为学习 App 付费
- **一次买断远优先于订阅**（墨墨、不背都走这条路）
- 行业趋势：免费额度在缩小，价格在涨 30%+
- 广告容忍度极低——墨墨和不背的无广告是卖点

### 成功的变现模式总结

1. **用量计费**（墨墨）：免费证明价值，撞墙自然转化
2. **AI 作为 upsell**（Quizlet/RemNote）：AI 有真实算力成本
3. **终身+订阅双轨**（RemNote）：捕获两种付费意愿的用户
4. **开源核心+付费便利层**（Anki）：桌面免费建信任，iOS 付费
5. **无广告差异化**（墨墨/不背/Anki）：干净体验就是溢价理由
6. **Bootstrapped 成功案例**：Noji — 3 人团队，5 年到 $4M ARR，被收购

### 对 KnowLoop 的意义

- **不要走订阅**：中国市场强烈偏好一次买断
- **参考墨墨模型**：免费证明核心价值，用量/功能墙自然转化
- **无广告是竞争力**：墨墨用户夸它"不卖课不卖广告"
- **AI 生成卡片**是有真实算力成本的付费理由

## 1. 付费 / 订阅系统

### 竞品定价参考

| 产品 | 模式 | 价格 |
|------|------|------|
| Anki (官方) | iOS 一次买断 | $24.99 |
| AnkiPro | 年订阅 | $29.99/年 |
| Quizlet Plus | 月/年订阅 | $7.99/月 |
| RemNote Pro | 月订阅 | $6/月 |
| Brainscape Pro | 月/年订阅 | $19.99/月 |

### 免费层设计（核心转化杠杆）

三个维度可以限制：

| 维度 | 免费 | 付费 |
|------|------|------|
| **卡片数量** | 无限 | 无限（不该卡这个） |
| **每日复习** | 30 次/天 | 无限 |
| **功能** | 核心 SRS | AI 生成卡片、图片附件、导入导出 |

> 关键原则：核心 SRS 永远免费。付费解锁 AI、附件、批量操作——这些是 power user 愿意掏钱的功能

### 推荐方案

- **免费层**：无限卡片，30 次复习/天
- **付费层**：29-39 元/月 或 199-299 元/年
- **中国定价**：价格以 .9 结尾（29.9、199），支持花呗分期
- **大促**：618、双 11 折扣驱动 70% 年收入
- **支付网关**：国内用 Dodo Payments（MoR，支持微信/支付宝）或 Airwallex；国际用 Stripe/Paddle

> ⚠️ 微信支付和支付宝不原生支持订阅扣款。解决方案：卖年费套餐（一次性付款），对应中国消费者偏好固定期限套餐的习惯。

## 2. 多租户数据隔离

### 当前状态：SQLite，`user_id` 列隔离

### 评估

| 阶段 | 数据库 | 理由 |
|------|--------|------|
| **0–1000 用户** | 保持 SQLite | 零运维成本，物理隔离强（每用户独立 .db） |
| **1000–10000 用户** | 评估迁移 | 监控并发写入冲突 |
| **10000+** | 迁移 PostgreSQL | 行级安全、审计日志、HA |

### 短期建议

- **不用迁移**。SQLite 对当前规模完全够用
- 加 Litestream 做实时备份到 S3/B2（RPO：秒级）
- 迁移信号：出现 `SQLITE_BUSY` 错误 或 跨租户报表需求

## 3. 安全合规

### 需要补齐

| 功能 | 说明 | 优先级 |
|------|------|--------|
| Rate limiting | 防止 API 滥用（Go 的 `golang.org/x/time/rate` 或 gin 中间件） | P0 |
| 密码策略 | 当前 bcrypt 已有，需加强最小长度 + 复杂度提示 | P1 |
| HTTPS | 已部署（火山云），保持 | — |
| JWT 刷新机制 | 当前 24h 过期，需加 refresh token 或滑动过期 | P1 |
| 审计日志 | 登录、支付、数据修改的 append-only 日志 | P2 |
| GDPR / PIPL | 数据导出 + 账户删除功能 | P1 |

### 中国市场合规

- **PIPL（个人信息保护法）**：需隐私政策 + 用户数据导出/删除能力
- **ICP 备案**：国内服务器需备案
- **公司注册**：出售 SaaS 建议注册个体工商户或有限公司

## 8. 法律 & 商务

### 隐私政策 & 服务条款

- 使用开源生成器（Tempest-Solutions / app-privacy-policy-generator）生成 GDPR + PIPL 合规模板
- 只需添加公司名、管辖区、数据处理商（Stripe/Paddle、云服务商）、数据保留期
- 对 solo founder 的简单 SaaS 覆盖 ~90% 合规需求

### 退款策略

- **14 天无理由退款**（降低首次购买摩擦）
- $25 以下自动退款，$100 以上人工审核
- 年费计划中途取消：退还剩余月份，扣除已享折扣
- 续费前 7 天 + 1 天发送提醒
- 健康退款率：1-2%

### PIPL（个人信息保护法）合规

面向中国用户的核心要求：

| 条件 | 要求 |
|------|------|
| 有中文界面 / 本地支付 / 市场意图 | PIPL 适用 |
| 处理 100 万+ 人数据 | 数据必须存储在中国境内 |
| 跨境数据传输 | 需 CAC 安全评估 或 标准合同条款 |
| 处理 1000 万+ 人数据 | 至少每 2 年合规自审 |
| 违规处罚 | 最高 5000 万 RMB 或全球年收入 5% |

> 起步阶段：用户量小（<10 万），数据放海外服务器（AWS 新加坡），用 Dodo Payments 做 MoR 不需要中国公司实体，无需 ICP 备案。用户超 10 万后再找中国数据律师做完整 PIPL。

### 公司注册 & ICP

不需要中国公司的路径（推荐起步阶段）：
- 海外服务器（AWS 新加坡/美国）→ 无需 ICP 备案
- MoR（Dodo Payments/Paddle）→ 代收微信/支付宝
- 网页 App（非小程序）→ 不需要中国主体

需要中国公司时：
- 北京/上海/深圳/海南等试点区允许 100% 外资 WFOE
- 注册资本最低 100 万 RMB
- 商用 ICP 许可证（B25）需要中国营业执照
