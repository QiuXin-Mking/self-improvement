# Web版本使用说明

## 简介

Web版本提供了移动端友好的界面，让你可以在手机上方便地使用知识库学习系统。

## 安装和启动

### 1. 安装依赖

```bash
# 确保已安装Go (>= 1.21)
go version

# 安装项目依赖
go mod tidy
```

### 2. 启动Web服务器

```bash
# 直接运行
go run web_server.go

# 或构建后运行
go build -o web_app web_server.go
./web_app
```

默认会在 `http://0.0.0.0:5000` 启动服务器。

### 3. 在手机上访问

确保你的手机和电脑在同一局域网（WiFi）下，然后在手机浏览器中访问：

```
http://你的电脑IP地址:5000
```

例如：`http://192.168.1.100:5000`

**如何查看电脑IP地址：**
- Windows: 在命令提示符中输入 `ipconfig`，查看 IPv4 地址
- Mac/Linux: 在终端中输入 `ifconfig` 或 `ip addr`，查看 IP 地址

## 使用方法

### 首次使用

1. 打开网页后，如果提示需要初始化知识库，点击"初始化知识库"按钮
2. 系统会自动扫描配置的问题目录（从 `question_input` 文件读取）
3. 初始化完成后，自动开始学习

### 学习流程

1. **查看问题**：屏幕上显示问题，你可以在心里或纸上默写答案
2. **查看答案**：点击"查看答案"按钮查看标准答案
3. **反馈记忆程度**：
   - ✅ **熟练** - 记得很清楚
   - 👍 **一般** - 记得但不熟练
   - 😐 **忘记** - 忘记了部分内容
   - ❌ **完全忘记** - 完全不记得
4. **自动进入下一题**：反馈后自动显示下一个问题

### 其他功能

- **删除问题**：如果问题质量不高，可以点击"删除问题"按钮
- **查看统计**：页面顶部显示待复习数量、总问题数和正确率
- **重新初始化**：如果需要重新导入问题，可以在"无待复习问题"界面点击"重新初始化"

## 功能特点

✨ **移动端优化**：
- 响应式设计，完美适配手机屏幕
- 大按钮设计，易于点击
- 流畅的交互体验

📊 **实时统计**：
- 实时显示学习进度
- 查看待复习问题数量
- 显示正确率

🎯 **智能复习**：
- 基于艾宾浩斯遗忘曲线
- 自动安排复习时间
- 根据记忆程度调整间隔

## 技术说明

- **后端**：Gin Framework (Go)
- **前端**：HTML + CSS + JavaScript (原生，无依赖)
- **数据存储**：与命令行版本共享 `data/learning_data.json`

## 注意事项

1. **网络访问**：确保手机和电脑在同一WiFi网络下
2. **防火墙**：如果无法访问，可能需要允许5000端口的防火墙访问
3. **数据同步**：Web版本和命令行版本共享同一份数据文件，可以混用

## 开发模式

如果需要修改代码，启动时会自动启用调试模式。

## 部署到服务器

如果想在公网上访问，可以：

1. 部署到云服务器（如阿里云、腾讯云）
2. 使用内网穿透工具（如ngrok、frp）

部署时注意：
- 设置监听 `0.0.0.0` 以允许外部访问
- 配置适当的并发处理能力
- 考虑安全性和访问控制
