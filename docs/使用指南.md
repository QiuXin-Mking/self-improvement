# 快速使用指南

## 第一步：准备问题文件

默认在 `questions/` 目录下创建 `.md` 文件，格式如下：

```markdown
# q
你的问题内容
# a
你的答案内容

# q
下一个问题
# a
下一个答案
```

### 配置多个目录（可选）

编辑 `question_input` 文件可以配置多个目录：

```
questions
my_questions
path/to/other/questions
```

## 第二步：初始化知识库

```bash
go run main.go --init
```

这会扫描配置的所有目录下所有 `.md` 文件，提取问题和答案。

## 第三步：开始训练

每天运行：

```bash
go run main.go
```

### 训练流程：

1. 程序显示问题，你在纸上默写答案
2. 输入 `a` 或 `answer` 查看答案
3. 根据你的记忆程度输入反馈：
   - `1` 或 `熟练` - 记得很清楚
   - `2` 或 `一般` - 记得但不熟练
   - `3` 或 `忘记` - 忘记了部分内容
   - `4` 或 `完全忘记` - 完全不记得
4. 系统会根据反馈自动安排下次复习时间
5. 继续下一个问题

## 第四步：查看统计

```bash
go run main.go --stats
```

## 提示

- 建议每天固定时间运行，养成习惯
- 诚实反馈记忆程度，系统才能准确安排复习
- 可以随时输入 `q` 退出训练
- 数据自动保存在 `data/learning_data.json`

## 艾宾浩斯遗忘曲线

系统根据记忆程度自动调整复习间隔：

- **熟练** → 7天后复习
- **一般** → 3天后复习
- **忘记** → 1天后复习
- **完全忘记** → 2小时后复习

连续正确回答后，复习间隔会自动延长！
